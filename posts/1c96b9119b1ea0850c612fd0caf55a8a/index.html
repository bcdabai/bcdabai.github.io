C语言--memset结构体初始化

memset可以方便的清空一个结构类型的变量或数组。如：

　　struct sample_struct
　　{
　　    char csName[16];
　　    int iSeq;
　　    int iType;
　　};

对于变量 struct sample_strcut stTest;
一般情况下，清空stTest的方法：

stTest.csName[0]='\0';
stTest.iSeq=0;
stTest.iType=0;

用memset就非常方便：

memset(&stTest,0,sizeof(struct sample_struct));

如果是数组：

struct sample_struct TEST[10];

则

memset(TEST,0,sizeof(struct sample_struct)*10);
