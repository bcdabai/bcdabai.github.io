网易云音乐直链提取

1 网易云音乐的下载方法，歌曲的id有两种方式可以获得。
第一种进入网页版的网易云，选中播放的歌曲，进入播放的详情页，此时地址栏的链接里就有音乐的ID（id）。

AIP 地址

http://music.163.com/song/media/outer/url?id=歌曲ID.mp3

2 这边得到的 ID 为 1831697228 替换掉上面的 API 里面的，得到下面的地址

http://music.163.com/song/media/outer/url?id=1831697228.mp3

3 方法二客户端，在一首歌上面点右键点击复制链接，会复制出类似的地址：

https://music.163.com/song?id=431357712&userid=1368532095

中间的为歌曲的ID，照第一个替换就ok了。
