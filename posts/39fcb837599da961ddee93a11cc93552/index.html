linux 安装git并拉取代码教程

#一步一步执行以下命令

sudo apt install git  #安装git

git --version #查看安装版本号

git config user.name jtr #设置用户名

git config user.email jiangtr@cloudskysec.com #设置邮箱

ssh-keygen -t rsa -C "jiangtr@cloudskysec.com" #生成秘钥，一直往下按回车键就行

出现下图ssh秘钥生成成功。

 

cat /home/jtr/.ssh/id_rsa.pub # 显示公钥

将公钥拷到gitlab或者GitHub setiing中保存

 拉取远端仓库代码
