修改git仓库地址

要修改Git仓库的远程地址，你可以使用以下命令：

  1. 查看当前的远程地址：

    git remote -v

  2. 修改远程地址：

    git remote set-url origin <新的远程地址>

在这个命令中，你需要将 <新的远程地址> 替换为你想要设置的新的远程地址。然后，你就可以使用 git remote -v 命令再次验证远程地址是否已经修改成功。
