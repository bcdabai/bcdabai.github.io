linux 安装redis need tcl 8.5 or newer

  1. wget http://downloads.sourceforge.net/tcl/tcl8.6.1-src.tar.gz           //直接下载 
  2. sudo tar xzvf tcl8.6.1-src.tar.gz  -C /usr/local/  
  3. cd  /usr/local/tcl8.6.1/unix/  
  4. sudo ./configure  
  5. sudo make  
  6. sudo make install
