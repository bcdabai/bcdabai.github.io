UNI-APP_ios自动适应底部安全区背景，修改安全区背景

自动适应（推荐）

将所有 iPhone X（刘海屏) 底部安全区域背景颜色 自动适应，当前页面什么颜色会自动调整。

1.打开 manifest.json ，打开源码视图
2.找到 app-plus 配置项，添加以下代码

"safearea": {
  "bottom": {  
        "offset": "none" 
    }   
}

统一设置

将所有 iPhone X（刘海屏) 底部安全距离的背景色，统一设置 为一种颜色

1.打开 manifest.json ，打开源码视图
2.找到 app-plus 配置项，添加以下代码

"safearea": {  
    "background": "#CCCCCC", //背景色
    "bottom": {  
        "offset": "auto"  
    }  
}
