解决方法
  1. 问题代码

由于 headerRight参数的值应该为一个组件，而不是方法，所以会报出此警告。

static navigationOptions = () => ({
    title: '更多',
    headerRight: () => <View />
  });
  1. 解决

将上述代码改为

  static navigationOptions = () => ({
    title: '更多',
    headerRight: <View />
  });
