解决pip安装numpy问题：ERROR: Failed building wheel for numpy/ERROR: numpy-1.22.4+mkl-cp38-cp38-win_amd64.wh

出现过问题

  1. ERROR: Failed building wheel for numpy
  2. 下载了whl文件后报错ERROR: numpy-1.22.4+mkl-cp38-cp38-win_amd64.whl is not a supported wheel on this platform.

综合多篇博客的解决方法：

1. 使用pip debug --verbose 命令即可看 pip 支持
在compatible tags中查看pip支持的版本
然后在https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta官网中下载对应的轮子
将下载的轮子存储在python所在的Scripts文件中



然后在vscode终端输入pip install numpy -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com


我根据上述步骤最终是成功了
踩过的雷
直接pip install 轮子文件(❌）
使用pip._internal命令查看支持版本（❌）
