linux 5060端口号,linux常用命令

所有指令都可以通过man查看使用方法，比如想知道cp的使用方法直接敲man cp，按q退出

1、网络类

功能 指令

查看端口占用情况 lsof -P -i -n | grep 5060

查看端口状态(Linux) netstat -antp | grep 5060

查看端口状态(Windows) netstat -anb | find "5060"

过滤方法(加上-v -E过滤不需要的内容) netstat -antp | grep 5432 | grep -v -E "postgres"

临时关闭防火墙 service iptables stop

永久关闭防火墙 chkconfig --level 345 iptables off

查看SELinux状态 sestatus

关闭SELinux状态(需重启) vim /etc/sysconfig/selinux

SELINUX =disabled

添加路由(Linux) route add -net 207.207.0.0 gw 208.208.69.1 netmask 255.255.0.0

添加路由(Windows) route add 207.207.0.0 mask 255.255.0.0 208.208.101.1 -p

查看所有网络接口的属性 ifconfig

查看防火墙设置 iptables -L

查看路由表 route -n

查看所有监听端 netstat -lntp

查看所有已经建立的连接 netstat -antp

查看网络统计信息 netstat -s

2、系统类

功能 指令

查看linux版本号 lsb_release -a 或 cat /etc/redhat-release

查看内核版本 uname -a

查看操作系统位数 getconf LONG_BIT

查看内存信息 free -m

查看CPU信息 cat /proc/cpuinfo | grep -i name

查看硬盘相关路径 ls -l /dev/disk/by-id/ | grep sda1

查看硬盘容量 df -h

查看挂载节点 mount

查看硬盘读写IO iostat -x 1

查看进程信息 top -d 2 -s -n 1 -p pid

查看进程IO信息 iotop -p pid

设置系统时钟 date -s '2014-01-01 00:00:00'

修改时区 ln -sf /usr/share/zoneinfo/Etc/GMT+10 /etc/localtime

设置语言 export LANG=UTF-8.zh_CN

查看历史命令 设置格式：export HISTTIMEFORMAT="%Y%m%d-%H:%M:%S"

查看指令：history

查找文件 find / -name 'core.*' 或 locate tms8500.tar.gz

查找进程 ps -ef | grep vmsserver

ps -axfw | grep vmsserver

ps -aux | grep vmsserver

杀掉进程 kill -9 pid

查看环境变量 env

3、文件类

功能 指令

压缩 tar czvf vm8500.tar.gz vm8500

解压缩 tar xzvf vm8500.tar.gz

查看日志最新信息 tail -f /var/log/imoslog/vmserver00.log

查看指定行数日志 tail -n 1000 /var/log/imoslog/vmserver00.log

查看linux系统错误码 基本：more /usr/include/asm-generic/errno-base.h

扩展：more /usr/include/asm-generic/errno.h

复制文件 cp  src dst

移动/重命名 mv  src dst(如果src和dst是指同一文件，则表示重命名)

显示所有文件 ls -a

详细列出文件 ls –l

建目录 mkdir

逐屏查看文件内容 more

修改文件权限 chmod 777 test

(修改文件test权限为：文件拥有者、所属组成员和其他用户都拥有该文件的读写执行权限)

远程拷贝文件 scp /home/tms8500.tar.gz root@208.208.69.101:/home
