#Q8022. 十进制转换成八进制

Description

用递归算法，把任一给定的十进制正整数转换成八进制数输出。 输入

Format

Input

输入一个正整数，表示需要转换的十进制数。

Output

输出一个正整数，表示转换之后的八进制的数。

Samples

输入数据 1

15

Copy

输出数据 1

17

思路：最淳朴的进制转换，把2改成8就可以了………… 

代码

#include <bits/stdc++.h>
using namespace std;
int main() 
{
    long long n,sum=0,b=1,x=1;
    cin>>n;
    while(n!=0) 
    {
        sum+=(n%8)*x;
        n/=8;
        x*=10;
    }
    cout<<sum<<endl;
    return 0;
}
