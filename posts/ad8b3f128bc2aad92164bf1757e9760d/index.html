js调用其他js文件

要在 JavaScript 中调用其他 JavaScript 文件，你可以使用 HTML <script> 标签来引入其他 JavaScript 文件。例如，假设你有一个名为 "main.js" 的 JavaScript 文件，要在 HTML 文件中调用它，你可以这样写：

<script src="main.js"></script>
