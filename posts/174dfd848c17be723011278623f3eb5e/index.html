vim: command not found 绝妙解决方案

出现 command not found 的原因是没有安装 xxx

mac 系统有预装 vim，直接可使用，可以通过 vim --version 来查看版本等信息。

不建议删除或更新系统预装的Vim，推荐新安装Vim的Mac实现--MacVim。

出现错误的原因

在安装软件修改环境变量时，不小心把环境变量删除或者搞乱了，导致vim不可使用。 

两个解决方案

1. 安装MacVim

教程地址：https://zhuanlan.zhihu.com/p/64024294

（ 我没有亲自安装过，仅提供上述地址作参考。）

2. 不安装，仅临时使用vim

（ 本人亲测好使。）

# 第一步：执行临时使用vim命令
export PATH=/usr/bin:/usr/sbin:/bin:/sbin:/usr/X11R6/bin
# 第二步：正常使用即可哈哈
