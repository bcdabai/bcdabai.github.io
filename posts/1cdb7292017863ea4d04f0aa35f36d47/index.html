vue 获取HTTP Session传值后端

在main.js里new vue 前写下

axios.defaults.withCredentials = true

HTTP Session 值在浏览器  --设置--高级--内容设置--cookie--查看所有cookie与网站数据--里请求的地址里

 

其中内容为加密内容，是base64加密。

其作用保持持续通话，可以用在后台仿重复提交。
