simulink MATLABFunction模块中实时函数调用函数的使用

样例
function Predyy = matlabceshi(input, Time_s)
input1 = input;
Time_s1 = Time_s;
Predyy = ee(input1) + mm(Time_s1);

end
上面是主要部分，下面是被调用部分
function A = ee(input1)
A = input1 * 100;
end
function B = mm(Time_s1)
B = Time_s1 * 100;
end

模型中函数功能调用时需要修改成matlabceshi(u(1),u(2))
