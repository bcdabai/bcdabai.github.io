泛型引入 入门

编辑两个类：

class Dog{
    private String name;
    private int age;

    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

}

class Cat{
    private String name;
    private int age;

    public Cat(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

无泛型时可能出现问题：

 1、没有toString方法，就需要向下转型访问get方法，如果需要很多数据的话，重复性较高
 2、在输入Dog类中数据时，若参入其他类，在编译阶段不会报错
    引入泛型：解决以上两个问题
public class Fan {
    public static void main(String[] args) {
        List list = new ArrayList();
        list.add(new Dog("刮刮乐", 3));
        list.add(new Dog("瓜蛋", 4));

        list.add(new Cat("汤", 4));//运行才会报错

        for (Object obj: list){//没有toString方法，则需要向下转型，数据庞大时，代码重复性会高
            Dog dog = (Dog)obj;
            System.out.println(dog.getName() + "-" + dog.getAge());
        }       

    }
}

泛型：

public class Fan {
    public static void main(String[] args) {
        List<Dog> list = new ArrayList<Dog>();
        list.add(new Dog("刮刮乐", 3));
        list.add(new Dog("瓜蛋", 4));
        list.add(new Cat("汤", 4));//泛型编译时就会报错提示


        for (Dog dog: list){//无需向下转型
            System.out.println(dog.getName() + "-" + dog.getAge());
        }

    }
}
