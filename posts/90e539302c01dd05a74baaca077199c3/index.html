简介

shtris 是一个由 shell 脚本，参考 俄罗斯方块指南 (2009) 实现的俄罗斯方块游戏。

首次用户

  1. 使用 x shtris 即可自动下载并使用

    • 在终端运行 eval "$(curl https://get.x-cmd.com)" 即可完成 x 命令安装, 详情参考 x-cmd 官网

  2. x-cmd 提供1分钟教程，其中包含了 shtris 命令常用功能的 demo 示例，可以帮你快速上手 shtris 。

  3. 使用案例:

    # 在命令行中开始俄罗斯方块游戏
x shtris

技术特点

  • 功能丰富：支持保持队列、下一个队列、随机生成、得分系统、T-Spin/Mini T-Spin、Back-to-Back Bonus、Extended Placement/Infinite Placement/Classic Lock Down、Super/Classic Rotation System和更改起始级别等功能。
  • 可跨平台运行：支持 Linux、FreeBSD、BusyBox 和 Solaris 等 shell 环境。

竞品和相关作品
