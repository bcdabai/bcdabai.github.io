Linux下安装vim编辑器

本文参考自：https://jingyan.baidu.com/article/046a7b3efd165bf9c27fa915.html

（1）打开命令窗口，输入命令 lsb_release -a 查看当前Linux系统版本

（2）根据你的Linux版本，输入对应的如下命令，我这里是centos系统，故而输入 yum -y install vim*

  • ubuntu 系统输入命令：sudo apt-get install vim-gtk
  • centos 系统输入命令：yum -y install vim*

（3）输入后等待完成，即可自动完成安装。
