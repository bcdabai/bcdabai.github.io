git代理（配合xshell）

假设已经设置好xshell代理端口1080



# 设置代理
git config --global http.proxy 'socks5://127.0.0.1:1088'
git config --global https.proxy 'socks5://127.0.0.1:1088'


# 清除代理
git config --global --unset http.proxy
git config --global --unset https.proxy


能直接访问后，把代理配置去掉就可以了
