NOIP2006开心的金明（普及组T2）————动态规划，01背包

题解：本题主要考查动态规划，属于01背包。
代码如下：

#include<iostream>
#include<algorithm>
using namespace std;
int total,n,i;
int dp[505000],m[550],y[550];
int main()
{
    cin>>total>>n;
    for(i=1;i<=n;i++)
    {
    cin>>m[i]>>y[i];
    y[i]=y[i]*m[i];
    }
    for(i=1;i<=n;i++)
    for(int c=total;c>=1;c--)
    {
        if(c>=m[i])
        {
            dp[c]=max(dp[c],dp[c-m[i]]+y[i]);
        }
    }
    cout<<dp[total];
    return 0;
}

ps：NOIP2005采药（普及组T3）与NOIP2001装箱问题（普及组T4）与本题相似就不写了。
