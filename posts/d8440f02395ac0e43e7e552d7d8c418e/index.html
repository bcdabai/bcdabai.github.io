zookeeper web管理工具

https://github.com/alibaba/canal/wiki/Zookeeper-QuickStart

  1. 首先安装maven 此处就不说了
  2. git clone https://github.com/DeemOpen/zkui.git
  3. cd zkui/
  4. mvn clean install
  5. nohup java -jar target/zkui-2.0-SNAPSHOT-jar-with-dependencies.jar &
  6. http://ip:9090/
