RadioButton、JFXRadioButton获取选中的值

RadioButton 是所属 JavaFX 单选框组件，JFXRadioButton 是所属 JFoenix单选框组件。

在使用这些组件时，建议配合 ToggleGroup 标签使用，以下以 JFXRadioButton 为示例：

<JFXRadioButton selected="true" text="选项一" userData="1" styleClass="custom-jfx-radio-button">
	<toggleGroup>
		<ToggleGroup fx:id="toggleGroup"/>
	</toggleGroup>
</JFXRadioButton>
<JFXRadioButton selected="false" text="选项二" userData="2" toggleGroup="$toggleGroup"/>
<JFXRadioButton selected="false" text="选项三" userData="3" toggleGroup="$toggleGroup"/>

获取 JFXRadioButton 选中的值，可以使用以下方式：

@FXML
private ToggleGroup toggleGroup;

public void getRadioSelectedValue() {
	Toggle selectedToggle = toggleGroup.getSelectedToggle();
	Object userData = selectedToggle.getUserData();
    System.out.println(userData);
}
