linux 安装lua

新建目录 放置压缩包

cd /usr/local
mkdir server

下载压缩包 解压文件

curl -R -O http://www.lua.org/ftp/lua-5.3.5.tar.gz
tar zxf lua-5.3.5.tar.gz

进入lua目录 进行安装

cd lua-5.3.5
make linux test
