MDK AC5和AC6是什么？在KEIL5中添加和选择ARMCC版本

前言

看视频有UP主提到“AC5”“AC6”这样的词，一开始有些不理解，原来他说的是ARMCC版本。

keil自带的是ARMCC5，由于ARMCC5已经停止维护了，很多开发者会选择ARMCC6。

在维护公司“成年往事”项目可能就会遇到新KEIL+旧版本编译器的需求。

步骤

  • 到ARM官网下载所需版本的编译器
  • 安装编译器到KEILv5\ARM\xxx。
  • 到KEIL -> Project -> Manage -> Manage Project Items -> Folders/Extensions添加刚安装的编译器。

此处就可以看到ArmCC版本。

  • 到KEIL -> Project -> Options for Target 'xxx' -> Target选择新添加的编译器。
