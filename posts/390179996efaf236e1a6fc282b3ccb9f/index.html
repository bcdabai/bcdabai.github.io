Macbook上Gitlab下载代码

项目在gitlab的http请求的url：

　　http://gitlab.xxx.com/test.git

用以下方式请求不需要再输入用户名和密码

git clone  http://用户名:密码@gitlab.xxx.com/test.git

 

密码不要包含@/等特殊符号，会跟后面的@gitlab.com冲突，导致clone失败
