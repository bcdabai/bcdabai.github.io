问题描述

第一次pull项目，运行npm run dev时报错：sh: vite: command not found

原因分析

查看了package.json，发现是有vite的。
没有安装依赖导致的；

解决方案

执行npm i重新安装依赖，之后再执行npm run build，再执行 npm run dev 可以正常运行了。
