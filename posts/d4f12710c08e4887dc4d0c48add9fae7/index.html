eNSP华为路由器交换机基本查询指令

  1. 查看接口信息——display ip interface brief
  2. 查看路由表——display ip routing-table
  3. 查看vlanif的IP地址配置——display ip interface brief
  4. 查看switch接口配置模式——display port vlan
  5. 查看ospf邻居建立详情——display ospf peer brief
  6. 查看ospf的链路状态数据库——display ospf lsdb
  7. 查看地址池的信息——display ip pool
  8. 查看DHCP中继信息——display dhcp relay all
  9. 查看STP的端口角色信息——display stp brief
  10. 查看ospf的路由表——display ip routing-table protocol ospf
