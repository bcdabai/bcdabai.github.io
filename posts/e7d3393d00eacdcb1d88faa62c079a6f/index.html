Wdows server 2003 ipv6下IP和 IIS的 WEB/ FTP设置

一、Windows 2003 配置ipv6
1.         打开运行 cmd->netsh
2.         添加IP：interface ipv6 add address “本地连接” 2001：da8：7008：12：：15
3.         显示IP：interface ipv6 show address
4.         显示DNS：interface ipv6 show DNS
注意：
1)        DNS服务器必须设置在IPV4 的IP地址上，IPV6的地址不支持DNS的部署；
二、站点设置
1.         网站地址选：全部未指定
2.         每个站点设置一个并至多一个完整的主机头，如[url]www.baidu.com[/url]
三、DNS设置
Dns 全名 [url]www.test.cn[/url] （站点主机头名）
添加正向解析的主区域
主机名
www（可以为空）
区域文件名
test.cn
（如果主机名为空则用此域名访问）
四、WINdows下支持ipv6的FTP SERVER下载网址
Xlight 2.5 正式版 
五、FTP设置
1.         FTP只使用IP地址（ipv6的）
2.         在DNS服务器做域名和地址的对应关系
