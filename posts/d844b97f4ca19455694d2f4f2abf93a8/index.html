1062：最高的分数


时间限制: 1000 ms         内存限制: 65536 KB
提交数: 103077     通过数: 75401

【题目描述】

孙老师讲授的《计算概论》这门课期中考试刚刚结束，他想知道考试中取得的最高分数。因为人数比较多，他觉得这件事情交给计算机来做比较方便。你能帮孙老师解决这个问题吗？

【输入】

输入两行，第一行为整数n（1 <= n < 100），表示参加这次考试的人数.第二行是这n个学生的成绩，相邻两个数之间用单个空格隔开。所有成绩均为0到100之间的整数。

【输出】

输出一个整数，即最高的成绩。

【输入样例】

5
85 78 90 99 60

【输出样例】

99

【思路分析】

  1. 首先，读取输入的第一行整数 n，表示考试参加人数。
  2. 接着，读取输入的第二行成绩，使用循环从第一个学生的成绩开始逐个读取，将当前的成绩存储到变量 a 中。
  3. 在循环中，比较当前成绩 a 和最高成绩 max 的大小关系：
    • 如果 a 大于 max，则更新 max 的值为 a。
    • 如果 a 不大于 max，则继续循环。
  4. 循环结束后，输出最高成绩 max。

【示例代码】

#include<bits/stdc++.h> 
using namespace std;
int main()
{
    // 声明变量 n, a 和 max，分别表示参加考试人数、当前成绩和最高成绩
	long long a, n, max=0;
	cin >> n; // 读取考试的参加人数
	for(int i=1; i<=n; i++)
	{
		cin >> a; // 读取当前学生的成绩
		if(a > max) 
		{
			max = a; // 更新最高成绩
		}
	}
	cout << max << endl; // 输出最高成绩
}
