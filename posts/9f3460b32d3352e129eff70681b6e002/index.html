ArcGIS Pro 使用总结

 

1、简介

ArcGIS目前有两个主要桌面应用程序是 ArcMap 和 ArcGIS Pro，且二者均为 ArcGIS Desktop 的一部分。ArcGIS Pro是ESRI公司最新的一款桌面GIS产品，在 2D 和 3D 模式下创建并编辑、轻松自动执行任务和工作流、实现信息共享和组织内协作并最大程度地利用现代硬件。

ArcGIS Pro是为新一代Web GIS平台，面向GIS专业人士（例如GIS工程师、GIS科研人员、地理设计人员、地理数据分析师等），全新打造的一款具有高效、强大生产力的桌面应用程序。ArcGIS Pro除了良好的继承了传统桌面软件（ArcMap）的强大的数据管理、制图、空间分析等能力，还具有其独有的特色功能，例如二三维一体化、矢量切片制作及发布、任务工作流等。 并且ArcGIS Pro整合了ArcGlobe, ArcScene, ArcCatalog, City Engine等软件的功能，集成于一款软件，使得制作与开发更加便捷。ArcGIS Pro 是真正的纯 64 位、多线程应用程序，ArcGIS Pro的运算及渲染效果更加强大，是 GIS 专业人员的主要应用程序。并能够与ArcMap在同一环境下工作，不存在环境冲突。

3、 许可

3.1 简述

ArcGIS Desktop 许可包含以下 5 个桌面 GIS 应用程序：

  • ArcGIS Pro
  • ArcMap
  • ArcCatalog
  • ArcScene
  • ArcGlobe

这些应用程序不是单独提供的；它们只能作为 ArcGIS Desktop 的一部分。

3.2 许可方式

  1. 指定用户许可（只限ArcGIS Pro产品）
  2. 浮动版许可
  3. 单机版许可

咨询过ESRI的商务人员，ESRI 目前商用的只卖单机版的授权； 授权类型分标准版和高级版（有点贵），可以打7折。

3.3 许可级别

  1. 基础
  2. 标准
  3. 高级

对于各个级别对应的功能范围请参考以下网址：

http://pro.arcgis.com/zh-cn/pro-app/get-started/license-levels.htm

 

3.4 许可期限

永久许可和短期许可是 ArcGIS Desktop 许可期限的两种可用模型。

永久许可不会过期。这意味着许可中包含的所有桌面应用程序可以无限期地使用。但是，需要与 Esri 维护计划保持同步，才能使您的 ArcGIS Online 组织保持活动状态，并获得产品更新和技术支持。

短期许可必须每年更新一次。在许可期限内，您的 ArcGIS Online 组织处于活动状态，并可获得产品更新和技术支持。

 

3.5 扩展模块产品许可

ArcGIS Desktop 扩展模块产品可无缝集成桌面应用程序以提供专用功能。扩展模块产品许可允许您将扩展模块与其适用的所有 ArcGIS Desktop 应用程序配合使用。例如，Spatial Analyst 许可允许您将扩展模块与 ArcMap 和 ArcGIS Pro 配合使用。

5.2 用户界面

  • 5.2.1 功能区

ArcGIS Pro 将使用跨应用程序窗口顶部的水平功能区以将功能显示和组织为一系列选项卡。

共有两种类型的选项卡：始终可见的核心选项卡，以及根据需要显示或隐藏的上下文选项卡。工程、插入、分析、视图和共享选项卡为核心选项卡。附加的地图或布局选项卡的显示取决于活动视图。

 

5.2.2 视图

视图是一个用来表示应用程序主要工作区并提供数据可视化的窗口。视图可以是地图、场景、表格、布局或图表。可以同时打开多个视图， 但在任何给定时间，只能激活其中一个视图。

 

 

5.2.3 窗格

窗格是一个可停靠的窗口，包含除功能区上的可用命令之外的命令和设置。常用窗格包括内容窗格（地图内容的列表）、目录窗格（工程中的项目清单以及用于管理这些项目的命令）和地理处理窗格（随 ArcGIS Pro 安装的地理处理工具箱列表）。

内容窗格显示当前激活的地图视图的内容，内容表中图层的展示方式有：按绘制顺序列出、按数据源列出、按选择列出、按编辑列出、按捕捉列出等。

5.5 GP工具

5.5.1 简介

地理处理是用于处理地理数据和相关数据的框架和工具集。大量的地理处理工具可用于执行空间分析或以自动化方式管理 GIS 数据。

 

 

5.5.2 GP工具类型

 

 

5.5.3 自定义GP工具

可以使用工程中随附的自定义地理处理工具，或构建并使用您自己的自定义工具。ArcGIS Pro 支持使用 ModelBuilder 或 Python 构建的自定义地理处理工具（脚本工具和 Python 工具箱工具）。

ArcGIS Pro 不支持使用 ArcObjects 构建的自定义地理处理工具。

 

 

5.5.4 GP服务

地理处理服务是另一种地理处理工具。要使用地理处理服务，您必须先将 ArcGIS Server 连接至服务器。这些工具的使用方法与任何其他地理处理工具都非常相似，唯一的区别是该工具需要在 ArcGIS Server 上执行。要开始使用地理处理服务，请先添加 ArcGIS Server 用户连接，然后展开目录窗格中服务器节点下的服务器连接，双击任意工具在地理处理窗格中将其打开。

 

5.6 布局与出图

5.6.1 简介

页面布局（通常简称为布局）是在虚拟页面上组织的地图元素的集合，旨在用于地图打印。常见的地图元素包括一个或多个地图框（每个地图框都含有一组有序的地图图层）、比例尺、指北针、地图标题、描述性文本和图例。为提供地理参考，可以添加格网或经纬网。
