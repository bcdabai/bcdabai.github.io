用C语言如何求四个数中的最大值

在 C 语言中，可以使用如下方法来求四个数中的最大值：

  1. 使用 if 语句：
int max(int a, int b, int c, int d)
{
    int m = a;
    if (b > m) m = b;
    if (c > m) m = c;
    if (d > m) m = d;
    return m;
}
