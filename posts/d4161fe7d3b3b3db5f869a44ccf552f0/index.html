安装Python版本的cairo模块：Pycairo

背景

1.折腾：

【记录】折腾Python中的一个绘制图表的模块：pycha

的过程中，需要用到cairo。

所以，就去找cairo。

但是在其主页中：

http://www.cairographics.org/

找了半天，还是没有找到，如何在Windows中安装cairo的，尤其是for python的。

2.后来搜索

python cairo

倒是找到了对应的Pycairo，结果去看，还就是刚才主页里面的：

http://cairographics.org/pycairo/

然后就可以去下载并安装了。

 

下载Pycairo

我这里是Python 2.7的，所以下载对应的：

py2cairo-1.10.0 for python 2.x

得到py2cairo-1.10.0.tar.bz2，解压后得到文件夹py2cairo-1.10.0。

 

安装Pycairo

1。参考：

D:\tmp\dev_tools\python\pycairo\py2cairo-1.10.0\INSTALL

的说明，去：

?
1
2
3
4
5
6
7
D:\tmp\dev_tools\python\pycairo\py2cairo-1.10.0>. /waf --help
'.' is not recognized as an internal or external  command ,
operable program or batch  file .
 
D:\tmp\dev_tools\python\pycairo\py2cairo-1.10.0>waf --help
'waf' is not recognized as an internal or external  command ,
operable program or batch  file .

很明显，此处waf是针对linux环境的。没法在此处的win7中使用。

2.知道需要编译后才能安装的。但是更知道，很是麻烦的。

3.后来看到这个：

How do you install PyCairo (Cairo for Python) on Windows?

也知道了，本身就没法此处的VS2010去编译的：

Pycairo can’t compile on Win32/VisualStudio

4.但是人家给出了windows的安装文件了，可以去：

http://www.lfd.uci.edu/~gohlke/pythonlibs/#pycairo

下载到，我此处

python2.7+win7 x64

对应的：

pycairo-1.10.0.win-amd64-py2.7.‌exe

http://www.lfd.uci.edu/~gohlke/pythonlibs/e8d7a9t8/pycairo-1.10.0.win-amd64-py2.7.exe

5.下载好了pycairo-1.10.0.win-amd64-py2.7.exe，双击去安装。

如此就安装完毕了。

 

接着就可以继续去折腾：

【记录】折腾Python中的一个绘制图表的模块：pycha

了。
