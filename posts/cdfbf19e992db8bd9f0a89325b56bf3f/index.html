vue项目netWork地址无法访问问题解决方案

1.配置vue.config.js

  devServer: {
    host: "0.0.0.0",
    public: "172.201.10.26:1888", // 此处是自己电脑IP地址！
    port: 1888,
    https: false,
    disableHostCheck: true,
    open: false, // 配置自动启动浏览器
}

 2.修改host为“0.0.0.0”，public：改为本机ip地址


（1）本机ip查询方法 win+R 后输入cmd弹出命令行

 （2）命令行中输入ipconfig后回车，找到IPv4，复制地址 

 

3.重新运行vue项目，如果出现网络无法访问状态，请确认配置端口号是否相同
