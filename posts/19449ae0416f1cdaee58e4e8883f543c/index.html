VMware使用ubuntu安装增强功能实现自动缩放

    • 1.下载 VMware Tools
    • 2.安装tool

1.下载 VMware Tools

1.需要先弹出DVD
2.虚拟机-安装VMware Tools
进入终端
3.把media下的VMware压缩包拷贝到home/下
4.去home下解压

2.安装tool

进入vmware-tools-distrib

sudo ./vmware-install.pl

根据提示yes,然后一直回车
