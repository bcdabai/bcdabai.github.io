如果你像拥有类似Mac的操作界面，或者需要将当前正常使用的软件图标缩放在右上角。等等一些扩展功能就需要了解GNOME的优化器。

1 gnome-tweaks

GNOME Tweaks Tool（GNOME优化工具），也被称为GNOME调整或简单的调整。GNOME Tweaks 是可以调整 GNOME 的高级选项，帮助你设置GNOME。

1.1 安装

(base) pang@pang-HP:/opt/deepinwine/tools$ sudo apt install gnome-tweaks
正在读取软件包列表... 完成
正在分析软件包的依赖关系树... 完成
正在读取状态信息... 完成                 
gnome-tweaks 已经是最新版 (40.0-3)。
下列软件包是自动安装的并且现在不需要了：
  gir1.2-keybinder-3.0 libkeybinder-3.0-0 libqt5concurrent5 python3-configobj python3-psutil
使用'sudo apt autoremove'来卸载它(它们)。

启动：

gnome-tweaks

优化器中可以进行一些常用优化设置

1.2 扩展安装

其他几个安装

sudo apt install gnome-tweak-tool gnome-shell-extensions chrome-gnome-shell

谷歌浏览器安装GNOME Shell integration插件，安装后，浏览器右上角，就会多出来一个小脚印哦～，下次直接点它。

Gnome插件地址，可以通过此网址搜索一些优化扩展插件，并通过谷歌插件下载安装并打开

1.3 手工安装扩展插件

  1. 选择需要下载的组件，并选择对应的版本；
  2. 下载后解压；
  3. 将解压后的目录移到：~/.local/share/gnome-shell/extensions/ 下；
  4. 打开对应扩展目录下的metadata.json文件，查看uuid；
  5. 重命名刚才移过来的目录名为uuid中的内容；
  6. 关闭并重新打开gnome-tweaks；
  7. 如果仍然没有刷出来，使用alt + F2调出gnome-shell的命令行，并输入r后回车；然后重复步骤6；

4.8 indicator-sysmonitorr

indicator-sysmonitorr这个软件，这个软件可以在顶栏显示cpu、内存、网速、cpu温度等信息，很方便,显示效果如下：

安装方法：

sudo add-apt-repository ppa:fossfreedom/indicator-sysmonitor
sudo apt-get update
sudo apt-get install -y indicator-sysmonitor

在程序中搜索sys就能找到System Monitor Indicator这个软件，点击打开就可以显示在顶栏。
在顶栏点击，打开Preferences就可以设置自启和显示样式，还可以设置多长时间刷新一次。

4.10 Harddisk LED

https://extensions.gnome.org/extension/988/harddisk-led/

4.11 Load Average

平均负载表示在三个时间间隔内获取的Linux平均负载。

https://extensions.gnome.org/extension/1381/load-average/

4.12 Uptime Indicator

正常运行时间指示器显示系统正常运行时间； 单击时，它会显示系统启动的日期和时间。

https://extensions.gnome.org/extension/508/uptime-indicator/

4.13 Sound Input & Output Device Chooser

您的系统可能有多个音频设备用于输入和输出。 例如，我的笔记本电脑有内置扬声器，有时我使用无线蓝牙扬声器。 声音输入和输出设备选择器扩展将您的声音设备列表添加到系统菜单，以便您可以快速选择要使用的设备。

https://extensions.gnome.org/extension/906/sound-output-device-chooser/

4.14 Drop Down Terminal

同为Opensource.com作家Scott Nesbitt推荐了以下两个扩展。 第一个是Drop Down Terminal ，通过按下某个键可以使终端窗口从顶部面板下拉； 默认值是Tab上方的键； 在我的键盘上，这是波浪号（〜）字符。 下拉终端具有用于自定义透明度，高度，激活击键和其他配置的设置菜单。

https://extensions.gnome.org/extension/442/drop-down-terminal/

4.15 Todo.txt

Todo.txt将菜单添加到顶部面板，以维护用于Todo.txt任务跟踪的文件。 您可以从菜单中添加或删除任务，也可以将其标记为已完成。

https://extensions.gnome.org/extension/570/todotxt/

4.16 Removable Drive Menu

Opensource.com编辑器Seth Kenlon建议使用“ 可移动驱动器菜单” 。 它提供了一个下拉菜单，用于管理可移动媒体，例如USB拇指驱动器。 从扩展名的菜单中，您可以访问驱动器的文件并将其弹出。 该菜单仅在插入可移动媒体时出现。

https://extensions.gnome.org/extension/7/removable-drive-menu/

4.17 Vitals

一瞥您的计算机的温度、电压、风扇速度、内存使用情况、处理器负载、系统资源、网络速度和存储统计信息。

https://extensions.gnome.org/extension/1460/vitals/

4.18 Activities Configurator

配置活动按钮和顶部面板。选择一个图标。更改文字。禁用“热角”或设置“热角阈值”。设置面板背景颜色和透明度以及更多功能来增强您的桌面。右键单击图标或文本以启动 GS Extension Prefs。

同类：Hide Activities Button

https://extensions.gnome.org/extension/358/activities-configurator/

4.18 Extensions

从顶部面板的菜单轻松 启用 / 禁用 / 设置 GNOME shell 扩展。

同类：Extension List

https://extensions.gnome.org/extension/1036/extensions/

4.19 Backslide

墙纸幻灯片。在预定义的时间段内自动切换壁纸。

同类软件：Variety

https://extensions.gnome.org/extension/543/backslide/

4.20 Lock keys

https://extensions.gnome.org/extension/36/lock-keys/
