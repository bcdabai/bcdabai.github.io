题目描述
输入三个整数，整数之间由一个空格分隔。把第二个输入的整数输出。

输入格式
只有一行，共三个整数，整数之间由一个空格分隔。

输出格式
只有一行，一个整数，即输入的第二个整数。

#include<bits/stdc++.h>
using namespace std;
int main()
{
	long a,b,c;
	cin>>a>>b>>c;
	cout<<b<<endl;
	return 0;
}
