“sudo pip：找不到命令” 怎么解决

1：先检查python版本

python --version


2：然后根据python版本下载相应的pip

sudo apt-get install python3-pip

(不同版本的python改变后面数字就行）

3：下载完成后再对pip进行一个升级

pip install --upgrade pip
