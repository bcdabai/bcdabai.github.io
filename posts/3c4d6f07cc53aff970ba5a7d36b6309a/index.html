Microsoft Visual Studio 命令行 编译 cl link rc

就像编译Java源程序需要设置环境变量一样，使用微软的Visual C++同样需要设置环境变量。
例如，在我的电脑里，
cl.exe 和link.exe在C:\Program Files\Microsoft Visual Studio\VC98\Bin下，
rc.exe在C:\Program Files\Microsoft Visual Studio\Common\MSDev98\Bin下
将这两个目录添加到环境变量path中，
然后打开命令行cmd，分别测试cl、link、rc命令，如果有正确结果就说明设置成功。

cl命令举例
cl hello.c
/out:hello.exe
hello.obj
