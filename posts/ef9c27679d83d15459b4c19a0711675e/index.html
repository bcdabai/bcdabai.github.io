ubuntu安装gnome以及gnome-tweak-tool

sudo apt-get install gnome-shell         #安装gnome桌面窗口管理程序
sudo apt install gnome-tweak-tool        #安装管理工具
sudo apt install gnome-shell-extensions  #扩展管理

gnome扩展下载地址：https://extensions.gnome.org/

下载方法："一起来找茬"

 

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

使用的扩展说明：

     1：Battery Status电源按百分比显示

     2：Dash to Dock

 

3：Coverflow Alt-Tab

 

4：Drop Down Terminal

按照默认，按下“~”会在屏幕上方弹出终端，很方便
