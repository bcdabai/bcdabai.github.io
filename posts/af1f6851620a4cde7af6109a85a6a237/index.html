Centos7安装wget

官网 https://www.gnu.org/software/wget/

环境 CentOS7

yum 安装

sudo yum install wget

这个报错，查资料是说，网络不通

ping 了下，返回 Name or service not known ，从错误看应该是没有配置DNS

配置DNS参考https://www.cnblogs.com/hans_gis/p/10639187.html

sudo vi /etc/resolv.conf

修改后

再次ping

可以了，再次执行安装命令sudo yum install wget

安装成功~
