android.os.Build详解

android.os.Build是安卓的一个类，获取安卓的一些系统信息

android.os.Build.VERSION.RELEASE  获取当前系统版本号字符串：4.4.2

android.os.Build.MODEL       获取手机型号

android.os.Build.VERSION.SDK_INT   获取当前系统版本号数字：19

android.os.Build.BRAND         手机定制商      

  1. Build.BOARD // 主板  
  2. Build.CPU_ABI // cpu指令集  
  3. Build.DEVICE // 设备参数  
  4. Build.DISPLAY // 显示屏参数  
  5. Build.FINGERPRINT // 硬件名称  
  6. Build.HOST  
  7. Build.ID // 修订版本列表  
  8. Build.MANUFACTURER // 硬件制造商  
  9. Build.MODEL // 版本  
  10. Build.PRODUCT // 手机制造商  
  11. Build.TAGS // 描述build的标签  
  12. Build.TIME  
  13. Build.TYPE // builder类型  
  14. Build.USER
