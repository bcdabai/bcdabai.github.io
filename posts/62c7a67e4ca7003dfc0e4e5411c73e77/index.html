编程练习2:找两个数中最大者

题目：本题要求对两个整数a和b，输出其中较大的数。函数接口定义：max (int a,int b);

解析：需要使用判定语句if(a>b)，如果a是大者就将a赋值给Max

代码：

#include<stdio.h>
int max(int a,int b)
{
    int max;
    if(a>b)
        max=a;
    else max=b;
    return  max;

}
int main()
{
    int a,b;
    scanf("%d %d",&a,&b);
    max(a,b);
    printf("max=%d",max(a,b));
    return 0;
}
