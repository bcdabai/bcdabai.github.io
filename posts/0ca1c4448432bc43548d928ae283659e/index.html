Docker容器中编辑文件时提示：vi: command not found

Docker创建好容器后，有时需要进入容器编辑相应的文件，但是进入容器后使用vi或者vim命令时提示：vi: command not found
产生问题的原因

容器中没有安装vi或者vim

解决办法

apt-get update   #更新软件列表
apt-get install vim  #下载

我这里使用的centos7操作系统，ubantu操作系统，使用yum下载来代替apt-get
