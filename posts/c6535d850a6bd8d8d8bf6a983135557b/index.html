前后端分离项目—通用后台管理系统（Vue + SpringBoot）【七】

通用后台管理系统

该系统主要结合Element组件编辑前端，后端实现业务逻辑，组成一个简单的前后端分离项目。

该部分动态背景功能的实现
使用vanta.js
效果：

代码

通过ref绑定

<div ref="vantaRef"></div>

import * as THREE from 'three'
import BIRDS from 'vanta/src/vanta.birds'
export default {
  name: 'App',
  mounted() {
    this.vantaEffect = BIRDS({
      el: this.$refs.vantaRef,
      THREE: THREE,
      wingSpan: 34,
      backgroundColor:0xccfef4,
    })
  },
  beforeDestroy() {
    if (this.vantaEffect) {
      this.vantaEffect.destroy()
    }
  },

}
