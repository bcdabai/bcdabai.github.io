函数的多种定义和调用方式

1、函数的定义方式

（1）函数发声明方式function关键字（命名函数）
function fn(){ }
（2）函数表达式（匿名函数）
let fun=function(){}
（3）new Function()
let fn=new Function('参数1','参数2','函数体');
fn();
  • Function 里面的函数都必须是字符串格式
  • 第三种方式效率低，较少使用
  • 所有的函数都是Function的实例对象
  • 函数也是对象

2、函数的调用方式

（1）普通函数
function fn(){
		console.log(1);
		}
fn();
fn.call();
（2）对象的方法
let o={
	sayHi:function(){
			console.log(1);
			}
	}
o.sayHi();
（3）构造函数
function Star(){ }
new Star();
（4）绑定事件函数
btn.onclick=function(){}
//触发
（5）定时器函数
setInterval(function(){},1000)
//时间触发 一秒钟触发一次
（6）立即执行函数
(function(){
	console.log(1)})();
//自动调用
