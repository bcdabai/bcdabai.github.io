flume系列之：监控JMX reporter


flume指标监控设置官方文档：

  • https://flume.apache.org/FlumeUserGuide.html#monitoring

通过使用flume-env.sh 在JAVA_OPTS 环境变量中指定JMX 参数来启用JMX 报告，例如

export JAVA_OPTS="$JAVA_OPTS -Xms1024m -Xmx2048m 
-Djava.security.auth.login.config=/moudle/apache-flume-1.9.0-bin/conf/flume_jass.conf 
-Dcom.sun.management.jmxremote 
-Dcom.sun.management.jmxremote.port=5455 
-Dcom.sun.management.jmxremote.authenticate=false 
-Dcom.sun.management.jmxremote.ssl=false"
  • 禁用密码验证，默认情况下启用远程监控的密码验证。要禁用它，请在启动 Java VM 时设置以下系统属性。com.sun.management.jmxremote.authenticate=false
  • 禁用 SSL，要在远程监控时禁用 SSL，您必须在启动 Java VM 时设置以下系统属性。com.sun.management.jmxremote.ssl=false
