js 正则表达式判断数字和数字范围

判断数字的正则表达式：

/^\d+$/　　 //非负整数(正整数 + 0)

/^[0-9]*[1-9][0-9]*$/　　 //正整数

/^((-\d+)|(0+))$/　　 //非正整数(负整数 + 0)

/^-[0-9]*[1-9][0-9]*$/　　//负整数

/^-?\d+$/　　　　 //整数

/^\d+(\.\d+)?$/　　 //非负浮点数(正浮点数 + 0)

/^(([0-9]+\.[0-9]*[1-9][0-9]*)|([0-9]*[1-9][0-9]*\.[0-9]+)|([0-9]*[1-9][0-9]*))$/　　//正浮点数

/^((-\d+(\.\d+)?)|(0+(\.0+)?))$/　　//非正浮点数(负浮点数 + 0)

/^(-(([0-9]+\.[0-9]*[1-9][0-9]*)|([0-9]*[1-9][0-9]*\.[0-9]+)|([0-9]*[1-9][0-9]*)))$/　　//负浮点数

/^(-?\d+)(\.\d+)?$/　　//浮点数

使用例：

test
var r = /^[0-9]*[1-9][0-9]*$/ ;　　//正整数

var t = 3;

var s = 3.2;

alert("整数:"+r.test(t)); //true

alert("小数:"+r.test(s)); //false
