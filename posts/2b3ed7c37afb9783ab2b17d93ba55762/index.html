Ubuntu 升级或安装时遇到 Errors were encountered while processing的解决办法

Ubuntu 在执行更新或者安装软件命令时

sudo apt upgrade

sudo apt install

遇到：Errors were encountered while processing 类似问题时

解决方法：：

cd /var/lib/dpkg

sudo mv info info.bak

sudo mkdir info

sudo apt update
