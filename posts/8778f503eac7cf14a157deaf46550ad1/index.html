ubuntu建立新用户

1.新建testuser用户

sudo adduser testuser    

2.设置root密码

sudo passwd root

3.更改sudoers编辑权限  

sudo chmod u+w /etc/sudoers

4.给testuser新增权限

sudo vim /etc/sudoers

添加一行：

testuser  ALL=(ALL:ALL) ALL

5.保存以后关闭sudoers的编辑权限

sudo chmod u-w /etc/sudoers

6.切换用户

su gerrit

7.删除用户

sudo userdel -r testuser 

ps:如果提示

删除用户出现user xxx is currently used by process xxx的

则是因为这个用户正在使用 , 我们要退出这个用户 ctrl +d(退出当前用户), 切换到其他用户上去,再执行命令
