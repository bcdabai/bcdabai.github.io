Jupyter Notebook HTTP报错 500：Internal Server Error

终端打开Jupyter, 内部服务报错500：Internal Server Error

终端返回报错内容：

 ImportError: cannot import name 'contextfilter' from 'jinja2' 
(/home/user/anaconda3/envs/myenv/lib/python3.7/site-packages/jinja2/__init__.py)

报错原因：

jupyter和当前安装的jinja2版本不兼容

可以用conda search jinja2查看 当前jupyter支持的jinja2版本只支持到3.0.3

解决方法：首先卸载当前的jinja2,再安装降版本的jinja2

conda uninstall jinja2
conda install jinja2==3.0.3

然后重新打开jupyter即可
