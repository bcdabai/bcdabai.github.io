时间轴的样式

有三种状态
例如：后台给的时间是6.23
一.6.23那天是红色的
二.6.23左边的数字包括轴线的颜色都是橘色
三.6.23右边的数字包括轴线的颜色是灰色

html

<div class="date"  ref="tabscroll">
            <ul id="tabscroll" v-if="btnShow=='1'">
                <li v-for="(item, index) in date_list" :key="index">
                    <p
                        :class="
                            item.state == 1
                                ? 'day1'
                                : item.state == 2
                                ? 'day2'
                                : 'day3'
                        "
                    >
                        {{ item.fmt_date }}
                    </p>
                    <p
                        @click="dayInfo(item)"
                        :class="
                            item.state == 1
                                ? 'point1'
                                : item.state == 2
                                ? 'point2'
                                : 'point3'
                        "
                    >
                        <span></span>
                    </p>
                    <p :class="item.state == 3 ? 'border2' : 'border1'"></p>
                </li>
            </ul>
            <ul id="btn-tabscroll" v-if="btnShow=='2'">
                <li v-for="(item, index) in date_list" :key="index">
                    <p
                        :class="
                            item.state == 1
                                ? 'day1'
                                : 'day3'
                        "
                        :id="btnDay==item.date?'btnDate1':''"
                    >
                        {{ item.fmt_date }}
                    </p>
                    <p
                        @click="dayInfo(item)"
                        :class="
                            item.state == 1
                                ? 'point1'
                                :'point3'
                        "
                        :id="btnDay==item.date?'btnPoint1':''"
                    >
                        <span></span>
                    </p>
                    <p :class="item.state == 3 ? 'border2' : 'border1'"></p>
                </li>
            </ul>
        </div>
