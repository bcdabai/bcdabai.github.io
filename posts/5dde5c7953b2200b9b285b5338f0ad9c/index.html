windows 查看mysql的错误日志

查找错误日志文件存储路径

用到的软件：everything

官网

voidtools

下载路径

https://www.voidtools.com/Everything-1.4.1.1024.x64-Setup.exe

直接点击下载即可

运行效果如下 

我们知道mysql有个配置文件是my.ini，里面配置了相关信息

我们需要先查这个文件在哪里

下图中我们看到路径在C:\ProgramData\MySQL\MySQL Server 5.7\my.ini，也直接双击打开该文件

可以看到日志相关配置

错误日志文件的路径如下

# Error Logging.
log-error="WIN-UCL3QDDK1UB.err"

我们再在everything中搜索WIN-UCL3QDDK1UB.err

直接打开即可

查看日志
