mac os cmake安装

1.下载安装程序，地址为

http://www.cmake.org/download/，下载

Unix/Linux Source (has \n line feeds) cmake-3.0.2.tar.gz

2.安装

进入到下载目录，解压缩 tar-zxvf filename

进入到解压之后目录下，可以参考README的说明，执行

sudo ./bootstrap && sudo make && sudo make install
