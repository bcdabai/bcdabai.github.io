NUC972开发板学习过程

1、搭建linux环境的过程，设置交叉编译器的环境变量过程中会出现各种问题；

解决方法：

第一步、一定要Ubuntu系统换源，换源后sudo apt-get update；

第二步、sudo  apt-get install lib32stdc++6

第三步、使用 vim编辑器编辑 bashrc 文件。

#vim /root/.bashrc
编辑/root/.bashrc 文件，在最后一行添加：
export PATH=$PATH:/usr/local/arm_linux_4.8/bin  （注意，要把arm_linux_4.8的放置到默认/usr/local/路径底下）
#source /root/.bashrc
#arm-linux-gcc -v

显示 gcc version 4.8.4(GCC)  成功。
