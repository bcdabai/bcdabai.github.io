sudo apt-get install mysql-server-5.7 报错，各种要求依赖包。

心一横，干脆全部更新得了，简单又粗暴

1.  sudo apt-get update
0x01 安装mysql 

1. sudo apt-get install mysql-server
2. sudo apt-get isntall mysql-client
3.  sudo apt-get install libmysqlclient-dev
