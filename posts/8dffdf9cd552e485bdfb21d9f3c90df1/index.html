ideapush代码报错：Push rejected: Push xxx to origin/xxx.x was rejected by remote 解决办法

解决方案如下：

1.切换到自己项目所在的目录，右键选择GIT BASH Here(也可以在ideal中操作)

git pull

git pull origin xxx

git pull origin xxx --allow-unrelated-histories

3.在idea中重新push自己的项目。

如果还不行： git push -u origin xxx -f
