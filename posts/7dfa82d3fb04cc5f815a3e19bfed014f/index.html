tt协议号服务器,TTcam协议的账号的写法

该楼层疑似违规已被系统折叠 隐藏此楼查看此楼

这几天，由于很多烧友刷了新的TTCAM协议的DM500系统，但很多烧友都觉得写入账号麻烦，其实它的写入跟OSCAM的写法差不多，我们可以通过系统的var下tuxbox下 config 的TTcam.server文件中写入账号的，文件内容可以这样写入

另提供TTcam.server文件下载，下载地址是http://good.gd/2209870.htm

# oscam.server generated automatically by Streamboard OSCAM 1.20-unstable_svn build #0

# Read more: http://streamboard.de.vu/svn/osc ... xt/oscam.server.txt

[reader]

label = cccam

enable = 0

protocol = cccam

device = 111.222.222.222.2000

user = 123

password = 123

inactivitytimeout = 100

reconnecttimeout = 2

group = 1

cccversion = 2.3.0

cccwantemu = 1

ccckeepalive = 1

cccreconnect = 1

[reader]

label = reader-1

protocol = cccam

device = abc.bbwbbs.org,18480 写入你的TT号的服务器地址及端口号，中间记得用英文的逗号隔开

user = bbw123 这里是写入你的TT号的用户名

password = xy789 这里是写入你的TT号的密码

group = 1

cccversion = 2.3.0

cccwantemu = 1

ccckeepalive = 1

cccreconnect = 0

[reader]

label = reader-2

protocol = cccam

device = yy.net,184580 这写入你平时用的CCCAM账号的服务器地址及端口号

user = bbw365 这里写入你的账号的用户名

password = 112893 这里写入你的账号的密码

group = 3

cccversion = 2.0.11

cccwantemu = 1

ccckeepalive = 1

cccttcam = 0

cccreconnect = 0

切记，CCCAM的账号与TTCAM的账号的区别就在于有没有cccttcam = 0这一行
