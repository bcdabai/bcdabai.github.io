Pyuic出现No module named ‘PyQt5.sip‘问题解决

跟着网上教程安装anaconda + pycharm + PyQt5 安装完成后，在PyCharm创建项目，随便建了一个test.ui。 然后在test.ui上右键 -> External Tools -> PyUIC， 然后就遇到一堆错误： ModuleNotFoundError: No module named

的时候报告错误：

1

No module named 'PyQt5.sip'

$ pip3 install SIP

$ pip3 install pyQt5

$ pip3 install --upgrade PyQt5

$ pip3 install PyQtWebEngine
