跨域的简单实现

利用flask启动后端服务器，前端是使用原生的js实现的。

只需要在后端响应头中增加 Access-Control-Allow-Origin 参数，如果想让所有的域都可以访问，那值设为 “*”就行。

后端代码：

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello World!', 200, [
        ("Access-Control-Allow-Origin", "*")
    ]

if __name__ == '__main__':
    app.run()
