工作记录 -- el-table 取消鼠标悬浮时行高亮效果


使用el-table组件时，会有默认事件

  • 当鼠标悬停时会高亮当前行

改变鼠标悬浮时高亮的效果

.el-table tbody tr:hover>td {
  background: greenyellow!important
}

但如果设置了stripe属性，表格会呈斑马纹两种颜色交替，设置颜色这个方法就失效了，这时候就可以禁用事件：

.el-table tbody tr { 
    pointer-events:none; 
}

用户使用鼠标单机行时，也会出现行颜色改变的情况，修改其单机的行颜色

.el-table__body tr.current-row>td {
        background: #ffb707!important;
    }
