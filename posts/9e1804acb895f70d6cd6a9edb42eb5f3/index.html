最小安装linux (centos 7) 注意事项

  1. ip地址查询
    安装工具 yum install net-tools.x86_64
  2. 彩色的编辑器
    yum install vim
  3. 传文件
    1. 检查 ssh yum list installed | grep openssh-server
    2. 没有的话 安装 yum list | grep openssh-server 安装对应版本 yum -y install **
    3. 找到/etc/ssh/ vim ssh_config 把 #Port #ListenAddress 0.0.0.0 挨着的两个 #PermitRootLogin yes #PasswordAuthentication yes 注释去除
    4. systemctl start sshd 状态查询 systemctl status sshd
    5. 查看端口监听 netstat -an | grep 22
