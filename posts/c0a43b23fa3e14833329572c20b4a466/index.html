测试代码

#include<iostream>
#include<pcl/io/io.h>
#include<pcl/io/pcd_io.h>
#include<pcl/io/ply_io.h>
#include<pcl/point_types.h>
#include<pcl/visualization/cloud_viewer.h>

int user_data;
using std::cout;
void viewerOneOff(pcl::visualization::PCLVisualizer& viewer) {
    viewer.setBackgroundColor(1.0, 0.5, 1.0);
}
int main()
{
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>);
    char strfilepath[256] = "rabbit.pcd";
    if (-1 == pcl::io::loadPCDFile(strfilepath, *cloud)) {
        cout << "error input!" << endl;
        return -1;
    }
    cout << cloud->points.size() << endl;
    pcl::visualization::CloudViewer viewer("Cloud Viewer");
    viewer.showCloud(cloud);
    viewer.runOnVisualizationThreadOnce(viewerOneOff);
    system("pause");
    return 0;
}

SuiteSparse

下载安装包并解压，在 suitesparse-metis-for-windows 目录下新建 build 文件夹
启动 cmake-gui，添加工程路径如下

Where is the source code ：D:/Automation/VS2017/suitesparse_DIR/suitesparse-metis-for-windows
Where to build the binaries ：D:/Automation/VS2017/suitesparse_DIR/suitesparse-metis-for-windows/build

    点击 “Configure”，在弾出的对话框中选择指定生成器：“Visual Studio 15 2017”，选择生成平台： “x64”。点击finish。

再次单击 “Configure”，配置成功Configuring done；单击 “Generate”，Generating done。单击Open Project，分别在debug和release模式下，右击ALL_BUILD选择“生成"，完成后再右击 INSTALL选择 “生成"。suitesparse-metis-for-windows\build\install 目录下生成了 include 和 lib 文件夹。
