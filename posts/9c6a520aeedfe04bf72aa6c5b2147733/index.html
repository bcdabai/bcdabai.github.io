FiddlerScript获取当前电脑的IP地址

可以通过如下代码在 Fiddler 脚本中获取到本地的 IP 地址：

  • 第一步，导入需要用到的类。
import System.Net;

  • 第二步，通过如下代码获取到本地 IP 地址。
// 获取本机的IP地址，通常是内网地址
var host = Dns.GetHostEntry(Dns.GetHostName());
var ip = host.AddressList[1];// AddressList 是一个数组，第一个值是IPv6地址，第二个值是IPv4地址
FiddlerObject.alert(ip);// 显示IP

注：实际上是应用了 C# 的 API。参考地址：C#**本机IP地址的4种方式。
