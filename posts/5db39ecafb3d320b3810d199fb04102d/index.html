There was a problem importing one of the Python modules required to run yum

为什么80%的码农都做不了架构师？>>>   

最近从python 2.6 升级到python2.7 导致 yum 不可用，

原因主要是yum 不支持python27 因此需要更改yum的可用路径

which yum

查看下yum的安装路径

/usr/bin/yum

vi /usr/bin/yum

更改#!/usr/bin/python 为#!/usr/bin/python2.6

更改完成之后就可以了
