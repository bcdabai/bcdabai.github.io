JavaScript 中testContent的用法

在 JavaScript 中，textContent 是一个常见的属性，用于获取或设置一个 HTML 元素及其子元素的文本内容。

要获取元素的文本内容，可以使用以下语法：

var element = document.getElementById("example"); // 获取元素
var text = element.textContent; // 获取元素的文本内容
console.log(text); // 输出文本内容

要设置元素的文本内容，可以使用以下语法：

var element = document.getElementById("example"); // 获取元素
element.textContent = "New Text"; // 设置新的文本内容

请注意，textContent 返回的是纯文本内容，不包含 HTML 标签，而且会保留换行符和空格。如果你希望获取包含 HTML 标签的内容，可以使用 innerHTML 属性。另外， textContent 是只读属性，你只能通过赋值来修改元素的文本内容。
