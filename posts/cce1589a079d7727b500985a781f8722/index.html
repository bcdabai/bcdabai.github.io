java时间戳与php时间戳转换

java时间戳是13位的 例如：javaTimeStamp=1475309160000

java -> php ，除1000，就得到10位php时间戳

phpTimeStamp=javaTimeStamp/1000

php时间戳是10位的 例如：phpTimeStamp=1475309160

php -> java ，乘1000，就得到13位java时间戳

javaTimeStamp=phpTimeStamp*1000
