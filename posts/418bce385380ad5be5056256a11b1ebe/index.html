Python中i = i + 1与i + = 1的区别

+=是对原本的实例做加1运算，l=l+[1]是对l+[1]之后重新把值赋给叫l的变量（和原来的l不同）

区别在于，一个修改数据结构本身（就地操作）b + = 1而另一个只是重新分配变量a = a + 1。

只是为了完整性

 

l = []
lst = []
lst.append(l)
print(lst)#[[]]
l += [1]
print(l)#[1]
print(lst)#[[1]]
l = l + [2]
print(l)#[1,2]
print(lst)#[[1]]
