Node直接执行ts文件

1、常规流程

node 执行 【ts 文件】 流程：
1、编写ts代码
2、编译成js代码 [命令如 ：tsc xx.ts]
3、执行js代码 [node xx.js]

2、直接执行

想要直接执行 ts 文件，需要安装如下依赖工具。
执行如下命令：

# 安装在项目中
npm install @types/node --save-dev
# 全局安装
npm install -g ts-node

直接运行ts的命令

ts-node xxx.ts
