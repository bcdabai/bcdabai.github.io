vue 3.0项目出现You may use special comments to disable some warnings.Use // eslint-disable-next-line 错误.

这个问题困扰了我好久，网上试过各种方法都解决不了，真的是操蛋了！终于试过大量错误后成功解决问题,这无非是eslint语法问题罢了。

下面来看一下 该如何解决这个问题

找到package.json 文件，把有关eslint的依赖都删掉(应该也有其他办法的）。

删除后 重新npm run serve ，项目成功跑起来了！
