【Android学习】将Android项目打包成APK

将Android项目打包成APK (app应用)

  • 在Android Studio 中打开项目
  • Build -> Generate Signed Bundle / APK ...

  • 选APK

  • 点击 Create new ...

  • 输入四次相同密码, 比如:zxc@123

  • ok , 等待打包完成!
