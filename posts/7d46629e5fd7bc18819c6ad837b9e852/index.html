nvm简介及使用，删除node

1、简介

nvm是一个node版本管理工具

2、下载

下载nvm地址

下载nvm-setup.zip => 安装在C盘

3、配置环境变量

这一步安装最新版自动就配置了，你只需要电脑的360之类设备别阻止就行

4、使用

查看版本，出来这个证明安装成功

nvm -v

5、安装nodeJS

你需要什么版本，就指定什么版本号（PS：这样下载可能有点慢，挂机/耐心等待）

nvm install 版本

6、使用新安装的node

nvm use 版本

7、常用命令

nvm install stable   //安装最新版 node

nvm install [node版本号]   //安装指定版本的node

nvm ls // 查看已安装版本方便切换

nvm use [node版本号]   //切换到指定版本的node

nvm alias default [node版本号] //设置默认版本

8、删除已安装node

nvm uninstall 版本


提示你手动去nvm安装目录删除
