C语言实现N个数中寻找最大值

一、练习

要求输入N个数，找出其中的最大值，并输出。

比如N是5的时候，输入5个数：1 2 3 4 5，最大值是5。

二、思路

这个练习比较简单，基本思路就是用一个变量存储最大值，然后用户一边输入一边与这个最大值比较，大就更新它。

三、C程序参考代码如下：

#include <stdio.h>
#define LEN 5

int main()
{
	int array[LEN] = {0};
	int index = 0;
	int max = 0;
	
	printf("请输入5个整数: ");
	for(index = 0; index < LEN; index++)
	{
		scanf("%d", &array[index]);
		if(array[index] > max)
		{
			max = array[index];
		}
	}
	printf("最大值是: %d\n", max);
	system("pause");
	
    return 0;
}
