C语言中文网提供的扫雷游戏不依赖 TC 环境，不依赖第三方库，可以在 VC6.0、VS、CodeBlocks、Dev C++ 编译通过。我们提供给大家的扫雷游戏，不要求大家精通 C 语言，也不会涉及到指针的使用，只需要学会以下几个知识点即可：

  1. 会使用变量(int、char)和二维数组；

  2. 能够懂得函数的声明和使用；

  3. 掌握 C 语言的分支结构(if-else 和 switch 语句)和循环结构(while 和 fo() 循环)；

设计扫雷游戏的主要目的是让大家夯实 C 语言基础，训练编程思维，培养解决问题的思路，领略多姿多彩的 C 语言。游戏开始后，会出现一个有关扫雷游戏的介绍，同时由用户选择，游戏是否开始。用户输入 “y” 或者 “Y” 都可开始游戏，输入 “n” 或 “N” 或其他字符会直接退出游戏。如下图所示：
