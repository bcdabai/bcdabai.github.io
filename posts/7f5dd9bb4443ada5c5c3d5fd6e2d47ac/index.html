python 通过命令行传递参数

1、通过argparse模块可实现通过命令行传递参数，并设置默认值

import argparse

#初始化参数构造器
parser = argparse.ArgumentParser()
#在参数构造器中添加命令行参数
parser.add_argument('--path',type = str,default = 'C:/Users/63058/Desktop/print')
#获取命令行参数
args = parser.parse_args()
#使用获取的命令行参数
path = args.path

若不赋值则使用默认值运行
