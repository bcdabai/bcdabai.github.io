Minimum supported Gradle version is 6.1.1. Current version is 5.6.4.

Android Studio 导入项目，Gradle 同步时，提示以下错误：

Minimum supported Gradle version is 6.1.1. Current version is 5.6.4.

Please fix the project's Gradle settings.
Fix Gradle wrapper and re-import project
Open Gradle wrapper properties
Gradle settings

一 、原因：其实就是你导入的项目使用的Android gradle 插件，  与当前你的Android studio 的Android gradle 版本不对应，

导致Gradle 版本对应不上.

查看两个版本号的方式：

1. Android studio 的Android gradle 版本：

Help -> About -> Android Studio 3.5.2 (其实就是当前studio版本)

2. 项目的Android gradle依赖

// Top-level build file where you can add configuration options common to all sub-projects/modules.
buildscript {
    dependencies {
        classpath "com.android.tools.build:gradle:4.0.1"

        // NOTE: Do not place your application dependencies here; they belong
        // in the individual module build.gradle files
    }
}

二、解决方法

1. 修改 gradle-wrapper.properties  里面 的版本号 （推荐）

该文件路径为： [项目根目录]/gradle/wrapper/gradle-wrapper.properties

distributionUrl=https\://services.gradle.org/distributions/gradle-6.1.1-all.zip

**如果你的项目下没有该目录，则从别的工程拷贝一个即可

2. 如果 方法1 不行，则可尝试在：  【项目根目录]\build.gradle  修改为 当前Studio 的版本号

classpath "com.android.tools.build:gradle:3.5.2"

再重新同步即可。
