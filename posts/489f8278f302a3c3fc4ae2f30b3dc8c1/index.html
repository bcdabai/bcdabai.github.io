node+微信小程序实现商城案例

说明：
1、本人也是初次完整使用小程序，如有BUG或者不足的地方请在Issues或者本文下方留言，作者会尽快修改，谢谢！
2、本项目适合初学者或者准备自学小程序的伙伴

小程序功能：

    |-- images               // 公共图片
    |-- lib                  // 公共文件
    |-- pages                // 页面
    |   |-- index            // 首页
    |   |-- list             // 商品列表
    |   |-- details             // 商品详情
    |   |-- user             // 个人中心
    |   |-- order            // 我的订单
    |   |-- coupon           // 优惠券
    |   |-- cart             // 购物车
    |   |-- addressList      // 我的地址
    |   |-- addressAdd       // 添加地址
    |-- README.md            // 说明
