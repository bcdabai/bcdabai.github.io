找出最小的值：输入一个正整数 n, 再输入 n 个整数，输出最小 值。试编写相应程序。

分析

思考：之前我们都是在循环输入正整数，但是这里需要比较前后两个数的大小，怎么办？
我们需要在循环之外先输入一个数设置为最小数，循环里面的<=要改为<,在与循环里面输入的数比较换值找出最小数

代码

#include <stdio.h>
 
int main ()
{
	/**
	找出最小的值：输入一个正整数 n, 再输入 n 个整数，输出最小 值。试编写相应程序。 
	*/
	
	//输入一个正整数
	int n;
	scanf("%d",&n);
	
	//思考:循环输入数的话，前一个数怎么与后一个数比较
	//所以我们可以把第一个数在循环之外输入，循环的<=改为<
	int num,i,min;
	scanf("%d",&num);
	min = num;
	
	for(i = 1; i < n; i++){
		scanf("%d",&num);
		//换值操作
		if(min > num){
			min = num;
		} 
	}
	
	//打印n个数中最小的数
	printf("min = %d\n",min); 
	 
	return 0; 
}
