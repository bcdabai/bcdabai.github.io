基础（一）C++语言 第一章 C++语言入门 1004：字符三角形

【题目描述】

给定一个字符，用它构造一个底边长5个字符，高3个字符的等腰字符三角形。

【输入】

输入只有一行，包含一个字符。

【输出】

该字符构成的等腰三角形，底边长5个字符，高3个字符。

【输入样例】

*

【输出样例】

  *
 ***
*****
#include <iostream>
#include <cstdio> 
using namespace std;
int main()
{
	char x;
	scanf("%c",&x);//输入给定1个字符，注意"&"不能漏 
	printf("  %c\n",x); //输出1个字符，先输出2个空格
	printf(" %c%c%c\n",x,x,x);
	printf("%c%c%c%c%c\n",x,x,x,x,x); 
	return 0; 
}
