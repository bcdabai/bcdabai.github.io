Vue Can‘t resolve ‘sass-loader‘ in...错误的解决办法

1.需要引入 sass-loader 和 node-sass 包

npm install sass-loader -D
npm install node-sass -D

2.如果第二个包引入报错，则使用

cnpm install node-sass -D

3.如果使用cpm命令报“不是内部命令”，则
先 cnpm -v 看看有没有安装cnpm，如果没有安装则使用 npm install cnpm -g --registry=https://registry.npm.taobao.org 进行安装，安装完成后就可以使用cnpm命令了
