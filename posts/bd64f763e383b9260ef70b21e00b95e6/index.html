【C语言】用递归实现字符串逆序的曲折之路

先看一下简洁版的。
1.成功版

#include<stdio.h>  
#include<windows.h>
char s[1000];
void reverse(char *s)
{

    if (*s)
    {
        reverse(s+1);
        printf("%c", *s);
    }
    return;     
}

int main()
{
    gets_s(s, 100);
    reverse(s);
    printf("\n");

    system("pause");
    return 0;
}
