git覆盖仓库

1.远程覆盖本地

git fetch --all

git reset --hard  origin master

git pull  origin 

2.本地覆盖远程

git push --force origin master

如果出现了仓库提交混乱的情况，Git报错：Another git process seems to be running in this repository

就执行rm -f .git/index.lock，即删除掉这个文件
