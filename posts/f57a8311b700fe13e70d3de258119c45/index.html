docker 操作所有容器的命令

列出所有的容器 ID

docker ps -aq

 

停止所有的容器

docker stop $(docker ps -aq)

 

删除所有的容器

docker stop $(docker ps -aq)

 

删除所有的镜像

docker image prune --force --all

或者

docker image prune -f -a

 删除所有停止的容器

docker container prune -f
