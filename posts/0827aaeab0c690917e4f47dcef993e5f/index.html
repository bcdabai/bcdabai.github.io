react实现修改url参数，但是不刷新页面，超级好用的hooks

useUrlState

使用起来特别简单

第一步

npm i @ahooksjs/use-url-state -S

第二步使用

import useUrlState from '@ahooksjs/use-url-state';

const [state, setState] = useUrlState({ 路径参数1: '1', 路径参数2: '10' });

state：即你当前路径上的参数

setState：即修改你当前路径上的参数

官网地址贴这里了 贼好用
