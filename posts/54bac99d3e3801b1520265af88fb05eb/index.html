docker-compose加入已经存在的网络

查看已存在的网络

docker network ls

在需要加入同一局域网的容器 .yml或yaml文件中添加下面的代码：

networks:
  default:
    external:
      name:  NetworkName

在高版本的compose中会提示：WARN[0000] network default: network.external.name is deprecated. Please set network.name with external: true

意思是上述方式已经弃用，应使用下述方式

networks:
  default:
    external: true
    name: mq-cluster-test
