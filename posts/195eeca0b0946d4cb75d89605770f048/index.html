如何解决Chrome禁止发送不安全的内网网络请求(ERR_FAILED)(How to fix Chrome block your insecure private network requests)

chrome更新到94之后，会自动阻止发送不安全的内网请求

比如，你从 http://www.website.com 发送请求到 http://192.168.0.1 的时候，就会被阻止，并且报这样的错:

net::ERR_FAILED

Network Error

 

 (Ensure private network requests are made from secure contexts，其实就是让你把服务改成https的)

解决办法，就是关闭这个新特性，不要让它禁用

①打开Chrome实验功能

在Chrome浏览器地址栏输入，chrome://flags/  ，访问。

或者在地址栏输入：chrome://flags/#block-insecure-private-network-requests

②找到 Block insecure private network requests. 并且禁用。

 

③重启浏览器即可
