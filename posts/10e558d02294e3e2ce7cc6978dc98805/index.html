超简单·Ubuntu系统Git安装

安装git只需要在终端terminal的base环境下执行三个命令即可：

1.更新

sudo apt update

2.下载git

sudo apt install git

3.查看git版本，查看是否安装成功

git --version

然后就安装完毕了

在其他环境下也可以用第三条命令同时检测到git版本
