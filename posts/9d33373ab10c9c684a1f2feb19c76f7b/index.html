用C语言实现删除字符串中所有重复的字符

  1. 要删除重复字符，首先要建立一个字符数组来存放字符串，建立一个char类型的一维数组初始化为零
  2. 通过使用gets（）从键盘输入字符串，运用for循环来判断是否有重复字符
  3. 如果有重复字符，将当前位置到字符串末尾向前移动一位，覆盖当前位置，从而达到删除重复字符
  4. 在从当前位置的前一位再次进行循环，因为当前位置的字符可能还和前一个位置的字符一样（连续几个相同字符）
  5. 最后输出字符串puts（）；
#include <stdio.h>
#include <string.h>
int main(int argc, char *argv[])
{ 
    char a[100] = {0};
    int n,i,j,k;
    printf("please input a:");
    gets(a);
    puts(a);
    for(i=0;a[i];i++)
    {
        for(j=i+1;a[j];)
        {
            if(a[i] == a[j])
            {
                for(k=j;a[k];k++)
                    a[k] = a[k+1];
            }
            else
            {
                j++;
            }
        }
    }
    puts(a);
    return 0;
}
