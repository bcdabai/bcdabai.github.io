BAT019:用指定字符替换文本文档中的指定内容

二、写入批处理代码

1.右键新建的批处理文件，点击【编辑】。

​

2.在打开的记事本中写入以下代码：

@echo off
setlocal enabledelayedexpansion

REM 提示用户输入要操作的文件名称
set /p "file=请输入要操作的文件名称(包括扩展名): "

REM 去除文件名中的双引号
set "file=%file:"=%"

REM 获取文件的绝对路径
for %%i in ("%file%") do set "file=%%~fi"

echo.

REM 提示用户输入即将被替换的内容
set /p "replaced=请输入即将被替换的内容: "

echo.

REM 提示用户输入替换字符串
set /p "all=请输入替换字符串: "

REM 遍历文件的每一行，并进行替换
(for /f "delims=" %%i in ('type "%file%"') do (
set "str=%%i"
set "str=!str:%replaced%=%all%!"
echo !str!
)) > "%file%"_tmp.txt

REM 用替换后的临时文件替换原文件
move /y "%file%"_tmp.txt "%file%"

REM 打开替换后的文件
start "" "%file%"
