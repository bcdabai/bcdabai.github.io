linux下的打包和压缩

原来没怎么搞懂打包和压缩的概念。今天查资料，实际操作了一翻，终于弄明白了。

打包：一般用tar指令。把很多文件打包成一个文件夹；

压缩：可以用gzip等指令，把一个文件压成更小的文件。

一般，直接用tar指令，形成.tar, .tar.gz和.tar.bz2 等文件。

tar -cvf  file.tar  file          //只打包

tar -xvf file.tar

 

tar -czvf file.tar.gz   file  //打包后再压缩 成.tar.gz文件

tar -xzvf tar.gz

 

tar -cjvf file.tar.bz2 file    //打包后再压缩 成.tar.bz2文件

tar -xjvf file.tar.bz2
