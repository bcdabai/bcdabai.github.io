前言

  刚进入大学的新生，对电脑的使用基本是处于能打开浏览器，打开文件资源管理器，填写Word，Excel的水平，这样是不行的，做起事来手足无措。
  本文是给那些新生的一个推荐，帮助新生更好地进行学习。因为刚进大学，估计对于电脑很多都不了解，所以自己是不知道要装什么软件的。自己要善于使用搜索，在网上查找别人推荐的软件工具。
  这里就给新生罗列一些我认为对于平常的使用有较大帮助的一些软件，网站等。这些软件在平常的使用中可以帮助自己解决不少问题，将会给大学生涯带来不少的帮助。

  • Ctrl + A，全选

1. PrtSc键（全屏截图）

  屏幕快照键，全称 Print Screen， 键盘上常印有Prt Sc 或 Prt Scn 字样。是用来屏幕截图的按键。在键盘右上角的位置。
  操作方式：按下即可对整个电脑屏幕进行截图，然后便可以用 Ctrl + V 粘贴到目标位置（输入框或者Word文档等）。

2. Alt + Prt Sc快捷键

  Prt Sc按键按下是截取全屏，而Alt + Prt Sc则是截图当前活动窗口。按下快捷键后，用Ctrl + V即可粘贴到目标位置（输入框或者Word文档等）。

3. 截图工具

Windows自带截图工具，现在已经由更强大的截图和草图替代。

原截图工具的升级版，按下快捷键可以调出截图功能，默认框选矩形区域截图。


按下这个快捷键可以调出截图功能，可以

  • 矩形区域截图
  • 任意形状截图
  • 窗口截图

可以设置由PrtScn键打开屏幕截图，这样就可以框选区域截图，而不是截图全屏，截图更灵活，快来试试吧。

Windows系统自带的Office

  在大学，平常的话，一般是看PPT，写Word文档，填Excel表格。这个是最主要的了，可以说是一直要用的。Windows系统都会自带 Office，但是不一定是激活的，有试用期限，可能需要自己额外购买，也可能买电脑的时候就已经是激活了的。


  如果Office不能用，那么这里有个奇怪的东西，你猜是啥。 http://www.yishimei.cn/network/319.html

推荐WPS
  但是我还是比较推荐使用WPS，理由有下面几点：
对个人用户免费
  WPS对于个人是免费的，对企业收费
  WPS分为企业版，个人版和校园版，企业版也就是专业版，是收费的，而个人版和校园版是免费的。可以安装个人版(PC版)和校园版。

我也安装过企业版，可能是适合工作办公的吧，用起来感觉没校园版好用。

校园版

校园版地址 https://education.wps.cn/

校园版是为在校学生推出的，比较符合在校学生的需求，比较推荐。

企业版

企业版地址 https://ep.wps.cn/

  WPS只对个人使用免费，是不能在公司办公使用的。
  在公司使用需要购买企业版，也就是常说的WPS专业版。这个是企业购买的，按年付费，至少3个人，费用还是比较贵的。网上有激活的方法，但是我觉得不好用，还是装回了校园版。


云文档同步,云空间

  WPS给用户1GB的云空间，并且文档实时同步，当你打开一个新文档后，会有文件记录，并且小文件自动上传云空间，可以减少文档丢失损坏情况。

  办公助手中，可以设置同步文件夹，将文件夹中的内容实时同步到云盘，功能还是挺多的。

移动端
  WPS移动端还是很强大的一款办公软件，配置WPS云文档的实时同步，可以做到移动办公，实时处理事务。WPS也是手机必备的一款APP。


金山文档

https://www.kdocs.cn/

  直接在网页编辑和查看云空间中的云文档。这里只提供简单的编辑功能，满足基本的编辑需求。

WPS会员免费获取

  但怎么说呢，WPS有很多实用的工具，这些工具会让文档处理有质一般的提升，但是需要WPS会员才能使用。
  会员特权非常多，最重要的PDF功能，文档格式转化，文字识别，云空间升级等等。可以说，有了会员，才真正解锁了这款软件。
  WPS会员权限分为 WPS会员 和 WPS超级会员， WPS会员基本就已经解锁完所需权限了，超级会员多的东西无关紧要。
  WPS会员其实有送的，不用买。在WPS的公众号上每天签到，就能获取会员，关注就送15天会员，然后每天签到能得个1~3天的会员，想氪金也行，98块钱一年，经常打折，便宜个20块钱，这样有一段时间忘记签到了，也不用担心会员到期。


WPS会员获取方法

关注微信公众号：WPS会员

一次性

  • 关注送15天会员
  • 做任务，就是简单的功能试用和关注任务，可得11天，一次性。

长期

  • 打卡领会员：每天打卡随机送会员天数，多数是1~3天，最低是1天。非RMB玩家就是要肝，坚持打卡！

  • 签到领福利：每天签到增加云空间容量

压缩工具

  手机照片，截图等图像体积有时较大，在有些需要限制文件体积或者想减少文件大小的情况下就用得到文件压缩了。在一些写作平台上，通常就有上传图片文件大小限制。(比如，CSDN上传图片大小限制为5MB，所以大一点的就上传不了了)
  PDF文件需要上传打印，自己由Word生成的PDF，有时会因为里面包含大量插入的图片，常常会造成文件体积十分庞大。
  文件压缩分为无损压缩和有损压缩。无损压缩通过算法改变部分数据的表示形式，消除数据的冗余，而不会改变实际的内容。而有损压缩会改变实际的内容，以满足更高的压缩比，使文件体积进一步减少。
  在图像文件上，通常又分为无损压缩，视觉无损压缩和有损压缩。视觉无损压缩是有损压缩，但是人类视觉系统并不能察觉出这种图像信息量的损失。
  文件压缩仅能压缩到一定程度，压缩程度太大会造成严重失真。

https://mathpix.com/


  Mathpix Snip是一款可以从图片中识别数学公式的神器！论文里经常需要包含众多的公式，而公式的输入十分的麻烦。通过这款工具，可以直接对包含公式的PDF截图，或对草稿纸手写公式拍照，直接识别生成标准清晰的公式图片或Latex公式，极大地节省时间，且识别正确率非常之高，是理工科论文写作必不可少的一款神器。

MATLAB

  MATLAB是一种用于算法开发、数据可视化、数据分析以及数值计算的高级技术计算语言和交互式环境。除了矩阵运算、绘制函数/数据图像等常用功能外，MATLAB还可以用来创建用户界面及与调用其它语言（包括C，C++和FORTRAN）编写的程序。同时还是数学建模中的常用工具。

MuPAD
