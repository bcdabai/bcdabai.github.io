Linux 2.6.21 之前关于usb网卡ZT6688驱动的解决方法

买了块USB网卡  硬件标识是ZT6688的,

这个网卡的驱动在2.6.21.1的内核中已经提供位于/lib/modules/`uname -r`/kernel/drivers/usb/net/dm9601.ko

如果想在2.6.21之前的内核中使用ZT6688的usb网卡：

1, 下载dm9601的linux-2.6驱动http://www.davicom.com.tw/big5/download/Driver/dm9601/dm9601-2.6.tgz

2, tar -zxvf dm9601-2.6.tgz

3, cd dm9601-2.6

4, 修改dm9601.h   在#endif  /* _DEV */前面加上下面语句:
     DM9601_DEV( "ZT6688 USB Ethernet", 0x0a46, 0x6688,
                DEFAULT_GPIO_RESET )

5, make

6, cp dm9601.ko /lib/modules/`uname -r`/kernel/drviers/usb/net/

7, depmod

8, modprobe dm9601

9, tail -100f /var/log/syslog

10, 可以看到已经成功找到了ZT6688网卡了
