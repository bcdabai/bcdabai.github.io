Python tolist()用法

tolist函数的作用：将数组或矩阵转化为列表

from numpy import *
a1=[[2,3,4],[5,6,7]]#列表
a2=mat(a1)#矩阵
a2
matrix([[2, 3, 4],
        [5, 6, 7]])
a3=array(a1)#数组
a3
array([[2, 3, 4],
       [5, 6, 7]])
       
a4=a2.tolist()
a4
[[2, 3, 4], [5, 6, 7]]
a5=a3.tolist()
a5
[[2, 3, 4], [5, 6, 7]]
a4==a5
True
