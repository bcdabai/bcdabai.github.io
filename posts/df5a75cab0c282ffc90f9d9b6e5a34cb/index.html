注册账号

npm

发包

在你需要上传的包的当前文件夹打开终端,准备登录你刚刚注册(已有)的账号
登录账号：npm adduser
然后根据提示输入账号、密码、邮箱；
输入完后会给你提示，你当前的源是在哪，有的可能装cnpm那么就会是在别的源，那么就需要切换到https://registry.npmjs.org源；在下边问题统一
发布：npm publish

E403问题以及修改源

源不正确的话，需要切换源

  • 查看源：npm config get registry
  • 切换源方法一：npm config set registry https://registry.npmjs.org
  • 切换源方法二：nrm use npm

报如下错的话， 是没有验证邮箱
解决：重新去网页端，登录邮箱验证一下，就可以了
