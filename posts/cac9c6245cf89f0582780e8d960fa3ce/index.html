用递归方法编程计算Fibonacci数列的前N项。当n=1或2时，fib(n)=1当n＞2时，fib(n)=fib(n-1)+fib(n-2)。**输入格式要求：“%d“ 提示信息：“Input

#include<stdio.h>
#include<stdlib.h>
int fib(int n);
int main()
{
    int n;
    int x=1;
    printf("Input n:\n");
    scanf("%d",&n);
    for(x=1;x<=n;x++)
    {
    printf("Fib(%d)=%d\n",x,fib(x));
    }
}
int fib(int n)
{
    if(n==1||n==2)
    {
        return 1;
    }else
    {
        return fib(n-1)+fib(n-2);
    }

}
