java jwt 如何设定过期时间

用metabase时候内嵌jwt协议，java库是jjwt，一直找不到实现过期的方法，终于找到

Jwt token = JwtHelper.encode("{\"resource\":{\""+resourceType+"\": "+dashboardId+"},\"params\":{},\"iat\":"+iat+",\"exp\":"+time+"}", new MacSigner(METABASE_SECRET_KEY))
复制代码

这里一直缺少的是iat：发现token 的时间，exp：过期时间 都是timestamp秒，缺一不可

      通过源码可以知道这些参数其实都是jwt的payload，有些自定义比如resource，params，有些默认比如过期时间 exp iat.
