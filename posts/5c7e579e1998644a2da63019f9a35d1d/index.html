【报错问题】Nginx 重启时，提示 “nginx : 无法将“nginx”项识别为 cmdlet、函数、脚本文件或可运行程序的名称......”

一、版本说明

本机系统： Windows 10 (64位)
nginx:  v1.16.1 (64位)  # nginx -v 查看

 

二、出现问题

         2.1、使用 Windows PowerShell 窗口，提示错误： nginx : 无法将“nginx”项识别为 cmdlet、函数、脚本文件或可运行程序的名称。请检查名称的拼写，如果包括路径，请确保路径正确，然后再试一次。   

 

三、切换为 cmd 命令窗运行命令即可          

         3.1、什么都不用改动，直接切换成 cmd 命令窗运行
