google地图聚合markerclusterer.js修改聚合点显示内容

MarkerClusterer.prototype.calculator_ = function(markers, numStyles) {
  var index = 0;
  var count = markers.length;
  var dv = count;
  while (dv !== 0) {
    dv = parseInt(dv / 10, 10);
    index++;
  }

  index = Math.min(index, numStyles);
  return {
    text: count,
    index: index
  };
};

找到markerclusterer.js文件的这段代码，text内容为聚合点显示内容，此处聚合点显示count，即聚合marker个数，可根据需要修改；index结合styles配置决定聚合点图片显示，此处index根据聚合marker个数改变，可根据需要修改。
