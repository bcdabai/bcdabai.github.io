JS输出为[object object]取值问题

问题：页面跳转传参

传参前数据能打印出来，传参后显示[object object]且无法取值

解决

传递

接收后端返回的json对象通常是一个字符串类型的object

所以
通过 JSON.stringify() 把 JavaScript 对象转换为字符串。

my.navigateTo({
      url: `/pages/component/receive?clickItem=${JSON.stringify(e.currentTarget.dataset.clickItem)}`
    });

接收

在从 web 服务器接收数据时，数据永远是字符串。

通过 JSON.parse() 解析数据，这些数据会成为 JavaScript 对象。

 onLoad(e) {
    console.log(e);
    this.setData({
      clickItem: JSON.parse(e.clickItem)
    });
  },
