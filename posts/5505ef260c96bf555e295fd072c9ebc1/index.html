pycharm安装于配置pyqt5环境

1、依次使用pip命令下载pyqt包（这里使用的国内下载源，可适当调整）

pip install pyqt5 -i https://pypi.douban.com/simple/  

pip install pyqt5-tools -i https://pypi.douban.com/simple/ 

pip install pyqt5designer -i https://pypi.douban.com/simple/

2、下载完成后，找到qtdesigner.exe、pyuic5.exe、pyrcc5.exe三个工具所在的路径

一般在python安装环境下的子目录中，以下是作者的实际路径，可以参考查找。

D:\python3.9\Lib\site-packages\QtDesigner\designer.exe

D:\python3.9\Scripts\pyuic5.exe

D:\python3.9\Scripts\pyrcc5.exe
