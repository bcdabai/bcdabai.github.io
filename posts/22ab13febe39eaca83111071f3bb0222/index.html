centos安装java8(一键安装)

安装JDK
# yum install java-1.8.0-openjdk* 安装java1.8.0所有程序

yum install -y java-1.8.0-openjdk-devel.x86_64

注意:输入确认(y)直到控制台返回Complete安装成功

查看java版本信息

# java -version

输入javac java命令后显示帮助信息就算是安装成功了

使用yum安装环境变量自动就配好了
