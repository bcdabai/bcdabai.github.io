C语言---将一个数组中的值按逆序重新存放

将一个数组中的值按逆序重新存放。

例如，原来顺序为8,6,5,4,1，要求改为1,4,5,6,8.
代码如下：

#include<stdio.h>
int main()
{
    int a[5]={0},i,j;
    for(i=0;i<5;i++){
        scanf("%d",&a[i]);
    }
    for(j=4;j>=0;j--){
        printf("%d ",a[j]);
    }
    return 0;
}
