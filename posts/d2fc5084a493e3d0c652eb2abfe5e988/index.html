解决linux终端按退格键无法回退的问题

问题：

在下载某些依赖或者安装包后发现终端乱序，输入命令时按退格键不能回退反而向后加空格；并且某些快捷键无法使用等问题；

原因

终端依赖的包ncurses-base被删除；

解决办法

终端输入：

sudo apt install ncurses-base
