Python:（使用matplotlib画图）在一张图上画多条曲线、设置坐标轴名称、设置坐标轴上的刻度、设置坐标轴长度

1、在一张图上画多条曲线

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(-3,3,50)  #设置横轴的取值点

y1 = 2 * x + 1
#曲线1
y2 = x**2
#曲线2
plt.figure(num=3,figsize=(8,5))
plt.plot(x,y2)
plt.plot(x,y1,color='red',linewidth=1,linestyle='--')

plt.show()

2、设置坐标轴的长度
