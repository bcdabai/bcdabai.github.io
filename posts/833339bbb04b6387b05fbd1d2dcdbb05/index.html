在idea中添加mapper.xml文件模板

文章目录

  • 1、创建模板
  • 2. 创建mapper.xml文件

1、创建模板

  1. 打开创建模板页面
    位置：File-》Settings-》Editor-》File and Code Templates


    点击+号，新建模板，填入Name（随意）、Extension（必须为xml）

    填入模板内容：
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="">
   
</mapper>

最后 点击 apply 、ok 大功告成

2. 创建mapper.xml文件



填入映射mapper接口文件即可编写sql。
