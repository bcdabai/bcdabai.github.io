Mac M1 使用sdkman安装指定版本jdk，并切换默认版本

一、查询JDK

打开终端，执行查询命令，查看可用的jdk版本，下图红框中的就是具体版本

sdk list java

二、安装SDK

使用下面的命令安装指定版本jdk，版本号获取方法：步骤一👆🏻

sdk install java <version>

三、切换默认JDK

sdk default java <version>

当前使用的jdk如图，可以看到现在默认使用的是jdk17，现在使用👆🏻的命令，切换默认jdk为8

执行命令：

sdk default java 8.0.352-zulu


再使用命令查看当前默认的jdk：

sdk list java


此时已经切换成功。

四、卸载JDK

sdk uninstall java <version>

sdk 其它命令可使用 sdk help 查看
