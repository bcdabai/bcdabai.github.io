c++学习之new int()和new int[]的区别

new int[] 是创建一个int型数组，数组大小是在[]中指定，例如：

int * p = new int[3]; //申请一个动态整型数组，数组的长度为[]中的值


new int()是创建一个int型数，并且用()括号中的数据进行初始化,例如：
int *p = new int(10); // p指向一个值为10的int数。
