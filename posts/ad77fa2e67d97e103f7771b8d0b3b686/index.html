Winpcap的安装使用方法

 

1．下载winpcap的安装包，程序员开发包。

2．安装winpcap。

3．解压开发包，在工程中加入winpcap的include和lib（在VC的tools-option-目录的include和lib，Eclipse的project-properties-c/c++ build-settings中的GCC C Compiler-Includes和MinGW C Linker-Libraries）。可能需要添加ws2_32.lib。

4. 在你的程序中加入#include <pcap.h>, #include <remote-ext.h>，然后VC在project-setting-CC++中加入预定义宏:WPCAP,HAVE_REMOTE，在project-setting-连接导入wpcap.lib库；Eclipse在project-properties-c/c++ build-settings中的GCC C Compiler-Symbols中中加入预定义宏:WPCAP,HAVE_REMOTE，在project-properties-c/c++build-settings-MinGW C Linker-Libraries中添加wpcap和packet。

5．enjoy
