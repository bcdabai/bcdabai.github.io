【Ubuntu 22.10】克隆虚拟机修改ip

查看网关

这里我采用的是VMware软件 -> 编辑 -> 虚拟网络编辑器

修改配置文件

sudo vim /etc/netplan/01-network-manager-all.yaml
# Let NetworkManager manage all devices on this system
network:
  version: 2
  renderer: NetworkManager
  # 新添加内容
  ethernets:
    ens33:
      dhcp4: false
      optional: true
      addresses: [192.168.217.128/24]
      # 网关
      routes:
        - to: default
          via: 192.168.217.2
      nameservers:
        addresses:
          - 114.114.114.114
          - 8.8.8.8

更新配置

sudo netplan apply
