解决IE浏览器下报错: Promise未定义

解决IE浏览器下报错: Promise未定义

yarn add babel-polyfill

或者

npm install babel-polyfill --save

在加载Promise之前或公共js中引入

import Promise from 'babel-polyfill';
// 解决promise 在ie中未定义的问题
if (!window.Promise) {
 window.Promise = Promise;
}
