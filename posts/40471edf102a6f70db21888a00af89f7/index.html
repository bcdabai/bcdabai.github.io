泛型的定义语法为：[访问符] class 类名<类型参数列表>{  //类体....   }

例如public class Gen<T>  { private T data;  ......  }

在调用泛型的时候指明类型即可Gen<String> g=new Gen<String>("hello");

泛型的通配符是“？”

泛型可以在其定义的时候指定类型参数的上届   [访问符] class 类名<类型参数 extends 父类（即上届）>{  //类体....   }

也可以在使用泛型时指定其上届     泛型类<? extends 父类（上届）>

而下届只能在使用时指定    

例如  public static void My(Gen <? super String> g)
