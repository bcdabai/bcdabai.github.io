django-redis-sentinel 哨兵配置

django使用redis哨兵指南

哨兵作用：通过配置redis的哨兵机制，可以为用户提供一个可用的redis服务。

话不多说，直接怼！

pip install django-redis-sentinel

在django的settting中的配置

 CACHES = {
        "default": {
            "BACKEND": "django_redis.cache.RedisCache",
            "LOCATION": "redis_master/10.2.158.41:26379,10.2.158.42:2639/0"
            "OPTIONS": {
                "PASSWORD": 's3cret_passw0rd!',
                "CLIENT_CLASS": "django_redis_sentinel.SentinelClient",
            }
        }
    }
    
    # "redis_master/10.2.158.41:26379,10.2.158.42:2639/0"
    # 注释： redis_master： 是redis哨兵配置中的名称
    # 10.2.158.41:26379  哨兵的ip port
