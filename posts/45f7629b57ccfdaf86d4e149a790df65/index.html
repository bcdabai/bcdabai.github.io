链表每一个节点的数据类型该如何表示

 1 #include<stdio.h>
 2 
 3 typedef struct node{
 4     int data;//数据域
 5     struct node * pNext;//指针域,指向跟本身一样的数据类型（struct node）
 6 }NODE,*PNODE;//NODE==struct node,
 7 PNODE==struct node *
 8 
 9 int main(){ 
10     return 0;
11 }

 

链表的分类:

　　单链表

　　双链表：

　　　　每一个节点有两个指针域，左边指针域指向前面的，右边指针域指向后面的

　　循环链表：

　　　　能通过任何一个节点找到其他所有的节点，最后一个节点的指针域指向第一个节点

　　非循环链表
