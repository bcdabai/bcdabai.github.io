前言

本教程所需前提本地Git安装，以及Gitee账号注册

一、Git配置

1.Git安装教程

链接: Git安装教程

2.IDEA配置Git

  • File->Setting->Version Control->Git
  • 选择Git安装目录


二、Gitee配置

1.安装Gitee插件

  • File->Settings->Plugins->Gitee
  • 安装后重启

2.注册Gitee账号

  • 注册Gitee账号，绑定邮箱
  • 链接: Gitee

3.配置Gitee相关账号信息

  • File->Setting->Version Control->Gitee

  • Add account

  • 填写Gitee账号（邮箱）和密码

4.进行代码绑定

  • 在Gitee云端新建仓库

  • 选择项目->VCS->Import into Version Control->Create Git Repository

  • 选择当前文件（不要选择父级目录）

  • 如果项目出现这些红色文件，说明项目和git绑定成功，处在版本控制内

5.Gitee代码上传下载

  • 复制云端仓库地址

  • 本地项目和云端地址绑定

  • 选中项目->Git->Repository->Remotes

  • ＋->粘贴Gitee仓库地址

1.云端文件下载到本地

  1. 选中项目->Git->Repository->Pull

  2. 将云端项目拉取到本地

  3. 云端文件拉取到本地

2.本地文件上传云端

  1. 选中当前项目->Git->Add
    • 选中当前项目->Git->Commit Directory
    • 尽量选中src文件目录下的和pom.xml文件提交，其他文件提交可能会导致冲突
