nvm下载安装教程

前言

nvm 官网地址：https://nvm.uihtm.com

一、nvm 下载

进入 nvm github 地址，下载最新版本：https://github.com/coreybutler/nvm-windows/releases

  1. 点击选择当前最新版本。

  1. 滑动到底部，点击 nvm-setup.exe 下载安装文件。

  1. 接受，下一步。
  2. 选择 nvm 安装目录，下一步。
  3. 选择 nodejs 安装目录，下一步。
  4. 点击安装。
  5. 进入 cmd，输入 nvm 指令查看版本。安装成功则如下显示。

nvm arch：显示 node 是运行在32位还是64位。
nvm install 18.17.0：安装 node 18.17.0， version是特定版本也可以是最新稳定版本 latest。可选参数 arch 指定安装32位还是64位版本，默认是系统位数。可以添加 --insecure 绕过远程服务器的 SSL。install 可简化为 i。
nvm list [available] ：显示已安装的列表。可选参数 available，显示可安装的所有版本。list 可简化为 ls。
nvm on ：开启 node.js 版本管理。
nvm off ：关闭 node.js 版本管理。
nvm proxy [url] ：设置下载代理。不加可选参数url，显示当前代理。将 url 设置为 none 则移除代理。
nvm node_mirror [url] ：设置 node 镜像。默认是 https://nodejs.org/dist/。如果不写 url，则使用默认 url。设置后可至安装目录 settings.txt 文件查看，也可直接在该文件操作。
nvm npm_mirror [url] ：设置npm镜像 https://github.com/npm/cli/archive/。如果不写 url，则使用默认 url。设置后可至安装目录 settings.txt 文件查看，也可直接在该文件操作。
nvm uninstall 18.17.0 ：卸载指定版本 node 18.17.0。
nvm use [version] [arch] ：使用制定版本 node。可指定32/64位。
nvm root [path] ：设置存储不同版本 node 的目录。如果未设置，默认使用当前目录。
nvm version ：显示 nvm 版本。version 可简化为 v。

二、 安装 node.js

  1. nvm list available 或 nvm ls available 显示可下载版本的部分列表。
  2. nvm install 18.17.0 或 nvm i 18.17.0 安装对应版本 。( 安装时可以在上面看到 node.js 、npm 相应的版本号，不建议安装最新版本。)
  3. nvm list 或 nvm ls 查看目前已经安装的版本。
  4. nvm use 18.17.0 使用指定 node 版本。nvm list 查看使用情况，前面带 * 号则使用成功。

三、nvm 常见问题

  1. 如果下载 node 过慢，请更换国内镜像源, 在 nvm 的安装路径下，找到 settings.txt，设置 node_mirro 与 npm_mirror 为国内镜像地址。
node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/
  1. 当 cmd 输入 nvm 提示没有找不到这个命令，检查 nvm 的环境变量配置是否正确。
