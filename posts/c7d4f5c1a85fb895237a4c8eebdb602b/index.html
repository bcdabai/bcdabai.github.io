成绩转换 switch全部列出

本题要求编写程序将一个百分制成绩转换为五分制成绩。转换规则：

  • 大于等于90分为A；
  • 小于90且大于等于80为B；
  • 小于80且大于等于70为C；
  • 小于70且大于等于60为D；
  • 小于60为E。

输入格式:

输入在一行中给出一个整数的百分制成绩。

输出格式:

在一行中输出对应的五分制成绩。

输入样例:

90

输出样例:

A
#include<stdio.h>
#include <stdlib.h>
int main(){
    int a;
    scanf("%i",&a);
    switch(a){
        case 90 ... 100: printf("A");break;
        case 80 ... 89: printf("B");break;
        case 70 ... 79: printf("C");break;
        case 60 ... 69: printf("D");break;
        case 0 ... 59: printf("E");break;
            default :printf("输入错误");break;
    }
}
