树莓派 查看CPU实时频率、占用率

树莓派上默认的任务管理器只能查看CPU使用率，无法查看运行频率等信息，功能太弱，没有windows的好使。

找了好半天，都说是通过

cat /proc/cpuinfo
cat /proc/meminfo

来查看CPU或者内存信息，但是里面并没有啥东西。

然后看到的就是一些C的代码，或者python的代码来获取的。难道没有一个简单的方法吗？

于是乎看到了这个，太实用了。

安装一下：

sudo pip install s-tui

然后启动运行：

sudo s-tui

效果如下：

非常的漂亮~~~

另外如果要看网速还有一款不错的工具：

sudo apt-get install speedometer

启动运行，查看有线网卡如下：

speedometer -r eth0 -t eth0

如果是用的无线网卡，如下：

speedometer -r wlan0 -t wlan0
