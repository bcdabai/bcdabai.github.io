【Go】依赖包管理

查看已安装的包（带着后边的三个点）

# 列出所有已安装的包
go list ...

下载包

# 不指定版本下载
go get github.com/IBM/sarama

删除包

 go clean -i github.com/IBM/sarama

修改版本

# 删除包
go clean -i github.com/IBM/sarama 
# 指定版本下载
go get github.com/IBM/sarama@v1.42.0
