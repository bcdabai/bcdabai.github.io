Android模块化编程——WebView使用之清理缓存

清理缓存：
    在WebView使用中可能会遇到一个问题，那就是加载了很多页面，退不出去了，你需要一直按返回键很多次。体验非常差。这是你除了按照上篇博客，在特定网页捕获返回键。还可以清理缓存。 
   
   

    
    

     
     	// 清理缓存
    
    

    
    

     
     	private void clearHistory() {
    
    

    
    

     
     		mWebView.postDelayed(new Runnable() {
    
    

    
    

     
     			@Override
    
    

    
    

     
     			public void run() {
    
    

    
    

     
     				mWebView.clearHistory();
    
    

    
    

     
     			}
    
    

    
    

     
     		}, 1000);
    
    

    
    

     
     	}
