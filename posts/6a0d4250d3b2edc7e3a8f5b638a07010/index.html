二、安装方法

1，下载对应的软件包

2，确保本地Java环境是1.8

3，解压后，进入logstash文件文件夹内，运行如下命令

bin/logstash -e 'input { stdin { } } output { stdout {} }'

启动完成后，

终端提示如下界面，则表示启动成功

然后继续输入hello world，即如下的显示

即表明logstash安装成功。

三、指定conf文件启动

启动命令为

bin/logstash -f logstash_elasticsearch.conf
