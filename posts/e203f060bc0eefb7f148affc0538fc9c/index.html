出现这个问题一般都是CSS或style版本过高的问题

如果你是从“最全最新Vue、Vuejs教程，从入门到精通”过来的

那么建议你使用以下版本

npm install css-loader@3.6.0

或者

npm install --save-dev style-loader@1.00

最终也是打包成功！
