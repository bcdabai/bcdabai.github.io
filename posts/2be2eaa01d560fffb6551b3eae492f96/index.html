计算机ping 的快捷键,巧用电脑快捷键

本人很喜欢琢磨快捷键,这里总结一些,有些东西其实看起来不高深，但其实作用很大，计算机是我们每天都要用的，

用惯一个快捷键有时能节省很多时间:

&nbsp

一般的ctrl + c,ctrl + x,alt + tab程序切换,F5刷新等等太常见了,建议首先把这些键用好.

&nbsp

1)&nbsp&nbsp&nbsp windows 组合键

Windows键就是左下角在ctrl 和 alt之间的那个键,也许很多人从来没用过吧,J

&nbsp&nbsp windows + R: 运行框,敲入notepad,cmd,ping,calc,control这些都是经常要用到的,凡是在系统变量path中的exe程序都可以运行.

&nbsp&nbsp windows + E: 资源管理器

&nbsp &nbspwindows + D: 桌面和程序间的切换

&nbsp

&nbsp&nbsp&nbsp 如果不幸用笔记本电脑而且没有Windows键,那可以按ctrl + esc,看看啥效果!

&nbsp

2) shift + F10: 强烈推荐用熟该快捷键,右键功能

&nbsp

4) alt + enter: 强烈推荐啊,察看文件属性.在查看目录大小和修改只读属性时很是有用

&nbsp

5) alt + v + t: 强烈推荐使用.按类型排序,当然也可以按大小日期排序，不过用的最多的是这个，当看到一个文件很杂很多的目录，

&nbsp&nbsp 首先用这个键排一下序. (Atl + v) 【内容提要】 + H 是

&nbsp

6) (alt + f) + w + f : 新建文件夹,也可以(alt + f) + w + m 然后enter确认,一般是新建Word文档.

&nbsp

至于在一些常用的应用程序中快捷键就更多了,多尝试下没准就能发现特别好用的快捷键,非常欢迎各位推荐一些好用的快捷键给我!
