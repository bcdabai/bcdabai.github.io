5.三种动态创建元素的区别

1.document.write

直接将内容写入页面。但是文档执行完毕，则会导致页面重绘。

2.innerHtml

如果使用拼接字符串进行创建，则效率极低。建议采用数组拼接的方式创建。缺点是结构不清晰。

  
 <script>
        var arr = [];
        for (var i = 0; i < 10; i++) {
            arr.push("<p>我是</p>")
        }
        console.log(arr);
    </script>

3.createElement()创建多个元素时，效率稍微比innerHTML数组创建的方式低一点，但是结构清晰
