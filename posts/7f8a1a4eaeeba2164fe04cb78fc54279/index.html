把新建的项目提到gitLab的一个组内

1.首先要配置SSHKey

2.在组内新建一个project，（里面可以设置项目的存储路径以及项目名称）

3.选中上传文件夹右键选择Git Bash Here

 

4.git设置

 5.项目上传

git init
git remote add origin git@gitlab.com:mizyy/test1.git（自己的git地址）
git add .
git commit -m "Initial commit"
git push -u origin master
