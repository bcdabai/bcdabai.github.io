npm install报错：This version of npm is compatible with lockfileVersion@1, but package-lock.json was generated for lockfileVersion@2. I’ll try to do my best with it!

下依赖的时候提示这个，但是仍然可以继续下依赖，但下的过程中，有的提示报错了，因为已经将解决了，报错的截图找不到了。

解决办法：

当前电脑安装的npm是lockfileVersion@1的，但package-lock.json是源于lockfileVersion@2的。项目中使用的某个包只能用特定版本的npm下载，所以会报错导致npm install失败。这时升级一下npm就可以解决了：

npm install -g npm
