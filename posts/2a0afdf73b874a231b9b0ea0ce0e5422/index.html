centos升级最新内核

最新内核地址：https://cdn.kernel.org

  1. 导入key，安装elrepo的yum源
rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org
yum -y install https://www.elrepo.org/elrepo-release-7.el7.elrepo.noarch.rpm
  1. 安装最新稳定版内核
yum -y --enablerepo=elrepo-kernel install kernel-ml* perf
  1. 查看当前的内核默认启动顺序，最新的内核在第一个位置，就是0
awk -F\' '$1=="menuentry " {print $2}' /etc/grub2.cfg
  1. 设置当前默认的启动顺序, 并重启
grub2-set-default 0
reboot
  1. 查看内核版本
uname -r   			  查看当前内核版本
rpm -q kernel	      查看系统中已安装的内核
  1. 将老内核一个个删除
yum -y remove kernel-3.10.0-* kernel-headers-3.10.0-*
reboot
