使用

初始化版本库

相当于git init 指定文件

发现全部文件都显示红色

  1. 红色：未进入缓存区（未被追踪）
  2. 绿色：代码进入缓存区
  3. 黑色：已提交（不需要在提交）
  4. 蓝色：提交后修改的文件

代码添加到暂存区

git add 文件操作 （这里就指定了所有的文件）
