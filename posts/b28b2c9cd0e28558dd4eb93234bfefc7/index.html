oracle 查看和修改global_name 的方法

1、查看global_names属性

1）、查看global_names属性是否开启：
SQL> show parameter global_name;

NAME                                 TYPE        VALUE

------------------------------------ ----------- ------------------------------

global_names                         boolean     FALSE

2）、查看golbal_names的值：
SQL> select * from global_name;
GLOBAL_NAME

2、修改global_name

1)、启用global_names:
SQL> alter system set global_names=true;

2)、修改global_name的方法一：
SQL> ALTER DATABASE RENAME GLOBAL_NAME TO ORCL;
数据库已更改。

SQL> select * from global_name;
GLOBAL_NAME
--------------------------------------------------------------------------------
orcl

3)、修改global_name的方法二：（不建议）
SQL> update props$ set value$ = 'orcl' where name = 'GLOBAL_DB_NAME';
已更新 1 行。

SQL> select * from global_name;
GLOBAL_NAME
--------------------------------------------------------------------------------
orcl
