ubuntu18.04 安装KVM

KVM是liunx下的虚拟机，下面记录一下我自己的经历吧，方便自己日后忘记能快速上手。

1.首先不用多说啦，既然是虚拟机，当然要自己的机器支持虚拟技术，重启机器进入biso，把是否支持虚拟技术改成enable。

2.养成好的习惯，在linux终端安装东西的时候都要sudo apt-get update一下，然后安装KVM依赖。

sudo apt-get install qemu-kvm
sudo apt-get install qemu
sudo apt-get install virt-manager
sudo apt-get install virt-viewer 
sudo apt-get install libvirt-bin 
sudo apt-get install bridge-utils

3.验证下，终端键入‘kvm-ok’，出现下面的界面，就说明已经装好了。

4. 终端键入“sudo virt-manager”,进入图形界面。
