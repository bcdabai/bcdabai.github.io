docker容器中提示bash: vi: command not found的处理方法

docker容器中使用vi或vim提示bash: vi: command not found的处理方法

    • 原因
    • 处理

原因

没有安装vi编译器

处理

使用如下两个命令

apt-get update
apt-get install vim

apt-get update：第一个命令是更新软件列表

apt-get install vim：第二个是安装vim编辑器

这里使用的是centos操作系统，如果是ubantu操作系统，请将apt-get换成yum即可
