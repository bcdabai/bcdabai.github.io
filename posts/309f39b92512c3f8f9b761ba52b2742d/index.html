从一个git仓库拷贝到另一个git仓库，保留git提交记录

1. 从原地址克隆一份裸版本库

–bare 创建的克隆版本库都不包含工作区，直接就是版本库的内容，这样的版本库称为裸版本库。

git clone --bare http:  //....(原始old仓库地址)

2、进入克隆下来的目录

cd project.git（project即为你的项目名称）

3、以镜像推送的方式上传代码到新的仓库地址

git push --mirror http：//...(目标new仓库地址)

4、删除旧仓库的克隆版

cd ..
 
rm -rf project.git
