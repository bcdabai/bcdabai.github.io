Vue中如何获取滚动条滚动的距离

1.先在App组件中定义一个获取滚动条距离的方法,方法名称是handleScroll

methods:{
   handleScroll() {
      let top = Math.floor(document.documentElement.scrollTop || document.body.scrollTop);
      console.log(top);

    },

}

2.然后在mounted中，设置监听事件

mounted() {
    window.addEventListener("scroll", this.handleScroll);
},
