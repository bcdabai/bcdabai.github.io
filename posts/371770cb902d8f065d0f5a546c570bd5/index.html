在pages下，添加了echarts/echarts.vue, 生成的dist里没有这个文件，是因为要在 pages.json里添加一下就好了：

{
        "path": "pages/echarts/echarts",
        "style": {
            "navigationBarTitleText": "曲线图"
        }
    },
