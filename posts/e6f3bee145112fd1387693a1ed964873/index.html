2021-5-8【CCF】【1031. 等腰三角形】

题目描述
输入一个正整数n，输出高为n的由*组成的等腰三角形。

输入
输入一个正整数。

输出
输出高为n的由*组成的等腰三角形。

样例输入
3

样例输出

  *
 ***
*****

数据范围限制
1<=n<=20

#include<bits/stdc++.h>
using namespace std;

int main(){
	int n;
	cin>>n;
	for(int i=0;i<n;i++){
		for(int j=0;j<n-i-1;j++){
			cout<<" ";
		}
		for(int k=0;k<2*i+1;k++){
			cout<<"*";
		}
		cout<<endl;
	}
}
