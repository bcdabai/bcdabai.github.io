练习4-7 求e的近似值 (15 分)

自然常数 e 可以用级数 1+1/1!+1/2!+⋯+1/n!+⋯ 来近似计算。本题要求对给定的非负整数 n，求该级数的前 n+1 项和。

输入格式:

输入第一行中给出非负整数 n（≤1000）。

输出格式:

在一行中输出部分和的值，保留小数点后八位。

输入样例:

10

结尾无空行

输出样例:

2.71828180

结尾无空行

#include<stdio.h>

int main()
{
	int n = 0;
	scanf_s("%d", &n);
	double sum = 1;
	int i = 0;
	for (i = 1; i <=n; i++)
	{
		int add = 1;
		int j = 0;
		for (j = 1; j <= i; j++)
		{
			add *= j;
		}
		sum += 1.0 / add;
	}
	printf("%.8lf", sum);

	return 0;
}
