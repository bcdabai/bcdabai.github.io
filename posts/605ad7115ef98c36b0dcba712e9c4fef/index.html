速率

速率：数据率、数据传输率、比特率
比特 1/0 位
连接在计算机网络上的主机在数组信道上传送数据位数的速率
单位是 b/s，kb/s，Mp/s，Gb/s，Tb/s

速率
千 1kb/s = 103b/s
兆 1Mb/s = 103kb/s = 106b/s
吉 1Gb/s = 103Mb/s = 106kb/s = 109b/s
太1Tb/s = 103Gb/s = 106Mb/s = 109kb/s = 1012b/s

在速率中，103进制转换

存储容量
1Byte（字节）= 8bit（比特）
1KB = 210B = 1024B = 1024 * 8b
1MB = 210KB = 1024KB
1GB = 210MB = 1024MB
1TB = 210GB = 1024GB

在存储容量中，210也就是1024的进制转换

带宽

  1. 带宽原本指某个信号具有的频带宽度，即最高频率与最低频率之差，单位是赫兹（Hz）
  2. 计算机网络中，带宽用来表示网络的通信线路传送数据的能力，通常是指单位时间内从网络中的某一点到另一点所能通过的最高数据率。单位是“比特每秒”，b/s，kb/s，Mb/s，Gb/s。网络设备所支持的最高速度

吞吐量

表示单位时间内通过**某个网络（或信道、接口）**的数据量。
单位b/s，kb/s，Mb/s
吞吐量受网络的带宽或网络的额定速率的限制。

时延

指数据从网络的一端传送到另一端所需的时间，也叫延迟或迟延，单位是s。
发送时延（传输时延）
发送时延是指主机或路由器发送数据帧所需要的时间（从发送第一个比特开始，到最后一个比特发送完毕所需的时间）

数据长度单位：bit
发送速率单位：bit/s
发送时延单位：s

传播时延
传播时延是电磁波在新党中传播一定距离所需要花费的时间。

电磁波在自由空间中的传播速度是光速，在网络传输媒介中的传播速率会低一些。

处理时延
主机或路由器接收到分组需要花费一定时间进行处理，这段时间就是处理时延。

排队时延
分组进入路由器后要先在队列中排队等待处理，等到路由器确定了转发接口后再进行转发，这段时间就是排队时延。

总时延 = 发生时延 + 传播时延 + 处理时延 + 排队时延

对于高速网络链路，我们提高的仅仅是数据的发送速率而不是比特在链路上的传播速率
因为比特在链路的传播速率是由传播介质材料决定的，与数据的发送速率无关
提高数据的发送速率只是减少了数据的发送时延

时延带宽积

时延带宽积 = 传播时延 X 带宽
时延带宽积单位：bit
传播时延单位：s
带宽单位：bit/s

时延带宽积就表示这个管道的体积，表示这样的链路可以容纳多少个比特。

时延带宽积又称为以比特为单位的链路长度
可以简单的看做是一个装满比特的容量

往返时延RTT

从发送方发送数据开始，到发送方接收到接收方的确认，总共经历的时延。（完成一次双向交互所需的时间）
RTT包括

  1. 往返传播时延 = 传播时延 * 2
  2. 末端处理时延（一般忽略不计）

有效数据率：P25

利用率

  1. 信道利用率

  2. 网络利用率

信道利用率或网络利用率过高就会产生非常大的时延
