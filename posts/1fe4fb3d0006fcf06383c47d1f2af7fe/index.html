银河麒麟服务器操作系统扩容swap分区

1，创建10G大小的swap分区
 fallocate -l 10G /usr/swapfile

2，修改swapfile文件的权限：
chmod 600 /usr/swapfile

3，将swapfile 初始化为交换文件：
mkswap /usr/swapfile

4，然后开启swap.
swapon /usr/swapfile

5，查看是否配置成功：
free -g

6，将swap分区挂在信息写入 /etc/fstab 文件内
[root@local ~]# vim /etc/fstab
/usr/swapfile swap swap defaults 0 0  添加这行进去
