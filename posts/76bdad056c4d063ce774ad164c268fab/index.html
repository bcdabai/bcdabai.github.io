Excel文件自毁VBA指令

新建一个Excel工作簿，按Ctrl+F11打开Excel VBE代码编辑窗口，把以下代码指令放到ThisWorkbook，只要一运行就会毁掉当前工作簿，需要做好备份：

Sub kills()
Application.DisplayAlerts = False
With ThisWorkbook '定位到ThisWorkbook
    .Saved = True '关闭保存时
    .ChangeFileAccess xlReadOnly '将文件设为只读
Kill .FullName '毁掉文件
    .Close
End With
Application.DisplayAlerts = True
End Sub

注意Kill与.FullName之间需留一空格。
