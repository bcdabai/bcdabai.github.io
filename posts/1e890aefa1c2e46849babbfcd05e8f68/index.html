系统报.NET Framework 3.5缺失无法安装问题解决方案

如果遇到以下弹窗问题无法下载安装

1.首先查看电脑内是否缺失net framework 3.5组件

控制面板内——程序——启用或关闭Windows程序

将.NET Framework 3.5(包括.net2.0和3.0)展开

勾选全部文件

 

2.打开 Windows Update

点击此电脑——管理——服务——找到Windows Update

然后点击属性，将其设置为自动

完成以上操作之后，再进行正常的下载安装
