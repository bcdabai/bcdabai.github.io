获取并输出当前日期时间
在Python 中，可以使用 datetime 模块来处理日期和时间。
datetime 模块非常好用，提供了很多日期格式、时间格式处理、转换的对象和方法。
使用时，首先要导入 datetime 库，代码如下:
使用 today 和now 获得当前日期和时间，时间精确到毫秒级，代码如下:

import datetime

print(datetime.datetime.today())  # 2023-08-29 10:55:24.833695

print(datetime.datetime.now())  # 2023-08-29 10:55:24.833694

# 使用format0函数格式化时间，“%Y-%m-%d” 年的格式，“%H:%M:%S,时间格式
print(format(datetime.datetime.now(), "%Y-%m-%d %H:%M:%S"))  # 2023-08-29 10:56:26
