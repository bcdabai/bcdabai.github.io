python实现计算最大公约数和最小公倍数的函数

首先我们可以知道在Python中，数学模块包含许多数学运算，可以使用该模块轻松执行。例如math.gcd()函数计算其参数中提到的2个数的最大公约数

计算最大公约数和最小公倍数的代码如下

import math

def gcd(x, y):#获取最大公约数

a = math.gcd(x,y)

return a

def lcm(x, y):#获取最小公倍数

b = x * y // math.gcd(x,y)

return b
