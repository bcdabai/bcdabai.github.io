1> 函数概述

函数声明，函数定义，函数调用

#include <stdio.h>

/* 子函数声明 */
int add(int a, int b);  

/* 主函数 */
int main()
{
    int c = 0;

	/* 函数调用 */
    c = add(5, 6);

    printf("c = %d", c);

    return 0;
}

/* 子函数定义 */
int add(int a, int b)
{
    return a + b;
}

函数是程序的基本单元；
