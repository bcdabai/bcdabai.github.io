编程实现，四个数求最大值

输入描述：一行，4个整数，用空格分开。

输出描述：一行，一个整数，为输入的4个整数中最大的整数。

具体代码：

#include <stdio.h>

int main()
{
	int arr[4] = {0};
	int  i = 0;
	for(i=0;i<4;i++)
	{
		scanf("%d",&arr[i]);
	}
	int max = 0;
	for(i=0;i<4;i++)
	{
		if(arr[i]>max)
		max = arr[i];
	}
	printf("%d\n",max);
	return 0;
 }
