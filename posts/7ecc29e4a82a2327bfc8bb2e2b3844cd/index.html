Xam.plugin.media调用相机， Unable to get file location

 

本来app 都很正常，升级xamarin.forms框架之后调用相机的时候，报 Unable to get file location. This most likely means that the file provider information is not set in....

            Plugin.Media.Abstractions.MediaFile file = null;
            try
            {
                file = await Plugin.Media.CrossMedia.Current.TakePhotoAsync(new Plugin.Media.Abstractions.StoreCameraMediaOptions
                {
                    Directory = "Sample",
                    Name = "23123213.jpg",
                });

            }
            catch (Exception ex)
            {

                return;
            }

 

找了半天原因，把xam.plugin.media什么的都更新了一下也没用，最后的最后发现是   fileProvider的P大写改成小写就可以了。。。
