Spring Boot 引入jar 包中的 controller

【问题】

Spring Boot 引入jar 包中的 controller

【解决】

  1. 默认情况下 Spring Boot 在启动时, 只会扫描启动类对应的包以及子包进行装配， 因此jar包中的controller与主应用的包名保持一致即可。
  2. 通过import注解引入对应的controller bean对象，在主程序中引入对应的注解类即可加载对应的controller.
