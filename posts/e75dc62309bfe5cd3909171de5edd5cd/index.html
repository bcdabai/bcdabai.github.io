1.2 代码段

代码如下（示例）：

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<script type="text/javascript">
			function check()
			{
				if(event.keyCode=="37")
				  alert('您选项了左键');
				if(event.keyCode=="38")
				  alert('您选项了上键');
				if(event.keyCode=="39")
				  alert('您选项了右键');
				if(event.keyCode=="40")
				  alert('您选项了下键');
			}
		</script>
	</head>
	<body onkeydown="check()">
	</body>
</html>
