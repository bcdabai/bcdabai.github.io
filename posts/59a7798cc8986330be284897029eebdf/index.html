报错:ImproperlyConfigured: Requested setting INSTALLED_APPS, but settings are not configured.

1. 报错现象

在idea的终端中执行

from myapp_api.models import Project, App, Server

后出现以下报错

ImproperlyConfigured: Requested setting INSTALLED_APPS, but settings are not configured. You must either define the environment variable DJANGO_SETTINGS_MODULE or call settings.config
ure() before accessing settings.

2. 解决方法

退出当前环境,在shell中,用以下命令启动

python manage.py shell

就不在发生报错了.
