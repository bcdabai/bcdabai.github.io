Openlayers 中code错误编码对应的问题

Openlayers 中code错误编码对应的问题

错误编码错误信息中文含义
1The view center is not defined.未定义视图中心。
2The view resolution is not defined.未定义视图分辨率。
3The view rotation is not defined.未定义视图旋转。
4image and src cannot be provided at the same time.image并且src不能同时提供。
5imgSize must be set when image is provided.imgSize必须在image提供时设置。
6A defined and non-empty src or image must be provided.必须提供已定义且非空的srcor image。
7format must be set when url is set.format设置时必须url设置。
8Unknown serverType configured.未知serverType配置。
9url must be configured or set using #setUrl().url必须使用#setUrl().
10The default geometryFunction can only handle ol.geom.Point geometries.默认geometryFunction只能处理ol.geom.Point几何图形.
11options.featureTypes should be an Array.options.featureTypes 应该是一个数组。
12options.geometryName must also be provided when options.bbox is set.options.geometryName还必须在设置时提供options.bbox。
13Invalid corner. Valid corners are top-left, top-right, bottom-right and bottom-left.无效角。有效的边角top-left，top-right，bottom-right和bottom-left。
14Invalid color. Valid colors are all CSS colors.颜色无效。有效的颜色都是CSS颜色。
15Tried to get a value for a key that does not exist in the cache.试图获取缓存中不存在的键的值。
16Tried to set a value for a key that is used already.试图为已使用的键设置值。
17resolutions must be sorted in descending order.resolutions 必须按降序排列。
18Either origin or origins must be configured, never both.无论是origin或origins必须进行配置，不可能兼顾。
19Number of tileSizes and resolutions must be equal.数tileSizes和resolutions必须相等。
20Number of origins and resolutions must be equal.数origins和resolutions必须相等。
22Either tileSize or tileSizes must be configured, never both.无论是tileSize或tileSizes必须进行配置，不可能兼顾。
23The passed ol.TileCoords must all have the same z value.传递的ol.TileCoords 必须具有相同的z值。
24Invalid extent or geometry provided as geometry.无效程度或几何形状提供为geometry。
25Cannot fit empty extent provided as geometry.不能适合作为 提供的空范围geometry。
26Features for deletes must have an id set by the feature reader or ol.Feature#setId().特色功能deletes必须由特征阅读器或一个id集 ol.Feature#setId()。
27Features for updates must have an id set by the feature reader or ol.Feature#setId()特色功能updates必须由特征阅读器或一个id集ol.Feature#setId()。
28renderMode must be ‘image’, ‘hybrid’ or ‘vector’.renderMode必须是’image’，‘hybrid’或’vector’。
29x must be greater than 0.x必须大于0。
30The passed feature was already added to the source.通过的feature已经添加到源中。
31Tried to enqueue an element that was already added to the queue.试图将element已添加到队列中的一个入队。
32Transformation matrix cannot be inverted.变换矩阵不能求逆。
33Invalid units. ‘degrees’, ‘imperial’, ‘nautical’, ‘metric’ or ‘us’ required.无效units。‘degrees’、‘imperial’、‘nautical’、'metric’或’us’必需。
34Invalid geometry layout. Must be XY, XYZ, XYM or XYZM.几何布局无效。必须是XY、XYZ、XYM或XYZM。
36Unknown SRS type. Expected “name” or “EPSG”未知的 SRS 类型。预期"name"或"EPSG"。
37Unknown geometry type found. Expected ‘Point’, ‘LineString’, ‘Polygon’ or ‘GeometryCollection’.发现未知的几何类型。预期’Point’, ‘LineString’,‘Polygon’或’GeometryCollection’。
38styleMapValue has an unknown type.styleMapValue 有一个未知的类型。
39Unknown geometry type found. Expected ‘GeometryCollection’, ‘MultiPoint’, ‘MultiLineString’ or ‘MultiPolygon’.发现未知的几何类型。预期’GeometryCollection’, ‘MultiPoint’,‘MultiLineString’或’MultiPolygon’。
40Expected feature to have a geometry.期望feature有一个几何。
41Expected an ol.style.Style or an array of ol.style.Style.应为ol.style.Style或 数组ol.style.Style。
43Expected layers to be an array or an ol.Collection.预期layers是一个数组或一个ol.Collection.
44logo.href should be a string.logo.href 应该是一个字符串。
45logo.src should be a string.logo.src 应该是一个字符串。
46Incorrect format for renderer option.renderer选项格式不正确。
47Expected controls to be an array or an ol.Collection预期controls是一个数组或一个ol.Collection.
48Expected interactions to be an array or an ol.Collection.预期interactions是一个数组或一个ol.Collection.
49Expected overlays to be an array or an ol.Collection.预期overlays是一个数组或一个ol.Collection.
50Cannot determine Rest Service from url.无法从 url 确定 Rest 服务。
51Either url or tileJSON options must be provided.无论是url或tileJSON必须提供的选项。
52Unknown serverType configured.未知serverType配置。
53Unknown tierSizeCalculation configured.未知tierSizeCalculation配置。
54Hex color should have 3 or 6 digits.十六进制颜色应为 3 或 6 位数字。
55The {-y} placeholder requires a tile grid with extent.的{-y}占位符需要tile网格与程度。
56mapBrowserEvent must originate from a pointer event.mapBrowserEvent 必须源自指针事件。
57At least 2 conditions are required.至少需要2个条件。
58Duplicate item added to a unique collection. For example, it may be that you tried to add the same layer to a map twice. Check for calls to map.addLayer() or other places where the map’s layer collection is modified.添加到唯一集合的重复项。例如，您可能尝试将同一图层添加到地图两次。检查对map.addLayer()修改地图图层集合的调用或其他地方。
59Invalid command found in the PBF. This indicates that the loaded vector tile may be corrupt.在 PBF 中发现无效命令。这表明加载的矢量切片可能已损坏。
