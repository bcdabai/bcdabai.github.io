C++ x的y次方对n取模

模运算满足分配率，对于任意的整数 a,b,q,r,n，可以构造：
a = K1 * n + q
b = K2 * n + r
则 (a*b) mod n = ((a mod n) * (b mod n)) mod n = q * r

int pow(int x, int y, int mod) {  
    int res = 1;  
    while(y)  
    {  
        if(y & 1) {
            res =  (res * x) % mod; 
        }
        x = (x * x) % mod;  
        y = y / 2;  
    }  
    return res;  
}
