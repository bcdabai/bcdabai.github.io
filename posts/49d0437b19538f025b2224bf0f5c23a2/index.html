git 日常操作

创建远程仓库

  1. 在你要创建仓库的文件目录下 右键 选择 Git Bash Here

  2. 输入命令 git init,执行成功之后 你会看见在此目录下会生成 名为“.git”的文件夹

  3. 执行命令 git status 查看文件状态

  4. 输入 命令 git add . 添加所有文件

  5. 输入命令 git commit -m ‘注释’ 提交所有操作

  6. 创建远程仓库：打开 github官网，登录账号 新建仓库

  7. 创建SSH KEY。先看一下你C盘用户目录下有没有.ssh目录，有的话看下里面有没有id_rsa和id_rsa.pub这两个文件，有就跳到下一步，没有就通过下面命令创建：

$ ssh-keygen -t rsa -C “youremail@example.com”

  1. 登录Github,找到右上角的图标，打开点进里面的Settings，再选中里面的SSH and GPG KEYS，点击右上角的New SSH key，然后Title里面随便填，再把刚才id_rsa.pub里面的内容复制到Title下面的Key内容框里面，最后点击Add SSH key，这样就完成了SSH Key的加密。具体步骤也可看下面：

  2. 使用 git remote add origin https://github.com/guyibang/TEST2.git 命令 本地仓库关联远程仓库
  3. 使用 git push -u origin master 命令上传代码

参考文章:https://blog.csdn.net/liuweixiao520/article/details/78971221

创建分支

打开 Git Bash Here 执行 git checkout -b 分支名称

合并分支（develop合并到master）

1、切换分支 ： git checkout master
2、合并分支： == git merge develop==
3、推送到远程仓库： git push origin master

问题

The following untracked working tree files would be overwritten by checkout
解决方法： git clean -d -fx

1、删除文件夹： git rm -r 文件夹名/
2、提交代码：git commit -m ‘注释’
3、推送到远程仓库： git push origin develop
