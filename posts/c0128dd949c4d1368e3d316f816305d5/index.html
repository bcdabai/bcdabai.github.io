在 windows 系统上 go build linux 系统的可执行文件

设置 GOOS 为 linux

go env -w GOOS=linux

设置完后 go env 查看更改成功

go build 执行完成后生成可执行文件，重新更改 GOOS 为 windows

go env -w GOOS=windows

go env 查看更改回来
