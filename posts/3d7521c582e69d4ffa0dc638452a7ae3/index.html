活动表达式

使用 Live Expressions 监视 JavaScript 中的更改

  • 在控制台中运行代码
let x = 0;
let y = 0;
document.addEventListener('mousemove', e => {
    x = e.x;
    y = e.y;
});
  • 点击小眼睛，创建活动表达式，输入[x,y]，并按ctrl+enter确定即可。
