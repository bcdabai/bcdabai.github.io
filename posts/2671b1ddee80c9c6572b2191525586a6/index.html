jsoncpp Json::Value 成员变量/全局变量析构时引起崩溃问题解决

崩溃案例

struct ValTest{
    Json::Value bugJson;
};
ValTest test;
int main(int argc, _TCHAR* argv[]){
    test.bugJson["init"] = 123;
    return 0;
}

程序退出时引发崩溃。

解决办法：在构造函数中加入 

bugJson["init"] = 123;

内容可以随便写，但是要初始赋值一次
