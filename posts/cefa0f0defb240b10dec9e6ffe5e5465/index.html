【docker专题_07】vim: command not found解决方案

前言

今天在公司使用docker部署redis集群的时候，遇到问题，就是有些配置文件需要进行编辑，虽然可以在容器外部拷贝进去，但有时候只是一点点修改，需要在里面修改配置文件，出现vim: command not found，记录下解决方案

1.更新安装源

apt-get update

2.安装vim

apt-get install vim

3.注意事项

#若是不先进行更新，会出现以下问题
Reading package lists... Done
Building dependency tree       
Reading state information... Done
E: Unable to locate package vim
