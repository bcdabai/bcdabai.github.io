VBA之正则表达式（27）-- 提取包含关键字的字段

实例需求：待处理字符串如下，需要提取其中包含“电子”的部分，即：电子计算机、微型电子管、新创电子

-电脑-电脑硬盘-电子计算机-晶体管-微型电子管-新创电子-

示例代码如下。

Sub Demo1()
    Dim strWord As String
    Dim objRegExp As Object, objMH As Object
    Set objRegExp = CreateObject("VBSCRIPT.REGEXP")
    strWord = "-电脑-电脑硬盘-电子计算机-晶体管-微型电子管-新创电子-"
    With objRegExp
        .Global = True
        .ignoreCase = True
        .Pattern = "[一-龥]*电子[一-龥]*"
        Set objMatch = .Execute(strWord)
        If objMatch.Count > 0 Then
            For Each objMH In objMatch
               Debug.Print objMH
            Next
        End If
    End With
    Set objRegExp = Nothing
End Sub

运行代码结果如下。

电子计算机
微型电子管
新创电子

【代码解析】
第4行代码创建正则表达式对象。
第5行代码设置字符串变量，从A1单元格读取文本。
第8行代码设置正则匹配忽略大小写。
第9行代码设置匹配模式。
第10行代码执行正则匹配。
如果匹配成功，第12~14行代码使用For…Next循环结构在立即窗口中输出匹配结果。

正则表达式说明
[一-龥]*匹配零个或者多个中文字符
电子匹配指定的关键字

简单总结，正则匹配模式查找“电子”关键字，其前可以有中文字符，其后也可以有中文字符

这个字符串其实比较规整，不使用正则也可以轻松处理，代码如下。

Sub Demo2()
    s = "-电脑-电脑硬盘-电子计算机-晶体管-微型电子管-新创电子-"
    arr = Split(s, "-")
    For i = 0 To UBound(arr)
        If InStr(arr(i), "电子") > 0 Then Debug.Print arr(i)
    Next
End Sub

【代码解析】
第3行代码使用split将字符串拆分为数组，分隔符为减号。
第4~6行代码循环处理数组中的字符串。
第5行代码判断是否包含关键字“电子”，如果包含，则输出到立即窗口。
