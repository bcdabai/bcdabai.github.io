TypeScript中枚举报错 ‘xx‘ is defined but never used

在使用eslint时 我们明明以及使用了枚举但是就是报错 这个是因为Eslint不支持 TypeScript 语法

只需要在eslintrc.js中配置

 "@typescript-eslint/no-unused-vars" : [ "error" ]

 就可以了
