flask项目配置和访问

Flask 是一个微型的 Python 开发的 Web 框架，基于Werkzeug WSGI工具箱和Jinja2 模板引擎。 Flask使用BSD授权。 Flask也被称为“microframework”，因为它使用简单的核心，用extension增加其他功能。Flask没有默认使用的数据库、窗体验证工具。然而，Flask保留了扩增的弹性，可以用Flask-extension加入这些功能：ORM、窗体验证工具、文件上传、各种开放式身份验证技术。

【官方】Flask中文文档

  1. 安装Flask包。
 pip install Flask==1.1.4

  1. 下载并安装Pycharm集成开发工具。

  2. 新建flask项目。（可使用virtualenv或者virtualenvwrapper管理工具，创建相应python版本的虚拟环境后选择使用，方便管理）python项目虚拟环境创建及管理

  3. 创建完成后，会有ststic，templates两个文件夹，以及app.py文件。

  4. 然后创建个第一个Flask文件程序。

from flask import Flask
#初始化
app=Flask(__name__)
#创建路由'/'
@app.route('/')
def index():
    return 'Hello World!'

if __name__ == '__main__':
    app.run()

使用 route() 装饰器来把函数绑定到 URL
6. 运行，在Pycharm的Terminal里输入 python app.py 运行flask程序。Terminal里按住ctrl + c 关闭停止运行该程序。

  1. 访问。浏览器输入访问：http://127.0.0.1:5000/
  2. 主机号和端口的设置。
from flask import Flask
#初始化
app=Flask(__name__)
#创建路由'/'
@app.route('/')
def index():
    return 'Hello World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0',port=8080)

设置主机号为 0.0.0.0 ，端口号为8080，当主机号为0.0.0.0 时，可以让外部客户端进行访问,访问地址是flask服务器的ip地址和你设置的端口(端口注意不要占用其他端口,如果是阿里云有可能要设置端口开放)。

查看ip地址：

打开命令提示符（cmd）查看当前主机的ip,终端输入：
ipconfig

然后在同一网下的其他主机上输入网址
http://192.168.1.107:8080/
即可访问。
9. 开发者模式配置。

from flask import Flask
#初始化
app=Flask(__name__)
#设置开发者模式
app.config['ENV'] = 'development'
app.debug = True
#创建路由'/'
@app.route('/')
def index():
    return 'Hello World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0',port=8080)


如果源码发生改变，服务器则会刷新更新内容。
一般情况下，我们会将这些配置放在一个单独的seetings.py文件内，便于管理。

###seetings.py
class Config:
    DEBUG = True

class DevelopmentConfig(Config):  #开发环境
    ENV = 'development'

class ProductionConfig(Config):  #产品模式
    ENV = 'production'
    DEBUG = False

####app.py
from flask import Flask

import seetings

app = Flask(__name__)
app.config.from_object(seetings.DevelopmentConfig)

@app.route('/')
def index():
    return 'Hello World!!'


if __name__ == '__main__':
    app.run(host='0.0.0.0',port=8080)
