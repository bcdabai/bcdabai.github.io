conda创建/复制/删除虚拟环境

1.创建新的虚拟环境

conda create -n test python=3.7

2.删除虚拟环境

conda remove -n test --all

3. 激活环境

conda activate test

4.退出当前环境

conda deactivate

5.复制虚拟环境

conda create -n conda-env2 --clone conda-env1

6.列出所有虚拟环境

conda env list
