JavaDoc文档生成二种方式

javadoc生成：

方式一：类名前面加上如下注释：

/**
 * @author ahtoh
 * @version 1.0
 * @since 1.8
 */

切换到当前类所在的路径，终端输入指令：

javadoc -encoding UTF-8 -charset UTF-8 JavaDoc.java

其中JavaDoc为所需生成文档的java类名。

方式二：Idea生成Java Doc文档

Idea版本：2020.1

Idea顶部Tools -> Generate JavaDoc -> 点击Output directory选择要生成文档位置

-> 底部Locale配置语言和编码 -> 点击确认生成JavaDoc文档。
