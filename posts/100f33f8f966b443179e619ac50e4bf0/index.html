django传值给模板, 再用JS接收并进行操作

今天用要django传值给模板, 然后需要用js处理一下.特此记录.

  1. 用json.dumps()方法将值传给模板.
    import json

return render(request,'wauth/freetime.html', {'slot_ids': json.dumps(slot_ids)})

  2. 用JSON.parse()接收django传过来的值.
    var slot_ids = JSON.parse('{{ slot_ids|safe }}');
