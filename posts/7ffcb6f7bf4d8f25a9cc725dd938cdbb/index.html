结构体的三种定义方法

1.单独定义

//先定义结构体再定义变量
struct students{
	int age;
	string name;
};
struct studengts s1,s2;

2.混合定义

struct students{
	int age;
	string name;
}s1,s2;

3.无类型名定义

struct{
	int age;
	string name;
}s1;
