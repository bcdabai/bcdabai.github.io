	int i = 1; //循环变量初始化
	while( i <= 10 ) {//循环条件

4. 课堂练习题

WhileExercise.java

  1. 打印 1-100 之间所有能被 3 整除的数 [使用 while ]
  • 化繁为简：即将复杂的需求，拆解成简单的需求，逐步完成。先输出 1-100 之间的数
	int i = 1;
	while (i <= 100){
	    System.out.println("i="+i);
	    i++;
	}

  • 先死后活：先考虑固定的值，然后转成可以灵活变化的值。再把被 3 整除的数用变量表示，1-100之间的范围的100也使用变量表示。
	int i = 1;
	int end = 100;
	int count = 3;
	while (i <= end){
	    if (i % count == 0){
	        System.out.println("i="+i);
	    }
	    i++;
	}

  1. 打印 40—200 之间所有的偶数 [使用 while]
	int j = 40;
	while (j <= 200){
	    if (j % 2 == 0){
	        System.out.println("j=" + j);
	    }
	    j++;
	}
