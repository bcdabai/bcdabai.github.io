Ubuntu apt-get 删除安装失败的包

 查看有哪些安装失败的包

$ sudo dpkg --configure -a

删除安装失败的包 

$ sudo dpkg --remove --force-remove-reinstreq package_name 
$ sudo apt-get update

删除软件包后，使用命令更新系统，成功更新后重新启动系统。

 

参考：https://ubuntuqa.com/article/10754.html
