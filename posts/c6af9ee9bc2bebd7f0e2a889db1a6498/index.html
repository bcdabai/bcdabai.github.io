django导入celery报错

在安装python虚拟环境后celery不能正常导入使用了
python ----- 3.7.7
celery ------ 4.4.7

ImportError: cannot import name 'Celery' from 'celery' (/root/.virtualenvs/venv/lib/python3.7/site-packages/cel
ery/__init__.py)

解决办法：

重新安装 importlib-metadata，且 版本必须小于5

pip3 uninstall importlib-metadata 
pip3 install importlib-metadata==4.13.0
