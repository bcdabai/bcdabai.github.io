Java两数相除四舍五入保留4位小数

例：

		double a = 5;
		double b = 11;
		double num = a/b;
		BigDecimal b1 = new BigDecimal(num);
		//保留四位小数
		double xxx= b1.setScale(4, BigDecimal.ROUND_HALF_UP).doubleValue();

       System.out.println(xxxx);
