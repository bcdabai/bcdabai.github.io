SQL Server报表服务

SQL Server报表服务：基于服务器的平台，用来创建、管理和发送报表。也可以使用SQL Server创建、分发和管理报表。

1、启动SSRS服务

步骤：我的电脑->管理->服务和应用程序->Sql Server配置管理器->Sql Server服务->Sql Server(MSSQLSERVER)右键启动服务

2、创建报表服务器项目

步骤：（1）打开Microsoft  SQL Server 2008 -> SQL Server Business Intelligence Developement Studio 启动开发平台。

（2）文件->新建->项目->新建项目->商业智能项目->报表服务器项目->确定

3、建立数据源

步骤：视图->解决方案资源管理器->右击“共享数据源”->添加新数据源->弹出共享数据源属性->点击编辑->显示“连接属性”->在服务器名框中，用户输入要连接的数据库服务器名称（127.0.0.1,1483）->用户选择或者输入一个数据库名->测试连接->确定之后会产生DataSource1.rds文件。

 

 

4、建立报表

步骤：解决方案资源管理器->报表->右键添加新报表->下一步->在共享数据源框中，选择刚建立的数据源->下一步->点击“查询生成器”->SELECT  输入列名 FROM  输入表名->确定，下一步->弹出选择报表类型界面，用户根据所需要选择报表的生成方法（表格/矩阵）表格，下一步->下一步->下一步->输入报表名称->完成。

打开数据集->DataSet1->右键列名，字段属性->可以对列进行修改。

5、运行报表

在“解决方案资源管理器”中，点击“报表”->右键“运行”。
