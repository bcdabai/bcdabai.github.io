Java-二进制转换

  1. 十进制转成二进制

    //将十进制数转成字符串，例如n=5 ，s = "101"
String s1 = Integer.toBinaryString(5);

  2. 将字符串转成整形

    //只能是数字类的字符串
int a = Integer.valueOf("1002");
//或者
int b = Integer.parseInt("1002");

  3. 将整形转成字符串

    String s3 = String.valueOf(1025);

  4. 二进制字符串转十进制

    int total = 0;
String s5 = "1002";
char[] binCharArray = s5.toCharArray();
for (int i = 0; i < binCharArray.length; i++) {
    total = total + binCharArray[i] - 48;
    if (i != binCharArray.length - 1) {
        total = total << 1;
    }
}
