题目



答案

List Delete( List L, ElementType minD, ElementType maxD )
{
	int i,j=0;
	for(i=0;i<=L->Last;i++)
	{
		if(L->Data[i]>minD&&L->Data[i]<maxD) continue;
		else L->Data[j++]=L->Data[i]; 
	}
	L->Last=j-1;
	return L;
}

注意

这里的Last就是数组最后一个数字的下标，如果数组有10个数，Last的值就为9
