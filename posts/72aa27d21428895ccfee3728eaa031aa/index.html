后端接收前端url请求传的一个数组ids

#方法一： 使用可变参数长度 接收

 @RequestMapping("/1")
    public void receive(String ...parms){//三个点表示可变参数
        System.out.println("parms.length="+parms.length);
        for (int i = 0; i < parms.length; i++) {
            System.out.println(parms[i]);
        }
    }

测试

传参的时候 多个参数用逗号隔开


方法二 ： 使用数组接收

@RequestMapping("/2")
    public void receive2(String[] parms){
        System.out.println("parms.length="+parms.length);
        for (int i = 0; i < parms.length; i++) {
            System.out.println(parms[i]);
        }
    }


小结

使用数组接收或者使用可变参数接收实际都是一样的
因为这两个方法不能重载
