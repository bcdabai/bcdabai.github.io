（C语言，慕课）矩阵的转置运算

题目内容：

编写程序，将一个n×n（）的矩阵进行转置，即将行与列互换，转置后再按n×n矩阵形式输出。

输入格式:

输入整数n，从第2行开始依次输入n*n个整数，各数以一个空格分隔。

输出格式：

按n行n列的形式输出转置后的二维数组，每个元素占4列，采用右对齐（即在左补空格），行末无空格。

输入样例：

3

1 2 3 4 5 6 7 8 9

输出样例：

1 4 7

2 5 8

3 6 9

#include<stdio.h>
int main()
{
    int a,i,j,num[10][10];
    scanf("%d",&a);
    for(j=0;j<a;j++)
    {
        for(i=0;i<a;i++)
        {
            scanf("%d ",&num[i][j]);
        }
    }
    for(i=0;i<a;i++)
    {
        for(j=0;j<a;j++)
        {
            printf("%4d",num[i][j]);
        }
        printf("\n");
    }
}
