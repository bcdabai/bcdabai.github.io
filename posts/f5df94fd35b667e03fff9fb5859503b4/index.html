permission denied, access ‘/usr/local/lib/node_modules‘ node_modules没有写权限

问题原因：node_modules文件夹操作没有权限
问题解决：
一行命令解决问题

sudo chmod 777 /usr/local/lib/node_modules

详细内容介绍

  1. 查看node_modules文件夹的权限，如下内容 说明普通用户执行没有写的权限
➜  ~ cd /usr/local/lib
➜  lib ls node_modules
total 0
drwxr-xr-x  5 root  wheel  160  7 16 09:19 .
drwxr-xr-x  9 root  wheel  288  7 16 09:19 ..
drwxr-xr-x  3 root  wheel   96  5 26 15:25 docker
drwxr-xr-x  3 root  wheel   96  7  1 03:17 dtrace
drwxr-xr-x  3 root  wheel   96  7 16 09:19 node_modules
  1. 修改node_modules文件夹权限
➜  lib sudo chmod 777 node_modules 
Password:
➜  lib ls -all
total 0
drwxr-xr-x  5 root  wheel  160  7 16 09:19 .
drwxr-xr-x  9 root  wheel  288  7 16 09:19 ..
drwxr-xr-x  3 root  wheel   96  5 26 15:25 docker
drwxr-xr-x  3 root  wheel   96  7  1 03:17 dtrace
drwxrwxrwx  4 root  wheel  128  7 16 09:56 node_modules
  1. 重新执行命令就可以了
