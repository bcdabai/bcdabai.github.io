解决办法

在config.ini中按照以下配置

### Emby / Jellyfin 服务器地址 ###
Host_Url=http://192.168.0.10:8096/
	
### Emby / Jellyfin API 密钥 ###
Host_API=38xxxxxxxxxx

Proxy_Range = ALL
Proxy_Link =

关闭所有代理，使用软件自带的通道，能够正常刮削

实验环境：重庆大学教育网ipv6
