【Go依赖】main.go:7:2: cannot find package “github.com/IBM/sarama“ in any of:

问题：

main.go:7:2: cannot find package "github.com/Shopify/sarama" in any of:
        /usr/local/go/src/github.com/Shopify/sarama (from $GOROOT)
        /home/gopath/src/github.com/Shopify/sarama (from $GOPATH)

解决：

go mod init  # 初始化go.mod
go mod tidy  # 更新依赖文件
go mod download  # 下载依赖文件
go mod vendor  # 将依赖转移至本地的vendor文件

go mod edit # 手动修改依赖文件
go mod graph # 打印依赖图
go mod verify # 校验依赖
