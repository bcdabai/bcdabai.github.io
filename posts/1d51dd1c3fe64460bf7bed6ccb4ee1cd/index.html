（python）最大公约数和最小公倍数

在两行中分别输入一个正整数M,N，输出这两个数的最大公约数和最小公倍数。

解决代码如下：

import math

num1 = int(input())
num2 = int(input())

m = int(input()) 
n = int(input())
gys = 1
if m > n:  
    m, n = n, m
for i in range(n, 0, -1):  
    if n % i == 0 and m % i == 0:  
        gys = i
        break  
print(gys, m * n // gys)
