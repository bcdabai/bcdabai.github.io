62. 不同路径

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

示例 1：

输入：m = 3, n = 7
输出：28

示例 2：

输入：m = 3, n = 2
输出：3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。

  1. 向右 -> 向下 -> 向下
  2. 向下 -> 向下 -> 向右
  3. 向下 -> 向右 -> 向下

示例 3：

输入：m = 7, n = 3
输出：28

示例 4：

输入：m = 3, n = 3
输出：6

提示：

1 <= m, n <= 100
题目数据保证答案小于等于 2 * 109

思路：动态规划
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> f (m,vector<int>(n));
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                if(!i && !j) f[i][j]=1;
                else{
                    if(i) f[i][j]+=f[i-1][j];
                    if(j) f[i][j]+=f[i][j-1];
                }     
            }
        }
        return f[m-1][n-1];
    }
};

复杂度分析

  • 时间复杂度：O(mn)
  • 空间复杂度：
    上述做法需要一个大小为m x n的二维数组，空间复杂度为O(mn)，但其实dp[i][j]只会用到dp[i-1][j]和dp[i][j-1]，不会用到之前的数据，因此可以用滚动数组减小空间复杂度。
