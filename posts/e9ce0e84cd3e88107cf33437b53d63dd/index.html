解决：Caused by: redis.clients.jedis.exceptions.JedisDataException: WRONGTYPE Operation against a key

前些天发现了一个巨牛的人工智能学习网站，通俗易懂，风趣幽默，忍不住分享一下给大家。点击跳转到教程。

 

1. 报错：

Caused by: redis.clients.jedis.exceptions.JedisDataException: 
WRONGTYPE Operation against a key holding the wrong kind of value

2. 报错代码：

    /**
     * 写入 list
     *
     * @param key
     * @param values
     */
    public <V> Long setList(String key, Collection<V> values) {

        listOperations = redisTemplate.opsForList();
        Long okCount = listOperations.leftPushAll(key, values); // 此行报错

        return okCount;
    }

3. 错误的原因是我在set的时候先放入的是A类型的数据。后来我由于代码写错了，又想要放入B类型的数据。放入失败并报错。

 

 

4. 保证数据类型一致就 OK了。
