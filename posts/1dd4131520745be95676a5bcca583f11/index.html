easyui学习之datagrid

easyui1.3.3

  1. 获得当前页的所有行数据: $('#id').datagrid('getRows');
  2. 获得所有数据源 :$('#id').datagrid('originData ');
  3. 当文字超过单元格长度可以自动换行nowrap:false
  4. 清除数据：$('#id').datagrid('loadData',rows);(var rows= [];)
  5. easyui手动跳转至几页:$('#id').datagrid('getPager').pagination('select',pageNum(数字));
  6. 重新给datagrid加载数据 不会自动跳转至第一页
  7. easyui隐藏列: hidden:true
  8. easyui datagrid加载成功函数：
    onLoadSuccess:function(data){
    if(data.total==0){
    }
    }
  9. 没有数据的json格式：
    {total:0,rows:[]}
    datagrid增加一行 ('#id').datagrid('appendRow',json格式的字符串（包含{field字段名：值}）);
  10. 若json格式不对或数据名称不匹配 行会为空
  11. datagrid url 访问 返回json格式匹配field字段 可以通过 ('#id').datagrid('load',{VAL:val});（{后台获取的数据字段名:数据} )

  12. datagrid更新 ('#id').datagrid('updateRow',index:索引数字,row:{更新的字段：数据});
    注:获取某行 通过直接赋值的方式改变它的值再调用reload不能更新数据

  13. 获得索引:index=('#id').datagrid('getRowIndex',row);

  14. 删除行('#id').datagrid('deleteRow',index);
  15. 获得选中行的数据:$('#id').datagrid('getSelected')
  16. 给一列添加按钮:
    formatter:btnFun
    btnFun(val,row,index){
    //val表示单元格的值
    //row当前行
    //index当前行的索引
    return '<button type="button"></button>'
    }
