数组地址加一详解

1.一维数组

int a[5]={1,2,3,4,5};
//int *a=new int[5];
int *ptr=(int*)(&a+1);             //&a+1相当于加了数组的一个单位（也就是一行），即&a[5]
printf("%d", *(p-1))               //指针p-1相当于前移一个元素，输出为5

a是数组首地址a[0]的地址，&a是对象（数组）首地址；

a+1，表示指针指向下一个元素a[1]，这是因为系统在内存中为数组分配一个连续大小为20字节的大小，并把首地址赋值给数组名a，那么&a+1就是表示这个内存的下一个地址。

（int）a表示的把数组地址即内存单元的编号转换成整数，那么此时的（int）a+1加一相当于加一个整数

 

2.二维数组

int a[2][3] = { {1,2,3},{4,5,6} };
a代表二维数组中第一个一维数组a[0]的首地址，a+1代表移动到下一个一维数组a[1]的首地址

int (*p)[3];//一定要加上括号，因为[]的优先级高于*
p=a;//相当于p&a[0]，也就是指向第一个数组a[0]的首地址
cout<<p+1<<endl;//数组a[1]的首地址

int *p=&a[0][0];//数组元素首地址p相当于&a[0][0]
cout<<p+1<<endl;//p+1相当于&a[0][1]
