[PTA]平均之上

输入一个整数n，再输入n个成绩，求大于平均分的成绩个数。
例如，输入5 68 79 56 95 88，输出3。

输入格式:

首先输入一个正整数T，表示测试数据的组数，然后是T组测试数据。
每组测试先输入n，表示数据个数，然后输入n个整数。

输出格式:

对于每组测试，输出大于平均分的成绩个数。

输入样例:

输出样例:

/*注：其实这道题不用指针还要简单些，但为了熟悉指针就用的这种
方式，如果还没有学习指针就不要用这种来*/

#include<stdio.h>
int function(int* f, int k);//被调用函数：对数字求平均值并输出大于平均值的数字个数
int main()
{
	int n = 0;//接收组数
	int A[1000][1000] = { 0 };//每行代表一组数字(不含第一个数字即数字个数)
	scanf("%d", &n);
	for (int i = 0; i < n; i++)
	{
		int count = 0;//用于接收大于平均值数的个数
		int k = 0;
		scanf("%d", &k);
		for (int j = 0; j < k; j++)
		{
			scanf("%d", &A[i][j]);
		}
		count = function(A[i], k);//A[i]即i行首个元素的地址
		printf("%d\n", count);
	}
	return 0;
}
int function(int* f, int k)
{
	int sum_ = 0;
	int count = 0;
	for (int i = 0; i < k; i++)
	{
		sum_ += *(f + i);//求出数字总和
	}
	sum_ = sum_ / k;//重新赋值为平均
	for (int i = 0; i < k; i++)
	{
		if (*(f + i) > sum_)//f+i是下标为main函数中i行function函数中第i个元素的地址
		{
			count++;//每次发现大于平均值的数，计数自增1
		}
	}
	return count;
}
