用递归实现字符串逆序函数。reverse_string(char * string)

函数功能：

函数形式为：void reverse_string(char* string)

参数：

        sting为指向需要逆序的字符串的首字符地址的指针

内容：

        将string所指向的字符与该字符往后第一个'\n'之间的字符串逆序。（包括首字符，不包括'\n'）

返回值：

        无返回值。

函数实现：

假设这是需要逆序的字符串。将a的地址传进函数。

基本思路就是先将*p1和*p2互换，然后再将中间的bcdef逆序；*p2就是*（p1+strlen（p1）-1）；接下来尝试代码实现

优化。

 

 最后进到最后一层，开始回归的时候吧b的字符赋给字符串的第一个\0就行了；

代码如下： 

#include<stdio.h>
#include<string.h>
void reverse_string(char* p)
{
	char a = *p;
	*p = *(p + strlen(p) - 1);
	*(p + strlen(p) - 1) = a;

	char b = *(p + strlen(p) - 1);
	*(p + strlen(p) - 1) = '\0';

	if (p < p + strlen(p) - 1)
		reverse_string(p + 1);

	*(p + strlen(p) - 1) = b;
}


int main()
{
	char s[] = "abc";
	printf("%s\n", s);
	reverse_string(s);
	printf("%s\n", s);
	return 0;
}

来重新理一遍思路

此时不满足递归条件，应该将b中的字符b返回到*p1中，但是代码中

 这里我们本来是想将b的字符给*p1，但是由于*p1被置为\0，改变了strlen（p）的大小，导致*(p + strlen(p) - 1)实际上是*(p-1)也就是指向c

所以这里把b给了c，字符串就变成了 b \0 \0 \0;

这一层结束，回到第一层，

 

第一层b中放的是a， 而strlen(p) 变为了1；即*(p + strlen(p) - 1)就是指的第一个字符：b

此时执行*(p + strlen(p) - 1) = b，就是将第一个字符b换成a；所以字符串变成了：a \0 \0 \0

此时递归结束，打印出来为a，

这就是问题所在，代码改进如下：

void reverse_string(char* p)
{
	int l = strlen(p);

	char a = *p;
	*p = *(p + l - 1);
	*(p + l - 1) = a;

	char b = *(p + l - 1);
	*(p + l - 1) = '\0';

	if(p< p + l - 1)
		reverse_string(p + 1);

	*(p + l - 1) = b;
}

 只需要将strlen（p）提前存起来，之后就不会在递归完毕回归时因为字符被赋为\0而改变strlen（p）的大小；
