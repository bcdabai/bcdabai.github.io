五、安装配置Qt Creator

安装：
sudo apt install qtcreator
配置：

  1. 打开qt creator,依次选择Tools->Options->Build & Run->Compilers

  2. 点击Add按钮选择GCC。 在Compiler path:’中选择加入路径/usr/bin/gcc。并依次设置ABI的选项如下图所示:（custom – arm – linux – generic – elf – 64 bit）

  3. 然后点击Apply按钮保存，然后点击Kits

  4. 继续点击Add按钮，按下图填写Name，Qt version, CMake Tool等信息，填好如下图所示。

  5. 点击Apply,然后点击OK，设置完毕。
