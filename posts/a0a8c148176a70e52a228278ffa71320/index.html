ifconfig 命令没有显示 eth0

虚拟机中输入 ifconfig 命令，没有显示 eth0，只有 lo。
解决办法：

  1. 进入 /etc/sysconfig/network-scripts 目录，查看是否存在 ifcfg-eth0，有的话即网卡存在但未启动；
  2. 输入 ifconfig -a，显示 eth0 和 lo；
  3. 输入 ifconfig eth0 up 启用网卡，输入 ifconfig，只能看到 inet6 的地址，无 inet 地址；
  4. 输入 vim ifcfg-eth0，修改该文件，把 ONBOOT=no 改为 yes；
  5. 输入 service network restart，重启。

这时再输入 ifconfig，就会显示 eth0 的 inet 地址，可以正常连接了。
