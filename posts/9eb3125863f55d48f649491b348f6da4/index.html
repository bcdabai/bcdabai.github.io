CSS文字或图片悬浮于另一张图片上，大小宽度适配手机的！

<div class="header clearfix">
	<div class="hd-c">
  	<div class="banner">
		<div class="pic">
		  <img  src="1.jpg" class="png-24" alt="banner"><!--背景图-->
		</div>	
  		<div class="frame">
		<img class="changing-over" src="progress.gif"><!--悬浮图-->
		</div>
		</div>
	</div>
</div>


上面是HTML布局，下面是CSS样式


.clearfix {
float: none;
clear: both;
}

.header {
position: relative;
width: 100%;
height: auto;
}

.hd-c {
position: relative;
overflow: hidden;
width: 100%;
height:auto;
-moz-background-size: 100% 100%;
-o-background-size: 100% 100%;
-webkit-background-size: 100% 100%;
-ms-background-size: 100% 100%;
}

.hd-c .banner {
width: 100%;
height: auto;
position: relative;
margin: 0 auto;
}

.content {
text-align:center;

}

.banner .frame {
position: absolute;
top:50%;<!--悬浮相对于背景位置的一半-->
left:32%;<!--相对于左边-->
z-index: 2;<!--让图片悬浮于背景之上-->
width: 45%;<!--图片宽度大小-->

}

.banner .pic {
position: relative;
width: 100%;
height: auto;
text-align: center;
}

.banner .png-24 {
_display: none;
width: 100%;
height: auto;
margin: 0 auto;
}

.banner .frame .changing-over{float:left;margin:1px 0 1px 1px;width:100%;height:auto;text-align:center}



注：红色框部分即悬浮的图片，是个动态的Gif图片进度条
