c++中的操作符

一：关系操作符

C 语⾔⽤于⽐较的表达式，称为 “关系表达式”（relational expression），⾥⾯使⽤的运算符就称 为“关系运算符”（relational operator），主要有下⾯6个。

  • > ⼤于运算符
  •  < ⼩于运算符
  • >= ⼤于等于运算符
  • <= ⼩于等于运算符
  •  == 相等运算符
  •  != 不相等运算符

二：条件操作符

条件操作符也叫三⽬操作符，需要接受三个操作数的：

形式如下： 1 exp1 ? exp2 : exp3

条件操作符的计算逻辑是：如果 exp1 为真， exp2 计算，计算的结果是整个表达式的结果；如果 exp1 为假， exp3 计算，计算的结果是整个表达式的结果。

练习1：使⽤条件操作符表⽰下⾯代码的逻辑

#include<stdio.h>
int main()
{
int a=0;
int b=0;
scanf("%d%d",&a,&b);
if(a>10)
   b=1;
else
   b=-1;
printf("%d\n",b);
return 0;
 
改造后：
#include<stdio.h>
int main();
int a=0;
int b=0;
scanf("%d%d",&a,&b);
b=a>10 ? 1:-1;
printf("%d\n",b);
return 0;
}
