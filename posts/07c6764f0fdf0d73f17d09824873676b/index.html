看起来该网站正在使用Jquery来执行交互和动画。 你可以这样做：

$('#button_id').on('click', clickButton);

function clickButton()

{

$(this).stop().slideToggle();

}

其中button_id是用户将单击的页面上元素的ID。

有关jquery slideToggle的更多信息，请访问： http ：//api.jquery.com/slidetoggle/
