解决 webstorm nodejs 在使用 require 的时候出现下划曲线，没有代码提示的问题

问题

如图，在 webstorm 中写 nodejs 脚本的时候， require 出现下划曲线提示，并且没有代码提示

解决办法

如果你的系统中已经安装了 nodejs 和 npm
打开设置，Languages & Frameworks -> Nodejs 将 Coding assistance for Node.js 勾选上即可

不需要往： JavaScript -> Libraries 添加 nodejs

结果

下划曲线没有了，代码也有提示了
