2. 核心代码

# 修改背景颜色为黑色
pixels = new_im.load()
width, height = new_im.size
r1, g1, b1 = pixels[0, 0]
for x in range(width):
     for y in range(height):
         r, g, b = pixels[x, y]
         if r == r1 and g == g1 and b == b1:  
            pixels[x, y] = (0, 0, 0)

原理就是：获取某一个点的rgb值（比如我这里以左上角，也就是背景颜色），然后把等于这个颜色的值都设置成黑色，就达到改造的目的。
