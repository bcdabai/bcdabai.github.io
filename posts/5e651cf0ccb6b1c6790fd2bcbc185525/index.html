1. 插件申请接入

首先要在小程序管理后台的“设置-第三方设置-插件管理”中添加插件。开发者可登录小程序管理后台，通过 appid【wx6885acbedba59c14】 查找插件并添加。等待插件开发者通过后，方可在小程序中使用相应的插件。

代码示例

"mp-weixin": {
  ...........

      "version": "1.1.2",

  }
}
