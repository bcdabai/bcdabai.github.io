FileOutputStream文件写入类

这个类是为了方便我们对将数据写入到文件中，

它的构造方法有两种，

一种是传入一个人File类

 File file = new File("D:\\我的电脑\\asd\\a.txt");
        try {
            FileOutputStream fos = new FileOutputStream(file);
//FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt");
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

也可以直接传入一个String路径 

 File file = new File("D:\\我的电脑\\asd\\a.txt");
        try {
            FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt");
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

写入文件的方法是write()重载了两次，分别用来写入一个字节，一串字节，和指定数量字节)

package io;

import java.io.FileOutputStream;
import java.io.IOException;

/**
 * @ClassName IoFile
 * @Author 瞿肖
 * @Date 2022/7/5 9:08
 */
public class IoFile {
    public static void main(String[] args) {
        try {
            FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt");
            fos.write(98);
            fos.close();//写入完毕后要关闭资源close()
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

}

写入一串字节，通常拿来写入字符，利用String的getBytes(StandardCharsets.UTF_8)

方法获取，如果为负数时，java在写入的时候会拼接下一个字节，三个字节一个汉字

package io;

import java.io.FileOutputStream;
import java.io.IOException;
import java.nio.charset.StandardCharsets;

/**
 * @ClassName IoFile
 * @Author 瞿肖
 * @Date 2022/7/5 9:08
 */
public class IoFile {
    public static void main(String[] args) {
        byte[] arr="参数".getBytes(StandardCharsets.UTF_8);
        try {
            FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt");
            fos.write(arr);
            fos.close();
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

}

指定写入个数：

package io;

import java.io.FileOutputStream;
import java.io.IOException;
import java.nio.charset.StandardCharsets;

/**
 * @ClassName IoFile
 * @Author 瞿肖
 * @Date 2022/7/5 9:08
 */
public class IoFile {
    public static void main(String[] args) {
        byte[] arr="参数".getBytes(StandardCharsets.UTF_8);
        try {
            FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt");
            fos.write(arr,0,3);
            fos.close();
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

}

 有三个参数，第一个为写入的数据，第二个为开始下标，第三个为写入的字节个数，由于UTF-8是三个字节一个汉字，所以这里只会有一个汉字

上面的方法是重新创建一个文件，将数据输入进去，我们也可以追加写入，只需要在创建对象的时候多传入一个boolean的参数；

package io;

import java.io.FileOutputStream;
import java.io.IOException;
import java.nio.charset.StandardCharsets;

/**
 * @ClassName IoFile
 * @Author 瞿肖
 * @Date 2022/7/5 9:08
 */
public class IoFile {
    public static void main(String[] args) {
        byte[] arr = "1324".getBytes(StandardCharsets.UTF_8);
        try {
            FileOutputStream fos = new FileOutputStream("D:\\我的电脑\\asd\\a.txt", true);
            fos.write(arr);
            fos.close();
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }
}
