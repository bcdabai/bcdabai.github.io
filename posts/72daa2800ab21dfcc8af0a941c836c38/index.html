01 Rust 即时模式 GUI 库

egui 是一个简单、快速且高度可移植的 Rust 即时模式 GUI 库，可以轻松地将其集成到你选择的游戏引擎中，旨在成为最易于使用的 Rust GUI 库，以及在 Rust 中制作 Web 应用程序的最简单方法。

项目地址：https://github.com/emilk/egui
