什么是占位符

1、概述

	占位符通常出现在sql语句中，目的是提高代码的复用性，和执行效率，我们可以通过占位符
来对数据先不处理，后续输入。

2、例子

  1. 什么是占位符
	//在这个sql语句中?即为占位符，对于这个完整的sql语句，我们不处理数据，先把框架写出来
	String sql = "update student set name = ? where id = ?";
