记录一个mysql update a,b a and b 的问题

mysql :   update    table  set    a = xxxxxxx  and  b =xxxxxxxx   。 这样  只会  执行 set  a=xxxxxxxx   而且 不会赋值xxxxxxxx   而是赋值 0！....

所以正确的应该是  mysql:   update  table set a  = xxxxxx  ,  b=xxxxxx
