设置注释文件模板

Ctrl + Alt + S 打开设置，Editor - File and Code Templates 选择class、interface、enum根据自己需要选择需要添加注释的文件，依次添加如下配置内容

/**
 1. @ClassName ${NAME}
 2. @Description TODO
 3. @Author ${USER}
 4. @Date ${DATE} ${TIME
 5. @Version 1.0
*/
