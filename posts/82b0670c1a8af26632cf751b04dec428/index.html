SQL语句把字符串类型数据转换成日期类型数据
查询某个时间段范围可以按照下面方式
select * from xxxtableName where date(字符串类型日期字段) >= date(“2020/01/01 00:00:00”)
查询统计某个字段在这个时间范围方法：
select count(xxx字段)，xx字段，xx字段 from xxxtableName where date(字符串类型日期字段) >= date(“2020/01/01 00:00:00”) group by xxx字段
