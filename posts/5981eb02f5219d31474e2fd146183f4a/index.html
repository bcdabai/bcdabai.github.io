splice删除数组中的指定的某些元素

splice删除我们常见的是如下，我们要删除addr数组中，包含属性a为空的元素

for(let i=0;i<==arrd.length-1;i++){
	if(addr[i].a==''){
		addr.splice(i, 1)
	}
}

但是这样是不能删除数组中所有a属性为空的数组的，因为splice会改变原数组，会导致i的变化，所以我们可以使i一直等于数组的长度-1

for(let i<=arrd.length-1;i>=0;i--){
	if(addr[i].a==''){
		addr.splice(i, 1)
	}
}
