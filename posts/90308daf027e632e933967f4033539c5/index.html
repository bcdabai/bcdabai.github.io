1. 显卡驱动

1.1. 各版本下载地址

https://www.nvidia.com/Download/Find.aspx?lang=zh-cn

1.2. 各版本文档地址

在下载页面选择具体驱动版本后，在 其他信息 框即可找到 README 文档链接

1.3. 安装、卸载方式

以 535.129.03 版本为例（文档地址：https://download.nvidia.com/XFree86/Linux-x86_64/535.129.03/README）

在 Chapter 4. Installing the NVIDIA Driver 安装章节中即可看到:

# 下载后的安装命令
sudo sh NVIDIA-Linux-x86_64-535.129.03.run


# 卸载命令
sudo sh NVIDIA-Linux-x86_64-535.129.03.run --uninstall

2. CUDA

2.1. 各版本下载地址

https://developer.nvidia.com/cuda-toolkit-archive

推荐使用 runfile 安装包。


2.2. 各版本文档地址

https://developer.nvidia.com/cuda-toolkit-archive

2.3. 安装、卸载方式

以 11.8 版本为例（文档地址：https://docs.nvidia.com/cuda/archive/11.8.0/）

在 Installation Guide Linux - 8. Runfile Installation 安装章节中即可看到:

# 下载后的安装命令
sudo sh cuda_11.8.0_520.61.05_linux.run


# 卸载命令
sudo /usr/local/cuda-11.8/bin/cuda-uninstaller
