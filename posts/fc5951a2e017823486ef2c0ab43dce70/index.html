在python中执行其他python文件的一种方法

可以利用模型导入的方法
要运行的其他代码need_run.py

print("hellow world")
a = 1

可以采用如下方式运行并保存变量:

  1. 把need_run.py内容换成函数
def aa():
	print("hellow world")
   result = 1
   return result
  1. 在另一个python文件中运行如下代码(导入模块)
from  need_run import aa
res = aa()
print(res)
