python webservice 安装suds提示ModuleNotFoundError: No module named 'client'

在开发python webservice测试接口时遇到的问题

安装 suds 时提示 ModuleNotFoundError: No module named 'client'  ，没有模块叫 client 的

需要先安装 client ,然后再安装 suds 

pip install client

pip install suds
