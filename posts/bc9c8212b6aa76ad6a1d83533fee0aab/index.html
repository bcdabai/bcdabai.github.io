Vue知识点：==与===的异同点

==与===的区别

=====
相同点两者均是用于比较判断相等情况，返回boolean值
不同点用于比较或判断两者是否相等，比较时可以自动转换数据类型用于较为严格的比较，除判断数据是否相等外，还会判断两者的数据类型是否相同，且不会进行自动类型转换
<template>
    <div>
        1==='1' {{1==='1'}}
        <br/>
        1=='1' {{1=='1'}}
    </div>
</template>

!=与!==的区别

!=!==
相同点与==、===相对的是!=、!==，两者也是用于比较判断，比较不相等的情况，返回boolean值
不同点用于比较或判断两者是否不相等，比较时可以自动转换数据类型用于较为严格的比较，除判断数据是否不相等外，还会判断两者的数据类型是否相同，且不会进行自动类型转换

if(!value)

当value 值为undefined、null、0、""、false时，转换成boolean，其值为false

!  取反运算符

value为false，!value为true；value为true，!value为false。

代码示例：

<script>
export default {
  name: "index",
  data() {
    return{
      testVal:null
    }
  },
  created(){

    this.test()
  },

  methods: {
    test(){
      if(!this.testVal){
        console.log('this.testVal='+this.testVal,'!this.testVal='+!this.testVal)
      }
    }
 }
}
</script>
