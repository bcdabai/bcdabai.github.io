【C语言】输入年月日，输出这天是这一年的第几天。

总结：利用数组、循环结构解决问题，闰年判断是关键。

代码如下：

#include<stdio.h>
main()
{
	int a[]={0,31,28,31,30,31,30,31,31,30,31,30,31} ;
	int n, y, r, i, sum=0;
	printf("请输入年月日：");
	scanf("%d%d%d",&n,&y,&r); 
	for(i=0;i<y;i++)
		sum+=a[i];
		
	sum+=r;
	if((n%400==0||n%4==0&&n%100!=0)&&y>2)	
		sum+=1;
	printf("%d年%d月%d日是该年的%d天",n,y,r,sum);
	
 }
