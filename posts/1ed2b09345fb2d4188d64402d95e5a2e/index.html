核桃的数量---蓝桥杯

 思路：

题目所代表的意思就是a,b,c这三个必须是核桃数量的因子，即a,b,c三个的最小公倍数

#include <iostream>
#include <algorithm>
using namespace std;
//
int main()
{  int a,b,c;
   cin>>a>>b>>c;
   int d=a*b/__gcd(a,b);
   int e=d*c/__gcd(c,d);
   cout<<e;
  // 请在此输入您的代码
  return 0;
}
