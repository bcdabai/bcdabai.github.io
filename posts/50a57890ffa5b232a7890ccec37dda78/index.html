任意输入一个整数，编程判断它的奇偶性。

#include<stdio.h>
int main()
{
   int a;
   printf("Iputa:");
   scanf("%d",&a);
   if(a%2==0)
   {
      printf("a是偶数");
   }
   else
   {
      printf("a是奇数");
   }
   return 0;
}

知识点：
1.if-else语句用于解决双分支选择的问题；
2.在if-else语句中，在两条语句中选择其中的一条来执行。

心得体会：
1.学会使用双分支条件语句编程；
2.在if-else语句中，如果表达式的值为真，执行语句1，否则执行语句2。
