iOS 完全复制UIView

如果要完全复制一个UIView和对象的时候可以使用对象序列化方法

  1. // Duplicate UIView  
  2. - (UIView*)duplicate:(UIView*)view  
  3. {  
  4.     NSData * tempArchive = [NSKeyedArchiver archivedDataWithRootObject:view];  
  5.     return [NSKeyedUnarchiver unarchiveObjectWithData:tempArchive];  
  6. }
