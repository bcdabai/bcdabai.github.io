tomcat面向接口的java反射

//Propertise 存储浏览器路径与servlet对应关系
Propertise pro = new Propertise();
//通过key获取value值
String classServletName = pro.getProperty(requestPath);
//反射机制创建对象
Class c = Class.forName(classServletName);
//面向接口强转
Servlet servlet = (Servlet)c.newInsatance();
//调用servlet接口方法。
servlet.service();
