Go学习(编译运行Go程序)

1.创建一个go文件hello.go
输入以下代码

package main
import "fmt"
func main(){
    fmt.Println("Hello")
}

2.切换到该文件所在目录
直接使用go run命令运行

go run hello.go

或
先编译，生成hello的二进制文件，然后通过./hello运行

go build hello.go
./hello
