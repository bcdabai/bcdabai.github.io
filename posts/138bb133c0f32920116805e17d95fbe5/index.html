[Python] Python脚本中执行另一个Python脚本

关于Python脚本如何执行另一个脚本，可以使用os.system()来实现

os.system(path)

os.system()的参数:执行命令 + 执行内容 

# os.system('python 脚本文件名')
import os
# 错误写法
os.system(r'E:\start.py')       
# 正确写法
os.system(r'python E:\start.py')

案例

在同一目录下有两个Python脚本文件，其中ATest.py执行BTest.py 

ATest.py

import os
res = os.system('python BTest.py')
# 打印执行结果 0表示success,1表示fail
print(res) # 0

BTest.py 

print("My name is BTest.py")
