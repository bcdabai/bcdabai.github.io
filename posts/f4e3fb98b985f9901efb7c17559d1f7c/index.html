银河麒麟系统飞腾CPU安装达梦数据库

之前在国产银河麒麟服务器部署项目， 这里记录一下安装达梦的流程。

安装流程

进入/usr/local目录

cd /usr/local

创建dm文件夹

mkdir dm

上传并解压达梦安装包：dm7_20210712_FTarm_kylin4_64_ent_7.6.1.108.iso

mount -o loop dm7_20210712_FTarm_kylin4_64_ent_7.6.1.108.iso /usr/local/dm

进入解压目录

cd /usr/local/dm

执行安装命令

./DMInstall.bin -i

基础配置

选择安装语言：c
是否输入key文件路径：n
是否设置时区：y
选择时区：21(GTM+08=中国标准时间)
选择安装类型：1(典型安装)
选择安装目录：/usr/dameng/dm
是否确认安装路径：y
是否确认安装：y
(/etc/dm_svc.conf已存在，是否进行替换：y)

初始化数据库

进入bin目录下，执行命令
./dminit

初始化配置

input system dir：/usr/dameng/dm/data
input db name：28S
input port num：5236
input page size：8
input extent size：16
input time zone：+8

string case sensitive 是否区分大小写：n
length in char：y
enable database encrypt：n
page check mode：0
input elog path：/usr/dameng/dm/log/db01.log
auto_overwrite mode：0

如果现在bin目录下没有达梦启动服务文件：DmServiceDMSERVE
则需要去安装达梦启动服务

cd dmdbms/script/root
./dm_service_installer.sh -i /usr/local/dm/dmdbms/data/28S/dm.ini -p DMSERVER -t dmserver

进入bin目录启动达梦服务

启动服务：service DmServiceDMSERVE start
查看启动状态：service DmServiceDMSERVE status
停止服务：service DmServiceDMSERVE stop

连接达梦

cd 到达梦安装目录下的tool目录
调用连接：./disql
连接服务：conn SYSDBA/SYSDBA@localhost:5236

创建用户

create user "用户名" identified by "密码" default tablespace "MAIN" default index tablespace "MAIN";

授权

grant "RESOURCE","PUBLIC","DBA" to "用户名";

退出连接，到bin目录导入数据库

同一用户导入：./dimp userid=用户名/密码@localhost:5236 file=/software/imp_exp_latest.dmp full=y

(不同用户模式之间的导入：./dimp USERID=SYSDBA/SYSDBA@127.0.0.1:5236 FILE=/数据库文件路径/xxx.dmp FULL=Y REMAP_SCHEMA=原用户名:现用户名)
