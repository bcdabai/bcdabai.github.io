1、一般情况

1.1 进入设置选择 Region&Language

1.2 点击 加号

1.3 点击 汉语（中国）

1.4 选择 汉语（智能拼音）

2、特殊情况

有些虚拟机可能没有 汉语（智能拼音）的这一项

2.1 安装

管理员权限执行：

yum install ibus ibus-libpinyin

2.2 设置

安装完后就会发现有 汉语（智能拼音）的这一项 。

注意：如果还是没有重启一下

3、进一步设置

3.1 选择 汉语（智能拼音）

3.2 可以根据需求设置

4、最后切换输入法

按键作用
shift + super + 空格切换至上个输入源
super + 空格切换至下个输入源

super 键 在 w i n d o w window window 系统中即为 window 键。
