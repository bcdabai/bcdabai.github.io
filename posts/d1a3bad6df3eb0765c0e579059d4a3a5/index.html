浙大版《C语言程序设计（第4版）》练习5-2 找两个数中最大者

本题要求对两个整数a和b，输出其中较大的数。

函数接口定义：

int max( int a, int b );

其中a和b是用户传入的参数，函数返回的是两者中较大的数。

输入样例：

-5 8

输出样例：

max = 8

 【代码1】

int max(int n, int m)
{
	int max = 0;
	if (n < m)
	{
		max = m;
	}
	else 
    {
		max = n;
	}
	return max;
}

【代码2----运用三目操作符】

int max( int a, int b )
{
    return (a > b) ? a : b;
    //解释 (a > b) ? a : b --->a大于b吗，如果这句话为真则整个表达式的值为a,否则为b.
}
