nrm安装之后无法使用的问题

  • 安装nrm: npm install -g nrm
  • 安装结果正常尝试使用: nrm ls 然后报错
  • 在网上找了一下资料, 找到一种方法: 找到nrm安装目录下的cli.js, 17行, 将变量NRMRC的值由path.join(process.env.HOME, '.nrmrc');
    改为path.join('HOME', '.nrmrc'); 之后再次尝试使用, 结果正常
