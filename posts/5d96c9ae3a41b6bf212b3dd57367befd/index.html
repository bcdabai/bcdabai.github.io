求四个数的最大数#C语言

题目描述

已知有四个不等的数，将其中的最大数找出来。

输入

输入只有一行，包括4个整数。之间用一个空格分开。

输出

输出只有一行（这意味着末尾有一个回车符号），包括1个整数。

样例输入 
1 9 8 6
样例输出 
9

代码如下

#include<stdio.h>
int main()
{
	int a[4];
	int max = 0;
	for (int i = 0; i < 4; i++)
	{
		scanf("%d", &a[i]);
	}
	for (int i = 0; i < 4; i++)
	{
		if (a[i] > max)
		{
			max = a[i];
		}
	}
	printf("%d", max);
	return 0;
}
