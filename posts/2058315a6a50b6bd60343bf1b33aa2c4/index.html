配置负载均衡

（1）准备两台 tomcat 服务器，一台 8080，一台 8081

（2）在两台 tomcat 里面 webapps 目录中，创建名称是 paga文件夹，在 page文件夹中创建页面 a.html，用于测试

在 nginx 的配置文件中进行负载均衡的配置

upstream myserver{
   #  ip_hash;
   server 192.168.56.10:8080;
   server 192.168.56.10:8081;


   proxy_pass http://myserver;

    }
