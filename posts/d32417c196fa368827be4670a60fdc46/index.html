【Vue】Vue CLI 3 使用

vue cli 3来初始化项目脚手架跟旧版本的稍微有些区别，以下是使用步骤

1.Install

npm install -g @vue/cli
# OR
yarn global add @vue/cli
2. Create a project:
vue create my-project
# OR
vue ui
3.Run
npm run serve

参考官网 https://cli.vuejs.org/
