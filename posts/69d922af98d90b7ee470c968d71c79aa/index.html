如何在VS2017中单步调试第三方DLL

用VS进行项目开发的时候，经常会用到第三方DLL。既然是项目开发，就免不了调试，那么如何能像调试本机代码一样方便的调试第三方DLL呢？

总结一下：

（1）下载安装.Net Reflector插件

 ( 2 )  打开.Net Reflector Object Browser



 

( 3 )  找到需要调试的DLL右键Enable Debuging




 （4）接下来打上断点就可以单步调试了
