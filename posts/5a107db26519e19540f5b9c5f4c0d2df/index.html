一、进程相关内容

1.1 查看进程启动时间 和 运行时间

查看进程启动时间

  • 命令：ps -eo pid,lstart,cmd
  • 某个进程：ps -eo pid,lstart,cmd | grep udhcp


查看进程运行时间

使用 ps -p <进程ID> -o etime 来查看某个进程运行的时间。

1.2 Linux 进程常见状态（R、S、D、T、Z、X）

  1. R (Running)运行状态：正在运行或在运行队列中等待执行的进程。
  2. S (Sleeping)睡眠状态：处于休眠状态或等待某个事件的进程。当进程被唤醒后，它将切换到R状态。
  3. D (Disk sleep)磁盘睡眠状态：等待磁盘I/O操作完成的进程。不可中断睡眠（深度睡眠，不可以被唤醒，通常在磁盘写入时发生）
  4. T (Stopped)停止状态：进程被停止或收到了SIGSTOP、SIGTSTP、SIGTTIN或SIGTTOU信号而暂停执行。
  5. Z (Zombie)僵尸状态：已经终止但是父进程还没有回收其资源的进程。
  6. X (Dead)死亡状态：进程已经终止。
