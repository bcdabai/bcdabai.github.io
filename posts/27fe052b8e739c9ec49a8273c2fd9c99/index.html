ubuntu下Vi编辑器方向键和退格键问题

刚刚新下载了ubuntu20.04的系统，感觉图形界面确实比centos7好用。但是又连续出现了很多问题（我太菜啦）。

问题描述

vi编辑器插入模式下，方向键分别对应ABCD以及退格键失效。以为是自己的问题qwq试了好几次，然后搜了一下是因为没有下载完整的vim包导致的。

解决方案

编辑/etc/vim下的vimrc.tiny文件

sudo vi /etc/vim/vimrc.tiny

将“set compatible改成 set nocompatible ，然后添加 set backspace=2即可解决

建议在命令模式下dd删除set compatible这一行，然后再粘贴（vi操作不熟悉的我qwq）
