初识while循环语句

一、

（1）用while循环语句输出1-10的数字

代码如下

#define  _CRT_SECURE_NO_WARNINGS
#include <stdio.h>


int main()
{
	int i = 1;
	while (i <= 10)
	{
		printf("%d", i);
		i++;
	}
}

输出结果如下

之前在分支语句中了解到了关键词break和 continue在程序中的作用，那么这两个关键词在while语句中有什么作用呢？

（2）break关键词在while循环语句中作用

在上述程序段中加入break，代码如下

#define  _CRT_SECURE_NO_WARNINGS
#include <stdio.h>


int main()
{
	int i = 1;
	while (i <= 10)
	{
        if(i==6)
        break;
		printf("%d", i);
		i++;
	}
}

输出结果如下

 可以看见break在while循环语句中作用是用于永久的终止循环。

(3）在while循环语句中加入continue关键字

将上述代码中break换成continue，代码如下

#define  _CRT_SECURE_NO_WARNINGS
#include <stdio.h>


int main()
{
	int i = 1;
	while (i <= 10)
	{
        if(i==6)
        continue;
		printf("%d", i);
		i++;
	}
}

输出结果如下

 在while循环中，continue作用是跳过本次循环continue后边的代码
直接去判断是否去执行下一次代码
