安装pyCharm后pip不能用，显示No module named pip如何解决?

我个人感觉应该是安装pycharm的时候卸载了pip的有关内容。若出现了这种情况，则运行以下指令：

python -m ensurepip

easy_install pip

python -m pip install --upgrade pip
