Python3.8解决No module named 'numpy'报错

报错：
解决：
python -m pip install --upgrade pip
pip install numpy
（pip是python包管理工具）
安装好执行
python
import numpy
没报错即可

不行的话（强行安装更新更高的版本）

sudo pip install numpy --ignore-installed numpy
