在一个元素上:hover，改变另一个元素的css属性

如果二者是父子关系，可以写成这种：

  .face:hover .eye-bottom {
                    margin-top: 30px;
                }

如果是兄弟关系：

.face:hover+.ear-wrap {
                transform: rotate(-30deg);
            }

如果是兄弟的儿子：

 .face:hover+.ear-wrap .ear {
                transform: rotate(-30deg);
            }

是用css的选择器来实现的。
