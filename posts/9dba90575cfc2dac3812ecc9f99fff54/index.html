运行vue报错npm ERR! A complete log of this run can be found in解决办法

在这里我们需要清除npm的缓存:

（1）在cmd命令行窗口中输入：npm cache clean --force
（2）然后再运行我们需要安装模块的命令，输入npm install。
有时是网络问题，依赖包加载不完整，删掉node_modules文件后，重新执行npm install即可。
