C语言输出图形：宝塔形（三角形）回文数字。即：第一行1，第二行121，第三行12321……

效果图：

代码部分：

#include <stdio.h>
void main()
{   
    int i,j;
    for(i=1;i<=4;i++)
    {
        for(j=1;j<=10-i;j++)
            printf(" ");
        for(j=1;j<=i;j++)
            printf("%d",j);
        for(j=i-1;j>=1;j--)
            printf("%d",j);
        printf("\n");   
    }
}
