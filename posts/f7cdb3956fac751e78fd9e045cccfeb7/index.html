requires Python ‘＞=3.7‘ but the running Python is 3.6.9 问题

过程

  • ubuntu18.04 使用如下命令安装protobuf

 pip3 install protobuf

  • 安装完毕后报错

protobuf requires Python '>=3.7' but the running Python is 3.6.9

解决

  • 更新pip

python3 -m pip install --upgrade pip

  • 再次安装之前安装的module

pip3 install protobuf

  • 不再报错
