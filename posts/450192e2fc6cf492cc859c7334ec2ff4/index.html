【完美解决·离线安装】Pip安装时遇到Microsoft Visual C++ 14.0 is required错误，下载离线安装包安装即可（2023年11月8日更新资源）

问题：

Windows环境下，在使用pip安装第三方Python库时，部分库会下载压缩包进行本地编译，才能运行，此时若缺少相关环境会提示Microsoft Visual C++ 14.0 is required。

解决方案：

（1）下载编译好的.whl文件

① 打开链接：点击跳转

② 选择相应Python环境与库版本，下载。

③ 使用pip install安装本地.whl即可。

（2）下载Microsoft Visual C++ Build Tools安装相应环境（推荐）

但是，从网络上正常下载相应Bulid Tools，打开后会提示安装包丢失或毁坏。

所以，这里提供了一个完整的离线安装包。
