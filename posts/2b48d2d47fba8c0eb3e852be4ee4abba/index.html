Linux_虚拟机常用目录汇总

根目录（cd /）：/ 表示根目录，cd和 / 之间有个空格！

用户目录（cd ~）：~ 表示用户目录，也称为家目录。cd 和 ~ 之间有个空格！

当前路径：执行 pwd 指令即可看到当前所在的路径信息。

绝对路径：只能由根目录写起，例如写到 alpha 目录下。

cd /home/alientek/alpha   //cd 和 / 之间有一个空格

相对路径：不是由根目录写起，例如写到 alpha 目录下。

cd ~/alpha

用户主目录文件：~/alpha

cd ~/alpha/

tree -L 2

用户主目录文件：~/linux

cd ~/linux

tree -L 2
