git命令 拉/推项目（精简）

1. 拉项目到本地

git init  // 初始化

git remote add origin http://XXX.git  // 连接仓库地址

git pull origin master  // 拉取项目

2.项目推远程仓库

git add .  // 全添加

git status  // 查看(可省略)

git commit -m "提交说明" 

git push origin master  // 推送到远程master分支

 3.报错 解决方法

git branch -m master main  // 把本地master仓库名称 修改为远端的main 

git push origin main  // 推送到远程main分支
