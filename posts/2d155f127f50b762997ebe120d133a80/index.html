extjs ajax请求返回值,Extjs4.0以上版本 Ext.Ajax.request请求的返回问题

Ext.Ajax.request({

url: posturl,

method: ‘POST‘,

params: {

ClassName: ‘XXXX‘,

FuncName: ‘XXXX‘,

param: param

},

success: function (response) {

var text = response.responseText;

if (text == "false") {

Ext.MessageBox.hide();

Ext.MessageBox.alert("警告", "此班级还有学员,无法删除!");

} else {

Ext.MessageBox.hide();

Ext.MessageBox.alert("成功", "删除成功!");

grid.getStore().load();

}

},

failure: function () {

Ext.MessageBox.hide();

Ext.MessageBox.alert("警告", "请求超时!");

}

});

如上代码,success和failure 返回的问题,

Ext.Ajax.request这个请求,只要没有超时,不论返回{success:true} 还是{success:false}

都会进入到success的方法,只有在请求超时的情况下,才会进入failure

表单的submit方式则不同,返回{success:true}进入success 返回{success:false}

进入failure
