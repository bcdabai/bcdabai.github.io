MongoDB数据批量更新语句

最新遇到mongoDB中需要对数据做批量更新，使用updateMuti效果不是特别理想，所以检索了下检索语句，发现mongo提供的原生document文档是真的好用，批量更新语句如下，每天进步一小点，加油！

db.getCollection('picture').find({
    "ss": 0
}).forEach(function(item) {
    db.getCollection('picture').update({
        "_id": item._id
    }, {
        $set: {
            "ss": 1
        }
    })
});
