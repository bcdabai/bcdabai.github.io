微信小程序跳转外部网站

以跳转百度为例，点击一个按钮，跳转到百度（百度的网页还是在小程序中打开） 

wxml:

<view >
    <button bindtap='to'>点击跳转</button>
  </view>

js:

 to: function (e) {
    wx.navigateTo({
      url: '/pages/out/out',
    })
  },


  要跳转的百度页面：out.wxml  (百度页面)，js,wxss,json略。


<web-view src="https://www.baidu.com/"></web-view>


要勾选上开发者工具详情里面，不检验合法域名、业务域名这个选项。
