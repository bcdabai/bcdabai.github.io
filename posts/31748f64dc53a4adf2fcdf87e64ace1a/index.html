寻找水仙花数
代码如下（示例）：

#寻找水仙花数
nums <-100:999
first.num <- nums %/%100
second.num <- (nums%%100)%/%10
third.num <- nums%%10
nums [(first.num^3 + second.num^3 + third.num^3) == nums]

运行结果
判断因子
代码如下（示例）：

nums<-1:6
c[6%%c==0]

判断完数
代码如下（示例）：

nums <- 1:(6/2)
sum(nums[6%%nums == 0])== 6
