npm ERR! code 1

新建VUE项目在执行npm install命令时，项目报错 npm ERR! code 1

报错信息

 解决办法：

1、删除 node_modules 文件夹

2、删除 package-lock.json 文件（有些文章说删除 node_modules 文件夹就可以，但是我删除后还是一样）

3、执行 npm install 命令

补充：

        如果运行时遇到版本问题，执行命令报错 npm ERR! code ERESOLVE，可以执行 

 npm install --legacy-peer-deps 命令 。

 

实验命令（不管用）：

        npm i -g npm

        npm cache clean --force
