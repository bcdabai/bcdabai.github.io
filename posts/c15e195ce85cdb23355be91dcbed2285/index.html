将分支A上本地修改且未提交的代码转移到分支B上

需求：在A分支上工作，本地修改了代码还没有提交到远程仓库，现在要将这些修改转移到B分支上。

1.在A分支上暂存： git stash

2.切换到B分支:        git checkout  B

3.在B分支释放暂存：  git stash pop

这样就将A分支代码转移到B分支的本地工作区了，且并未提交。
