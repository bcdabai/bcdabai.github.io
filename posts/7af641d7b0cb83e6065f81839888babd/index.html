java循环语句实现自然数1~99的遍历求和

1.1~99的全遍历求和

注意:

sum变量要写在外面

main方法必须有 

public class Leijia {
    public static void main(String[]args){
        int sum=0;
        for (int i=1;i<=99; i++){
            
            sum=sum+i;
            
        } 
        System.out.println("s="+sum);
    }
}

2.  1~99只遍历奇数的求和

public class Leijia {
    public static void main(String[]args){
        int sum=0;
        for (int i=1;i<=99; i++){
            if(i%2==1){
                sum=sum+i;
            }   
        } 
        System.out.println("s="+sum);
    }
}
