洛谷刷题-P5708 【深基2.习2】三角形面积

AC代码：

#include <stdio.h>
#include <math.h>
int main()
{
	double a,b,c;
	double p,s;
	scanf("%lf %lf %lf",&a,&b,&c);
//	要判断能不能组成三角形
	if(((a+b)>c)&&((a+c)>b)&&((b+c)>a))
	{
	p=1.0/2*(a+b+c);
	s=sqrt(p*(p-a)*(p-b)*(p-c));
	printf("%.1lf",s);
	} 
	else
	printf("");
	return 0;
 }

a,b,c没有用double时是WA，csdn中有文章表面，用double是为了不丢失精度。
