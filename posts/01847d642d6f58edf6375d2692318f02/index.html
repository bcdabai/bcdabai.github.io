sqlplus显示乱码--Windows10 处理方法

设置环境变量 export NLS_LANG="AMERICAN_AMERICA.AL32UTF8"

格式：NLS_LANG=NLS_LANGUAGE_NLS_TERRITORY.NLS_CHARACTERSET

执行下面的SQL找出这3个值设置环境变量后，重新开启一个CMD窗口登陆sqlplus问题解决。

SQL> select * from nls_database_parameters where parameter in 

('NLS_LANGUAGE','NLS_TERRITORY','NLS_CHARACTERSET');

PARAMETER
--------------------------------------------------------------------------------
VALUE
--------------------------------------------------------------------------------
NLS_LANGUAGE
AMERICAN


NLS_TERRITORY

AMERICA

NLS_CHARACTERSET

AL32UTF8
