pip安装依赖疑难杂症处理

杂症1：XXX.whl is not supported …

报错长以下这样

  • ERROR: Failed building wheel for XXX-
  • ERROR: xxx.whl is not a supported wheel on this platform.

解决方案

  1. 去清华源地址找到对应的包下载到自定义文件夹并解压
    快速查找：https://pypi.tuna.tsinghua.edu.cn/simple/xxx xxx为你的包名
  2. cd 到解压的包里边
  3. 执行 python setup.py install
  4. 成功

其他问题大多数可以通过升级pip或者换源解决，欢迎留言你遇见的各种pip安装问题，我会定期在这里更新解决方案
