yum查看可用的软件包

显示系统中可用的软件包及其版本：

yum list available --showduplicates  | grep 'bash-completion'

在yum软件包管理器中搜索能够提供"tree"命令的软件包，并列出这些软件包的名称和版本信息：

yum provides tree

启用 EPEL 存储库：

yum install epel-release
