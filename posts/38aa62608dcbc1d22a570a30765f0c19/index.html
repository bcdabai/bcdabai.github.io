java求输入一个正整数，计算并输出整数的各位数字之和

输入一个正整数，计算并输出整数的各位数字之和。

如输入12345，结果显示15，如下图所示。

【代码如下】：

import java.util.Scanner;
public class NumSum{
  public static void main(String[] args){
    Scanner sc=new Scanner(System.in);
    int num=sc.nextInt();
    int sum=0;
    int flag=10;
    while(num>0){
      sum=sum+num%flag;
      num=num/flag;
    }
    System.out.println(sum);
  }
}
