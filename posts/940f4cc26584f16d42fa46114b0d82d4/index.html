2 安装

2.1 解压到指定路径

  • 解压到：C:\Program Files\OpenSSH

2.2 CMD 到 OpenSSH 目录下

  • 快捷键：Win + r 调出 命令运行窗口，并执行下列命令
cd C:\Program Files\OpenSSH-Win64

2.3 安装 sshd 服务

powershell.exe -ExecutionPolicy Bypass -File install-sshd.ps1

2.3 开放端口 22

netsh advfirewall firewall add rule name=sshd dir=in action=allow protocol=TCP localport=22

22 是 SSH 的默认端口号

2.4 配置开机自启 sshd 服务

sc config sshd start= auto

2.5 配置环境变量 path

将 C:\Program Files\OpenSSH 添加到 path 中，免得每次都要切到 C:\Program Files\OpenSSH 才能使用 ssh
启动 ssh 服务

net start sshd
