Python操作环境变量

1、使用os读取环境变量

import os
os.getenv('path')
os.environ.get('path')
os.environ['path']

2、遍历打印所有环境变量
通过访问os.environ可以获取整个系统环境变量。

import os
env_dist = os.environ
for key in env_dist:
    print key + ' : ' + env_dist[key]

3、添加新的环境变量

git_path = 'C:\\Program Files (x86)\\Git\\bin'
os.environ["PATH"] = git_path + ";" + os.environ["PATH"]
