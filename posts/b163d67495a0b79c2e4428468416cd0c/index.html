java算法总结之二分查找法

前提:要实现二分查找法的前提是有序的顺序存储结构.
思想:二分查找法的实现思想是:将有序的集合分成三部分,前半部分，中间点,后半部分,首先用要查找的数据与中间点索引处的数据做比较，如果比中间点索引处的数据小，就从前半部分查找，在前半部分得到新的中间点比较…以此类推，直到找到为止。如果比中间点索引处的数据大，就从后半部分查找。

实现方式分为迭代和循环两种方法。

迭代实现

private void binarySerach(int[] array, int leftIndex,int rightIndex, int searchNum){
    if(leftIndex<rightIndex){
     system.out.println("not find"+searchNum);
     return;

    if(searchNum<array[middleIndex]){

    }else if(searchNum>array[middleIndex]){

    system.out.println("find data"+middleIndex);
    }
}
