MySql 8 如何开启慢日志

1. 查询慢日志是否开启   

show VARIABLES like 'slow_query_log%'

  slow_query_log   （ON-开启  OFF-关闭）

  slow_query_log_file  （慢日志存放地址）

2.开启慢日志   

 set global slow_query_log='ON'

3. 查询慢日志时间

show VARIABLES like 'long_query_time%';

 4. 修改时间大于10秒记录日志

set global long_query_time=10;

 5. 查询日志存放位置

find / -name iZ2vcay9yqsrl588etpwahZ-slow.log

   注： iZ2vcay9yqsrl588etpwahZ-slow.log    第一步中 slow_query_log_file  的文件名称
