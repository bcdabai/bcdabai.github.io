1. 题目描述

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

示例:

输入: nums = [1,2,3]
输出:
[
[3],
[1],
[2],
[1,2,3],
[1,3],
[2,3],
[1,2],
[]
]

题目链接：中文题目；英文题目

2. 递归（Recursion）

2.1 解题思路

如果做过77. Combinations（组合），那么递归思路是一样的，只是上一题是找长度为k的组合，这题是列举全部长度情况：0 ~ n。所以我们可以重复使用上题的代码，只需要在最外层递归套一层循环即可。下面两张图解释了递归的思路，大家可以结合上题的递归思路讲解来加深理解：



注：原图来自参考资料所列举的网页；

2.2 实例代码

class Solution {
    vector<vector<int>> ans;
    vector<int> temp;

    void CombineRecursionMethod(vector<int>& nums, int idx, int k) {
        if (temp.size() == k) { ans.push_back(temp); return; }
        for (int i = idx; i < nums.size(); i++) {
            temp.push_back(nums[i]); CombineRecursionMethod(nums, i + 1, k); temp.pop_back();
        }
    }

public:
    vector<vector<int>> subsets(vector<int>& nums) {
        for (int i = 0; i <= nums.size(); i++) { CombineRecursionMethod(nums, 0, i); temp.clear();}
        return this->ans;
    }
};

3. 迭代（Iteration）

3.1 解题思路

首先我们来看看下面的图片：

通过观察图片，我们可以发现：初始化一个空子集，往所有已经存在的子集中依次添加nums里面的数字，即可生成所有的子集。

注：原图来自参考资料所列举的网页；

3.2 实例代码

class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> ans(1, vector<int>()); // 初始化包含一个空对象的二维数组

        for (int& num : nums) {
            int lenNow = ans.size(); // 固定现有数组的长度，即获取现在一共有多少个子集需要处理
            for (int i = 0; i < lenNow; i++) { vector<int> temp(ans[i].begin(), ans[i].end()); temp.push_back(num); ans.push_back(temp); }
        }

        return ans;
    }
};

4. 位操作（Bit Manipulation）

4.1 解题思路

位操作的思路不难理解，但是非常巧妙，我们看看具体的思路是什么样的：

如果我们把上面递归方法中，需要添加到子集的数用1来表示，不用要的用0表示，那[1,2,3]所有的子集可写成下面这样：

[] - [0, 0, 0]
[3] - [0, 0, 1]
[2] - [0, 1, 0]
[2,3] - [0, 1, 1]
[1] - [1, 0, 0]
[1,3] - [1, 0 ,1]
[1,2] - [1, 1, 0]
[1,2,3] - [1, 1, 1]

大家注意观察右边的数组，是不是很像二进制表示的数字？所以我们可以用二进制来标记哪些数字需要添加，哪些不需要添加。只需要生成相应的数字，通过这种二进制标记法生成对应的子集即可。还不懂の童鞋可以看看下图加深理解：


我提供的代码里面是直接使用数字来进行位操作，不需要把二进制的1和0进行转换成其他标记数组，如果想生成标记数组可以看看参考资料里面最后一种方法的代码。

注：原图来自参考资料所列举的网页；

4.2 实例代码

class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> ans;
        int len = nums.size(), bitMax = 1;

        for (int i = 0; i < len - 1; i++) bitMax = (bitMax << 1) + 1; // 生成nums所表示的最大二进制数

        for (int bitMask = 0; bitMask <= (len ? bitMax : 0); bitMask++) {
            int idx = len - 1, bitManipulation = bitMask;
            vector<int> temp;
            
            // 按照二进制1和0的规律，生成相应子集
            while (bitManipulation) {
                if (bitManipulation & 1 == 1) temp.push_back(nums[idx]);
                idx--;
                bitManipulation >>= 1;
            }
            
            ans.push_back(temp);
        }

        return ans;
    }
};
