formData 传二维数组

在网上找了各种方法，后端都不能正常解析，无意间发现这样可行，记录在册希望可以帮到其他人

let table = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]
const formData = new FormData()

for (let i = 0; i < table.length; i++) {
  const row = table[i]
  
  // 这里的'table'是后端接受的字段名
  formData.append(`table[${i}]`, row.join(','))
}
