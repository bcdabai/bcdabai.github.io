python打包运行在Linux上

1、借助工具PyInstaller

工具在window下可以打为exe，在Linux下可以打为执行程序

# 安装PyInstaller：
pip install pyinstaller
# 进入程序目录，用以下命令执行你的python文件
# 会生成dist目录
pyinstaller yourprogram.py
# 进入dist下yourprogram目录，运行
./yourprogram
