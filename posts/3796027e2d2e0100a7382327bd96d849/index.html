c++ 通过函数获得数组的大小

question ?  怎么样写一个函数返回数组的大小
思考：  1. 因为当把一个数组传入函数的时候， 传入的其实是一个首地址的指针， 所以一般方法不能实现
        2. 但是注意到 c++ 虽然不能通过函数参数得到一个数组对象 ，但是   c++ 可以得到" 一个数组的引用"
        3. 比如：void arrPreffence(int (&arr)[3])   int （&arr）[3]   就是一个具有三个成员的数组的引用
        4. 注意到上面的数组的引用的大小是固定的， 怎么使其一般化   =》   模板函数    
        5. 那么我们可以得到下面的函数
