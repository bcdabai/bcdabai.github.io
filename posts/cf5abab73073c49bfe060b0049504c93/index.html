Mac bash切换zsh 之后报 zsh: command not found:XXX 的解决方案

    • 解决方案
        • 1.第一步
        • 2.第二步
        • 2.第三步

解决方案

将 bash shell 中 .bash_profile 全部环境变量加入zsh shell 中 .zshrc 配置文件里就好

1.第一步

在终端执行

vi  ~/.zshrc

2.第二步

找到 “# User configuration”

在其下面添加下面内容

source ~/.bash_profile

2.第三步

在终端里执行

source ~/.zshrc
