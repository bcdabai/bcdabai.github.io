网站被劫持怎么办？传奇网站打开跳到其他站的解决方法

网站劫持一直是工信部着重打击的，刚开始流量劫持行为还没有被定义为刑事犯罪，不少人把它当做一种快速牟利的手段。2015年流量劫持首次被认定为犯罪，上海浦东新区人民法院判决了全国首起流量劫持刑事案件，两名被告人被判有期徒刑3年，缓刑3年。尤其在sf这个灰色产业链上，劫持相当严重，很多玩家朋友的电脑打开A网站无缘无故就跳到了B网站，令人头疼。今天飞飞和你们分享几种劫持方式和解决办法，但不保证100%有效，可以逐个尝试。

1、HOSTS劫持

HOSTS劫持是早期常用的方法，通过安装一些软件或者外挂修改hosts，这个文件的位置在C:\Windows\System32\drivers\etc下面，如果不显示我们就设置下系统显示隐藏文件，因为有些不法分子把属性设置成了隐藏，如果没有可以手动建一个hosts.txt文件，加入以下内容：

# Copyright (c) 1993-2009 Microsoft Corp.
#
# This is a sample HOSTS file used by Microsoft TCP/IP for Windows.
#
# This file contains the mappings of IP addresses to host names. Each
# entry should be kept on an individual line. The IP address should
# be placed in the first column followed by the corresponding host name.
# The IP address and the host name should be separated by at least one
# space.
#
# Additionally, comments (such as these) may be inserted on individual
# lines or following the machine name denoted by a '#' symbol.
#
# For example:
#
#      102.54.94.97     rhino.acme.com          # source server
#       38.25.63.10     x.acme.com              # x client host

# localhost name resolution is handled within DNS itself.
#	127.0.0.1       localhost
#	::1             localhost

然后直接保存关闭就行了（显示没有权限修改的话直接把目录下的文件删除了，然后在桌面新建一个hosts.txt写入以上内容复制到C:\Windows\System32\drivers\etc下就行了

2、电脑临时劫持

我见过的电脑劫持就是运行了一个软件，然后再打开A网站就跳到B网站去了，奇怪的是电脑重启后又自动恢复了，也有可能是修改了hosts文件，系统重启后自动修复了就恢复了。

3、IP劫持

IP劫持就是在你访问网站向网页发出访问请求时，服务器还没给你反馈，其它操纵者就已经把自己的网页展现给你了。遇到IP劫持了可以利用路由器reset键重启路由器，看重启后是否恢复正常。

4、DNS劫持

DNS劫持要相对麻烦些，劫持不严重的，可以打开网络-属性-本地连接-属性-IPv4-属性-修改DNS，比如我们可以修改成谷歌的DNS，8.8.8.8，备用DNS8.8.4.4。
修改DNS并不是全都有用的，如果是修改DNS后没有效果，可以进路由看下是不是路由器也被DNS劫持了。
打开路由器在左侧菜单“DHCP服务器“中的”DHCP服务“选项中，我们可以填写主DNS为8.8.8.8，备用DNS为114.114.114.114，然后点击保存。

5、网吧劫持

网吧劫持可以说是很顽固了，分为两种：一种是网管和劫持站有利益往来，另外一种是电信网络劫持。不管哪种，可以先和网管反映，看能不能重装系统，或是换台机子试下。

6、电脑驱动劫持

现在传奇登录器使用的就是驱动劫持，只要运行了带有劫持的登录器，就打不开其它登录器，网页也一样，被驱动劫持后打开的就是其它的劫持站点。
驱动劫持可以用360急救箱急救2遍，要选择强力模式，强力模式后还是不行就全盘扫描，一般清理后重启电脑就可以恢复正常。

7、地区网络劫持

地区网络劫持就是有地域性的，比如城西区域的网吧，试了多家都会被劫持，但是去五福街的网吧就没问题。这种地区网络劫持只有联系运营商客服解决了。
