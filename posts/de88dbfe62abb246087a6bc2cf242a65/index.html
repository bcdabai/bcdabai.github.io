GDB 调试输出到文件：

1.gcc -g 源文件

2.gdb 可执行文件

3.进入gdb模式后，设置日志存储的文件

set logging file xxx.txt

4.开始将日志拷贝到文件

set logging on

5.关闭拷贝

set logging off

6.结束调试后，在当前文件夹打开日志即可查看。
