vs2017安装驱动环境版本的选择

 我的VS是10.0.17763的，应该下载哪个WDK呢？

下载地址：以前的 WDK 版本和其他下载 - Windows drivers | Microsoft Docs

发现下载这个1809的，安装的时候才和VS里的SDK保持一致，

如果不一致安装的时候会有红色信息提示。

安装大概要好几分钟，安装完成后即可创建驱动WDF了
