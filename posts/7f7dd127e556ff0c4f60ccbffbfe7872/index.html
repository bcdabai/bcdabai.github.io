Mybatis的四种模糊查询的方法

第一种写法：当调用的时候我们需要写%%或 “_”

select * from user where uname like  #{like}

第二种写法 ：吧%%或者_ 卸载映射文件里，调用传参的时候，不需要写%了。

select * from user where uname like "%"#{like}"%"

第三种方法：使用sql拼接串，不能防止sql注入。

select * from user where uname like '%${value}%

第四种：时有mysql的cancat方法

select * from user where uname like CONCAT("%",#{s},"%")
