-bash: vim: command not found(快速解决)

 Step1 : vim /etc/hosts

Step2 : 检查系统中是否存在安装包。rpm -qa|grep vim

Step3: 安装vim  yum -y install vim*

Step 4: 测试 vim /etc/hosts
