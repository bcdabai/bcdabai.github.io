sftp的操作命令

bye     退出sftp
cd      change your remote working directory
clear   clear screen
exit    退出sftp
explore 浏览本地目录
get     下载
help    give help
lcd     更改和/或打印本地工作目录
lls     list contents of a local directory
lpwd    print your local working directory
ls      查看当前目录下文件
mkdir   创建文件夹
mv      移动或重命名远程服务器上的文件
put     上传
pwd     查看当前目录
quit    退出sftp
rename  修改文件名
rm      删除文件
rmdir   删除文件夹
