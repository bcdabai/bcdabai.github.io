小冰商城（手机app）

小冰商城

整体框架

  • xiaomiServe 后台node-koa服务器
  • 数据库mysql
  • xiaomi_front 移动端小冰商城app
  • xiaomi_admin 后台管理系统

没有文件服务器，文件保存在后台服务器的静态文件夹 static文件夹下

安装教程
  1. cd xiaomiServe
  2. cnpm install
  3. cd xiaomi_front
  4. cnpm install
  5. cd xiaomi_admin
  6. cnpm install
  7. mysql 导入数据库 shops.sql
  8. 修改 xiaomiServer/router/koa-router.js 中关于数据库的配置
启动说明
  1. cd xiaomiServe
  2. node index.js
  3. 打开新的终端
  4. cd xiaomi_front
  5. npm run serve
