Git如何将一个分支上的修改转移到另一个分支

一.未commit

git stash //把暂存区内的修改存储起来

git checkout dev //切换到正确的分支

git stash pop //将存储的修改取出来

操作：

 1、左上角Git–>Uncommitchanges–>Stash changes
 2、切换到目标分支
3、左上角Git–>Uncommitchanges–>Stash pop
