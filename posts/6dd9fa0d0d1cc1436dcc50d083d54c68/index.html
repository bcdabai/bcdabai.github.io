【C语言】分支循环语句

分支

  1. if else
  2. if else if else if else
  3. switch() case default

循环

  1. while 防止死循环
  2. for 已知所需循环次数
  3. do while 至少执行一次

break 和continue在循环中的应用
break ：退出当前循环；
continue：推出本次/轮循环

goto 太乱了，不建议使用。
