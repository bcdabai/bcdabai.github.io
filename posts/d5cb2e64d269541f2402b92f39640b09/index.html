06-Ubuntu18.04&16.04安装PHP(7.0|7.1|7.2|7.3)

安装 PHP 7.3 on Ubuntu

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:ondrej/php
sudo apt-get update
sudo apt-get install -y php7.3

php -v
