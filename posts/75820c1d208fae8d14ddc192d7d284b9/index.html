第七步：修改包路径

默认情况下，我们在执行npm install -g XXX 是，下载一个全局包，这个包默认存放路径为：
C:\Users\Administrator\AppData\Roaming\npm\node_modules
可以通过命令 npm root -g 查看全局包的安装路径

通常我们都会把node安装在 除 C 盘以外的其他位置，全局的包也希望能放在node安装目录下
1：在node安装目录下新建两个新文件夹 node_global、node_cache ，分别用来存储node全局包，和 node 缓存

2： 修改路径

在cmd窗口分别执行下面两条命令：

// 设置包下载位置
npm config set prefix "D:\node\node_global"
// 设置node缓存
npm config set cache "D:\node\node_cache"

同时我们还需要修改系统环境变量
计算机右键——属性——高级系统设置
将系统环境变量的路径修改为： D:\node\node_global
新增系统变量 NODE_PATH ，变量值为： D:\node\node_global\node_modules
