编写函数，通过指针求字符串的长度。**输入格式要求：“%s“ 提示信息：“Enter string:“**输出格式要求：“string length=%d\n“

#include<stdio.h>
#define N 20
int len(char *p)
{
    int i=0;
    for(;*p!='\0';p++)
    {
        i++;
    }
    return i;
}
int main()
{
    int length;
    char str[N];
    printf("Enter string:");
    gets(str);
    length=len(str);
    printf("string length=%d\n",length);
}
