创建一个 .es 文件

sample.es

使用 VScode 打开这个文件

code sample.es

我们选择 elasticsearch 语言

我们选择 localhost:9200 作为 Elasticsearch 的地址。我们点击 enter 键确认。

安装上面的工具。

我们在刚才的 .es 文件的目录中创建一个叫做 default.esenv 的文件：

{

   "name": "dev"
 }
