离线安装npm包

1.先在联网机器上下载安装npm-bundle和http-server

以管理员身份运行打开cmd

然后npm install npm-bundle 或者 cnpm install npm-bundle

npm install http-server或者 cnpm install http-server

2.用npm-bundle将http-server打包，打包会生成http-server-14.1.1.tgz文件

 

3.将http-server-14.1.1.tgz文件复制到（离线）目标服务器上

使用npm install ./http-server-14.1.1.tgz -g 安装到（离线）服务器上

4.执行http-server -p 1234命令，在端口1234上启动http-server
