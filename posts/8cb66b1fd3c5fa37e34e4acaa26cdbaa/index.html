Visual Studio Code (VSCode)，快速跳转到指定行的代码

在 Visual Studio Code (VSCode)，快速跳转到指定行的代码可以通过以下步骤实现：

  1. 使用快捷键: 在 Windows 或 Linux 系统中，按下 Ctrl + G；在 macOS 中，按下 Cmd + G。
  2. 输入行号: 弹出的对话框中输入你想要跳转到的行号，例如 123。
  3. 回车确认: 输入行号后按 Enter，VSCode 会立即跳转到指定的行。

这个方法非常高效，特别是在处理大型文件时，能快速定位到你感兴趣的部分。
