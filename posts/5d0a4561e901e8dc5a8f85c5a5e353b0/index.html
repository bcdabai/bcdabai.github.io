洛谷入门记录B2001

#include <stdio.h>

int main() {

 long long a, b;

 scanf("%lld%lld", &a, &b);

 printf("%lld", a + b);

 return 0;

}

萌新小白的自我总结：1.没有注意ab的精度问题，要用long long类型，而不是int类型。

2.long long类型用scanf函数输入要用%11d。
