go项目运行方法

操作前提条件：系统已经安装go

项目代码：

hello.go

//package hello
package main

func main() {
	println("hello golang!")
}

方法1：直接终端运行.go文件

命令：go run hello.go

方法2：编译.go文件后，终端运行.exe文件

编译后会产生一个 .exe编译文件，可直接运行

方法3：golang编辑器中运行

        注意golang中运行需要保证golang已经设置GOROOT.
