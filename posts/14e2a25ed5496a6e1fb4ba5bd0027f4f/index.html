连接远端redis踩的坑

报错 (error) DENIED Redis is running in protected mode because protected mode is enabled

  • 注释掉绑定ip
    # bind 127.0.0.1
  • Redis默认不是以守护进程的方式运行，可以通过该配置项修改，使用yes启用守护进程，设置为no
    daemonize no
  • 保护模式,关闭保护模式，否则外部ip无法连接
    protected-mode no
然后重启redis就可以了
