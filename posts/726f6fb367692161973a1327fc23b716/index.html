C语言实现-求m到n之和

练习5-1 求m到n之和（C语言）

题目要求：本题要求实现一个计算m~n（m<n）之间所有整数的和的简单函数。 

函数接口定义：int sum (int m, int n);

其中m和n是用户传入的参数，保证有m<n。函数返回的是m~n之间所有整数的和。

#include <stdio.h>
int sum(int m,int n);
int sum(int m,int n){
	int i,s=0;
	int x=m;
	for(i=0;i<=n-m;i++){
		s+=x;
		x++;
	}
	return s;
}
int main(){
	int m,n;
	scanf("%d%d",&m,&n);
	printf("sum = %d",sum(m,n));
	return 0;
}
