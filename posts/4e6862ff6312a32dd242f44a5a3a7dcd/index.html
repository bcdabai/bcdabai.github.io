友元

友元的目的就是让一个函数或者类 访问另一个类中私有成员

全局函数做友元

friend void goodGay(Building * building);

类做友元

friend class goodGay;

成员函数做友元

	//告诉编译器  goodGay类中的visit成员函数 是Building好朋友，可以访问私有内容
	friend void goodGay::visit();
