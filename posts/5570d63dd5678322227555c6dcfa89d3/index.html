输入一个日期，判断这一天属于这一年的第几天（Python）

方法一

import datetime

date = input("请输入某年某月某日，格式是 yyyy-mm-dd: ")
y = int(date[0:4])  # 获取年
m = int(date[5:7])  # 获取月
d = int(date[8:])  # 获取日

targetdate = datetime.date(y,m,d)  # 将输入的日期转化为标准日期
thisyeardate = datetime.date(y,1,1)  # 获取当前年第一天的标准日期
daycount = (targetdate - thisyeardate + 1).days
print(daycount)
