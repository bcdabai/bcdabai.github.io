C++子集生成方法

这道题是数据结构、算法与应用c++的书籍的练习题25题，首先我先列出详细题目

题目

编写一个C++递归函数，输出n个元素的所有子集。例如，三元素集{a,b,c}的子集是{}（空集）,{a},{b},{c},{a,b},{a,c},{b,c},{a,b,c}。这些子集用0/1组成的代码序列来表示分别是000,100,010,001,110,101,011,111(0表示相应的元素不在子集中，1表示相应的元素在子集中)。因此你的程序输出长度为n的0/1序列即可。

分析

题目已经告诉大概的方向了，用0/1代码序列表达，递归，从集合里0个元素到n个元素输出。

代码

#include <iostream>

using namespace std;
template<class T>
void f(T* a, int* mark, int n, int i)
{//a为集合元素，mark为标记数组，n为起点，i为元素个数
	if (n == i)
	{
		cout << "{ ";
		for (int k = 0; k < i; k++)
		{
			  if (mark[k] == 1)
					cout << a[k];
		}
		cout << "}" << endl;
		return;
	}

	mark[n] = 0;
	f(a, mark, n + 1, i);
	mark[n] = 1;
	f(a, mark, n + 1, i);
}
int main()
{
	int mark[3];
	char a[3] = { 'a','b','c' };
	f(a, mark, 0, 3);
}
