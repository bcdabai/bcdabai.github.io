Linux配置IP地址

一、CentOS上查找IP连接Linux客户端

下载好虚拟机上并安装Linux之后输入指令查找IP地址：

# 查看ip地址
	ip a
	ip addr

 当查不到IP地址时，输入以下指令查找到对应的 ens33网卡配置：

输入vi /etc/sysconfig/network-scripts/ifcfg-ens33（vi后加空格）

将NOBOOT的权限改为yes， 再按住ESC退出，之后再次输入：wq，再按Enter。

输入指令重启网络服务，看到[ok]即配置成功：

sudo service network restart
