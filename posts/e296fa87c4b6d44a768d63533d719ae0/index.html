C语言isdigit()函数:判断字符是否为十进制数字

函数名: isdigit

头文件：<ctype.h>

函数原型:  int isdigit(int ch);

功 能:  判断字符是否为十进制数字

参数： int ch  待检查的字符

返回值： ch不是十进制数字  返回0 , ch是十进制数字    返回非0

#include<ctype.h>

#include<stdio.h>

int main(){

   char ch;

   printf("input a character:");

   scanf("%c",&ch);

   if(isdigit(ch)){

      printf("%c is digit.",ch);

   }else{

      printf("%c is not digit.",ch);

   } 

   putchar('\n');   

   return 0;  

}
