return返回值用法

在Java中

public static int add(int x, int y）{  

    z = x + y

    return z;  

}

返回值是必须有的，作用就是把固定不变的、常用的代码封装起来写在类里，每次用到的时候就调用就行了。如果需要这个功能返回一个值，甚至一个对象，就需要用返回值了即用return来把值返回给调用者。

在Python中

def add(a, b):
    c = a + b
    return c

return直接返回c，若不需要也可以不要return

若在return下再加一句

def add(a, b):
    c = a + b
    return c
    print(c)

这时的print(c)将不会被执行，return之后直接跳出函数。
