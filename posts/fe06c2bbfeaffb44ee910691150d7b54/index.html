【IDEA】idea写java代码运行按钮是灰色的解决办法

问题描述

Run按钮灰色状态，无法点击运行，如图：

原因分析

上图估计大佬一看就看出问题了，其实就是因为少了一个main函数

问题解决

我现在把main函数补上去

public static void main(String[] args){
    int a;    
    for (a=0;a<=100;a++){
        System.out.println(a);
    }
}

把main函数补上去后，右上角的运行就会亮了起来，这时候就可以运行了
