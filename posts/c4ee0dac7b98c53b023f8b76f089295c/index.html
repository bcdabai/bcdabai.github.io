正则表达式实现删除所有注释

1. 要删除 /****/这种块注释： 直接匹配      /\*(.|[\r\n])*?\*/   即可。  转自：简书：利用正则表达式去掉注释/****/

2. 要删除  // 开头的注释：直接 匹配 //.*即可。  转自：csdn: Notepad++删除代码中的注释，可删除//单行注释和/**/多行注释

3. 要删除 <!-- xxxx --> 这种html块注释，直接 匹配 <!--(.|[\r\n])*?--> 即可。

4. 删除 java 注释 /* */：/\*{1,2}[\s\S]*?\*/

5. 删除 java 注释 //：//[\s\S]*?\n

6. 删除xml注释：<!-[\s\S]*?-->

7. 删除空白行：^\s*\n
