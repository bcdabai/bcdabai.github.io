错误: Too many arguments to function call,expected 0,have *

错误

将XCode升级到6后，报Too many arguments to function call, expected 0, have *，在XCode5.1里能编译通过的，到xcode6就报错

查了下工程,原来是 objc_msgSend() 方法报的错误，在 MJRefresh 和 SDWebimage使用的项目中曾经都出现过此错误。

解决方案

Build Setting–> Apple LLVM - Preprocessing–> Enable Strict Checking of objc_msgSend Calls 将YES 改为 NO
