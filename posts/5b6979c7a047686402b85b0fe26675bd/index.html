每日一课 | 将 Python string 转换为 float 的方法

在Python中，我们可以使用float()将String转换为float。

num = "3.1415"
 
print(num)
print(type(num)) # str
 
pi = float(num) # convert str to float
 
print(pi)
print(type(pi)) # float

输出量

3.1415
<class 'str'>
3.1415
<class 'float'>
