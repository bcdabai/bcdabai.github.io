Hive增加文件映射到表

1，打开hadoop的文件系统

 2，使用IntelliJ通过JDBC链接到hive，也可以用其它工具

3，创建数据库，创建表，选择拆分规则

create database hivedemo;

use hivedemo;
drop table users;
create table users(
    id int comment 'ID',
    name string comment '姓名'
) comment '用户表'
row format delimited
fields terminated by ',';

4，查看新建表目录

5，上传文件到服务器，通过hadoop 命令推送到新建表的目录下

文件内容如下：

 hadoop fs -put file2.txt /user/hive/warehouse/hivedemo.db/users

6，查询hive自动映射的表内容

select * from users;
