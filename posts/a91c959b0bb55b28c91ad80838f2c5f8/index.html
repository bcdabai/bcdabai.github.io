用C语言写一个三数求最大值代码

可以这样写：

#include <stdio.h>

int main(void) {
  int a, b, c;
  printf("请输入三个数字：");
  scanf("%d%d%d", &amp;a, &amp;b, &amp;c);

  int max = a;
  if (b > max) {
    max = b;
  }
  if (c > max) {
    max = c;
  }

  printf("最大的数是：%d\n", max);
  return 0;
}

这段代码会从用户输入三个数字，然后找出这三个数字中的最大值并输出。
