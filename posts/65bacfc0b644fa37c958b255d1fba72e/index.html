ubuntu终端连接ARM板卡（赛灵思ZYNQ）USB无线网卡

板卡ubuntu18.04系统没有桌面，因此必须使用命令终端连接WIFI。首先修改软件源（见软件源）；具体连接方法如下：

  1. 修改软件源(此处为中科大Arm架构Ubuntu源)，否则可能不能下载相应软件。并更新软件源 sudo apt update
  2. 打开RF-kill：sudo apt install rfkill -y && sudo rfkill list all && sudo rfkill unblock wifi
  3. 启动usb网卡模块：sudo ip link set wlx08beac2f6c05 up
  4. 安装网络管理工具：sudo apt install network-manager -y
  5. network manager 服务 打开: sudo service network-manager start
  6. 终端连接网络：sudo nmcli dev wifi connect TianYi-office password tiAnyI0725 wep-key-type key ifname wlx08beac2f6c05
  • 网络名称：TianYi-office
  • WiFi密码：tiAnyI0725
  • USB网卡名称（通过ifconfig -a 即可获取）：wlx08beac2f6c05

软件源：

deb http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial main restricted universe multiverse

deb-src http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial main restricted universe multiverse

deb http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial-updates main restricted universe multiverse

deb-src http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial-updates main restricted universe multiverse

deb http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial-security main restricted universe multiverse

deb-src http://mirrors.ustc.edu.cn/ubuntu-ports/ xenial-security main restricted universe multiverse

脚本文件如下（请先换源）：

#!/bin/bash
echo "请确保修改软件源为ustc的源或其他arm的ubuntu国内源，否则软件可能无法现下载..."
sudo apt install rfkill -y && sudo rfkill list all && sudo rfkill unblock wifi
sudo apt install network-manager -y && sudo service network-manager start
ifconfig -a 
read -p "please enter wifi name: " SSID
read -p "please enter password to connect WiFI:" passwd
read -p "please enter name of USB_WIFI:" USB_WIFI

sudo nmcli dev wifi connect $SSID password $passwd wep-key-type key ifname $USB_WIFI
