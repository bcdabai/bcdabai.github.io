Nginx常用命令（Ubuntu系统下）

systemctl status nginx.service #查看日志

sudo apt-get install nginx  #安装

/etc/init.d/nginx start  #启动

/etc/init.d/nginx stop  #关闭

/etc/init.d/nginx restart  #重启

基本上这几个日常就够用了。
