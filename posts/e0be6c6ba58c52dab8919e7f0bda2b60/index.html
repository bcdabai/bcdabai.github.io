例1 反转一个三位数（Python）

问题描述：
反转一个只有三位数的整数。
例如输入number = 123，输出 321；输入number = 900，输出 9。 

方法一：字符串分割

str = input("输入：")
a,b,c = str[0],str[1],str[2]
if c == "0":
    if b == "0":
        result = int(a)
    else:
        result = int(b+a)
else:
    result = int(c+b+a)
print("输出:",result)

方法二：纯计算（取余）

number = int(input("输入："))
a = number // 100
b = (number // 10) % 10
c = number % 10
if c == 0:
    if b == 0:
        result = a
    else:
        result = b * 10 + a
else:
    result = c * 100 + b * 10 + a
print("输出：",result)

方法三 构造类、函数

class Solution:
    #参数number：一个3位整数
    #返回值：反转后的数字
    def reverseNumber(self,number):
        bai = int(number / 100)
        shi = int(number % 100 / 10)
        ge = int(number % 10)
        return(100 * ge + 10 * shi + bai)
if __name__=='__main__':
    solution = Solution()
    num = int(input("输入："))
    ans = solution.reverseNumber(num)
    print("输出：",ans)
