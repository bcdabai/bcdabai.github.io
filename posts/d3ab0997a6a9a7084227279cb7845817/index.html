Linux文件去掉每行末尾^M

Linux下去掉^M的四种方法_linux shell_脚本之家

参考了上面的博客

cat -A filename 就可以看到windows下的断元字符 ^M（我用的是vi filename，也可以看到^M）

要去除^N，最简单用下面的命令：

dos2unix filename

（如果没有安装，直接使用 sudo apt install dos2unix 安装即可）
