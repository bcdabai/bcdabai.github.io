dSPACE Mid-Size Simulator Introduction

dSPACE Simulator Mid-Size是一种闭环的硬件在环仿真器，其有以下特性：

  • 支持信号调理

  • 支持给ECU提供等效或真实负载

  • 失效模拟

  • 两路电压系统支持远程控制供电

主机需配备一块链接板卡-Link Board来进行总线接口连接。dSAPCE提供3种不同类型的Link Board来匹配主机不同的总线类型（有ISA/PCI/PCMIA接口）。总线接口用来将real-time model下载到仿真器中，对仿真过程进行控制。主机和仿真器之间通过RS232串口来控制故障注入单元(FIU).

dSPACE Mid-Size仿真包含以下部件:

  • 处理器板块DS1005/DS1006

  • I/O板卡DS2211/DS2202

  • 链接板卡DS814

  • Load/FIU单元：5个DS790负载板卡，每个板卡包含10个通道，可配置低功率的阻性或感性负载；5个DS791故障注入单元，每个单元包含10个通道，可配置低功率的阻性或感性负载

  • 一个DS686 Backplane

  • 一个DS685 Midplane

  • 可远程控制的供电单元

DS1006 Processor Board

The DS1006 Processor Board is based on a single-core or a multicore AMD Opteron TM processor. This real time processor (RTP) is the main processing unit. It can access modular I/O boards via its PHS bus. It is multiprocessing-capable via the (optional) DS911 Gigalink Module.

DS1006处理器板基于单核或多核AMD Opteron TM处理器。此实时处理器 （RTP） 是主要的处理单元。它可以通过其PHS总线访问模块化I / O板。它通过（可选）DS911 千兆链路模块实现多处理能力。

Figure 1. DS1006 Processor Board

DS2211 I/O Board

The DS2211 HIL I/O Board is tailored to simulate and measure automotive signals. It combines a variety of typical HIL I/O functions on one board. The board also contains signal conditioning for typical signal levels of 12 V and 42 V automotive systems.

DS2211 HIL I/O板专为仿真和测量汽车信号而定制。它将各种典型的HIL I/O功能组合在一块电路板上。该板还包含针对 12 V 和 42 V 汽车系统典型信号电平的信号调理。

Figure 2. DS2211 I/O Board

DS4302 CAN Interface Board

The DS4302 CAN Interface Board allows data transfer between dSPACE real-time systems and various other control units via 4 CAN bus interfaces.

DS4302 CAN接口板允许通过4个CAN总线接口在dSPACE实时系统和其他各种控制单元之间传输数据。

Figure 3. DS4302 CAN Interface Board

DS4505 & DS4340 & DS4342

The DS4505 Interface Board together with different interface modules allows data transfer between dSPACE real-time systems and various other control units:

  • Via FlexRay bus by means of DS4340 FlexRay Interface Modules

  • Via CAN bus by means of DS4342 CAN FD Interface Modules

DS4505接口板卡与不同的接口模块一起，允许在dSPACE实时系统和各种其它控制单元之间传输数据：

  • 选择DS4340 FlexRay接口模块，支持FlexRay总线通信

  • 选择DS4342 CAN FD接口模块，支持CAN FD总线通信

Figure 4. DS4505 Interface Board(Left); DS4342 CAN FD Interface Module(Middle); DS4340 FlexRay Interface Module(Right)

DS790 Load Card

dSPACE Simulator Mid-Size is equipped with five low-power load cards (DS790 Load Card) with the following characteristics:

  • 10 single-ended loads or 5 double-ended loads per card, or mixed

  • 2 W maximum continuous power per load

  • 50 W maximum continuous power per load/FIU unit

  • 6 A maximum load current

  • LED indicators displaying current load states, battery voltage and switched voltage

  • Front panel for measurement or connection of real loads

dSPACE Mid-Size模拟器配备五个低功耗负载板卡（DS790 负载板卡），具有以下特性：

  • 每张卡 10 个单端负载或 5 个双端负载，或混合负载

  • 每个负载支持2 W 最大连续功率

  • 每个负载/FIU 单元的最大连续功率为 50 W

  • 6 A 最大负载电流

  • 显示当前负载状态、电池电压和开关电压的 LED 指示灯

  • 用于测量或连接实际负载的前面板

Figure 5. DS790 Load Card

Loads are installed on the load cards inside the simulator or plugged to the panel on the front of the simulator. Inside the simulator there are load sockets that the loads can be connected to. One load socket connects the load to the ECU channel, the other load socket connects the load to a high rail or to the adjacent channel (for double-ended loads).

负载安装在模拟器内部的负载卡上，或插入模拟器正面的面板上。在模拟器内部，有负载插座，负载可以连接到。一个负载插座将负载连接到ECU通道，另一个负载插座将负载连接到高轨或相邻通道（用于双端负载）。

Figure 6. Install a single-ended load on the load socket(Left); Install a double-ended load on the load socket(Right)

DS791 Failure Insertion Unit

A Load/FIU unit is equipped with 5 failure insertion units (DS791 FIU) with the following characteristics:

  • Support of 50 ECU pins in total (each DS791 Failure Insertion Units (FIU) have 10 channels.)Failure simulation

  • for ECU outputs (Channels are directly connected to the pins of the ECU 1 connector, which contains the signals for the ECU outputs)

  • Remote control using Control Desk's Failure Simulation module

  • Simulation of short circuits:

    • Short circuits from one or more ECU pins to ground or battery voltage via fail plane 1 or fail plane

    • Short circuits between two or more ECU pins via fail plane 1 or fail plane 2

  • Cable break simulation

  • Simultaneous simulation of multiple failures

  • 6 A maximum load current

负载/FIU 单元配备 5 个故障插入单元 （DS791 FIU），具有以下特性：

  • 总共支持50个ECU引脚（每个DS791故障插入单元（FIU）有10个通道）故障模拟

  • 用于 ECU 输出（通道直接连接到 ECU 1 连接器的引脚，该连接器包含 ECU 输出的信号）

  • 使用ControlDesk的故障模拟模块进行远程控制

  • 短路模拟：

    • 通过故障面板将一个或多个 ECU 引脚短路到地或电池电压

    • 两个或多个ECU引脚之间短路

  • 开路模拟

  • 同时模拟多个故障

  • 最大6 A的负载电流

The DS791 FIUs are connected to the ECU 1 connector and the DS685 Midplane.

DS791 FIU 连接到 ECU 1 连接器和 DS685 背板。

Figure 7. DS791 Failure Insertion Unit
