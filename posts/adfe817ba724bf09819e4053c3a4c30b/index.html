JS检查日期是否为工作日

JavaScript检查日期是否为工作日(1-5)

使用这个方法，你就可以检查函数参数是工作日还是周末；周一到周五为工作日，周六周天不是工作日。 代码片.

const isWeekday = (date) => date.getDay() % 6 !== 0;
console.log(isWeekday(new Date(2021, 0, 11)));
// Result: true (Monday)
console.log(isWeekday(new Date(2021, 0, 10)));
// Result: false (Sunday)
