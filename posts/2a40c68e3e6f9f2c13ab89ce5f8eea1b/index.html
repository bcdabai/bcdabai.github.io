遍历数组在数组中每一条数据里增加一个字段

var data = [
{name:'小明',value:'1'},
{name:'小红',value:'2'}
]

遍历数组给每条数据都加上age字段:

data.map((item, index) => {
item.age = '18'; //添加的字段
})

再次打印数组结果如下:

[
{name:'小明',value:'1',age:'18'},
{name:'小红',value:'2',age:'18'}
]
