洛谷P1001 题解

传送门

题意

输出两数之和

分析

本题数据范围是 − 1 0 9 ≤ a , b ≤ 1 0 9 -10^9 \le a,b \le 10^9 −109≤a,b≤109

用int绰绰有余

代码

#include <bits/stdc++.h> //万能头
using namespace std; //标准命名空间
int main()
{
	int a,b; //定义变量
	cin >> a >> b; //输入数据
	cout << a+b; //输出和
	return 0; //别忘了返回0
}

注意事项

输出时直接使用+，不用另外定义变量
