H3C SS5500 交换机配置操作记录

交换机Port1 vlan1 上联华三网关，port4 5 vlan100  被镜像到 port9，dhcp 服务器，可接电脑或者AP。

<H3C>system-view
#配置DHCP 服务器

1)创建V100
dhcp server ip-pool vlan100
network 172.16.100.0 mask 255.255.255.0
gateway-list 172.16.100.1 
dns-list 114.114.114.114
expired day 8
dhcp enable

2)创建V100

vlan 100
qu
interface Vlan-interface 100
ip address 172.16.100.1 255.255.255.0    #配置vlan100 网关和掩码：注意要和dhcp网关配置一致，否则网络也不通

3) 无标题

qu
dhcp server forbidden-ip 172.16.100.1 172.16.100.100 
interface range GigabitEthernet 1/0/4 to GigabitEthernet 1/0/5 
port access vlan 100
dis vlan 100 查看

4)保留交换机的第1个口做为连接到路由器的口，把他配置到默认的vlan1中
[H3C]dis vlan 1
[H3C]int vlan 1
[H3C-vlan-interface1]ip address 10.10.10.11 255.255.255.0  #交换机的第一个口的IP地址为10.10.10.11
[H3C-vlan-interface1]quit
[H3C]vlan 1
[H3C-vlan1]port GigabitEthernet 1/0/1
[H3C-vlan1]quit

5)配置一个默认路由指向路由器
[H3C]ip route-static 0.0.0.0 0.0.0.0 10.10.10.10  ，#所有ip路由到华三网关，网关地址为10.10.10.10

6)退出配置模式，并保存配置
[H3C]quit
<H3C>save

#在H3C路由器配置三个VLAN的静态路由表  {账号和密码都是admin}
    高级设置---->路由设置---->静态路由
    目的的地址           子掩码       下一跳地址       出接口
    172.16.100.0      255.255.255.0   10.10.10.11     VLAN1
    172.16.200.0      255.255.255.0   10.10.10.11     VLAN1     #为200vlan备用

#设置镜像端口 交换机的第9口为镜像口
[H3C]mirroring-group 1 local

[H3C]dis mirroring-group 1
Mirroring group 1:
    Type: Local
    Status: Incomplete
#设置镜像的源端口 #(镜像可以根据实际情况灵活选择入方向、出方向及全部流量；both，全部流量；inbound，入方向流量；outbound，出方向流量)
[H3C]mirroring-group 1 mirroring-port GigabitEthernet 1/0/4 both    
[H3C]mirroring-group 1 mirroring-port GigabitEthernet 1/0/5 both    
[H3C]dis mirroring-group 1
Mirroring group 1:
    Type: Local
    Status: Incomplete
    Mirroring port:
        GigabitEthernet1/0/4  Both
        GigabitEthernet1/0/5  Both
[H3C]mirroring-group 1 monitor-port GigabitEthernet 1/0/9    #设置镜像的目的端口
[H3C]dis mirroring-group 1
Mirroring group 1:
    Type: Local
    Status: Active
    Mirroring port:
        GigabitEthernet1/0/4  Both
        GigabitEthernet1/0/5  Both
    Monitor port: GigabitEthernet1/0/9
