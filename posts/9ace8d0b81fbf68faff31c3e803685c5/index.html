echarts 中如何将 legend 设置成「直线」？

所有图例均为直线


可以通过 itemHeight 和 itemWidth 来统一控制

legend: {
    itemHeight: 2,
    itemWidth: 20,
    data: ['Expenses', 'Income']
  }

部分是直线
