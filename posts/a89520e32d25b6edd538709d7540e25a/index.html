python开平方计算（求平方根）

方法一：简单粗暴，内存消耗大

x = int(input())
res = int(x **0.5)
print(res)

方法二：使用math模块

# 引入math模块
import math

x = int(input())
ans = math.sqrt(x)
ans = str('%.4f' % ans)  # 保留四位小数
print(ans.center(20, '*')) # 居中显示，20个字符填充

方法三：二分法

# 二分法
x = int(input())
low, high, ans = 0, x, -1
while low <= high:
    mid = (low + high) // 2
    if mid * mid <= x:
        ans = mid
        low = mid + 1
    else:
        high = mid - 1
print(ans)
