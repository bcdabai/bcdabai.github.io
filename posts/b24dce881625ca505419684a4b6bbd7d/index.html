PTA——7-3 我是升旗手

输入格式:

输入包括两行。 第一行:包括一个整数n，表示班级里共有n位同学。 第二行:包含n个三位数，表示每一位同学的身高。

输出格式:

输出身高最高的同学的身高。

样例">输入样例:

4
130 125 129 140

输出样例:

140
#include<stdio.h>
int main()
{
    int n,h=0,a;
    scanf("%d",&n);
    while(n!=0)
    {
        scanf("%d",&a);
        if(a>h)
        {
            h=a;
        }
        n--;
    }
    printf("%d\n",h);
    return 0;
}
