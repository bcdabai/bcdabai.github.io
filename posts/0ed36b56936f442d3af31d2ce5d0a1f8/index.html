ubuntu nfs 客户端挂载

安装客户端

sudo apt-get install nfs-common

查看服务端共享目录

showmount -e “ip(服务端ip)”

进行挂载

mount  192.168.3.6:/data /data/share
mount ip:共享目录 挂载目录
