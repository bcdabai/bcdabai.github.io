下载和安装

AS

每次重新下载都需要配置，记录下

Android Studio 4.2 :https://developer.android.google.cn/studio/

Sdk

如果出现Sdk下载不下来，或者想修改Sdk到其他磁盘：

跳转：
http://tools.android-studio.org/index.php/sdk，下载如图所示ZIP

AndroidStudio 中文网：http://www.android-studio.org/

Jdk

Jdk 也可以从中文网下载，或者Oracle官网。

https://www.oracle.com/java/technologies/downloads/#java8-windows

从官网下载的时候，需要Oracle账号登陆，因为国内的缘故可能登陆不成功，没有关系，只要你登陆时正确的，Jdk就开始下载了，等待下载好了直接用就行了。

5. Android ButterKnife Zelezny

移除butter的插件：RemoveButterKnife

ButterKnife Zelezny插件，可以自动引入layout内定义的属性


在布局文件上点击 Alt + insert ,弹出进入插件选项.


进入插件，可自定义名称，选择性勾选 ==

6. Android Parcelable code generator

生成序列化代码

鼠标放在 Parcelable 上，按Alt + insert ，弹出此窗口，最下面的Parcelable选项既是；点击进入，全选参数即可。
