一.了解SpringBoot

1. springBoot

最主要作用就是帮我们快速的构建庞大的spring项目，并且尽可能的减少一切xml配置，做到开箱即用，迅速上手，让我们关注与业务而非配置。

2. springBoot的特点

  1. 开箱即用 - 如果需要使用到一个框架时直接添加一个带有starter（启动器）的依赖就可以
  2. 无需任何xml
  3. 约定大约配置 SpringBoot已经配置好了很多默认项

3. 编写一个controller

package com.leyou.controller;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.bind.annotation.RestController;

/**
 * @author cyy
 * @date 2020/3/30 11:12
 */
@Controller
//@RequestMapping
public class Hello {

    @RequestMapping(value = "/hello")
    /*
     *注解 @ResponseBody，使用在控制层（controller）的方法上。
     *作用：将方法的返回值，以特定的格式写入到response的body区域，进而将数据返回给客户端。
     *如果返回值是字符串，那么直接将字符串写到客户端；如果是一个对象，会将对象转化为json串，然后写到客户端。
     */
    @ResponseBody
    public String hello(){

        return "springboot-测试";
    }

}

4. 注解

  1. @RestController =@Controller+@ResponseBody
  2. @ResponseBody
    作用：将方法的返回值，以特定的格式写入到response的body区域，进而将数据返回给客户端。
    如果返回值是字符串，那么直接将字符串写到客户端；如果是一个对象，会将对象转化为json串，然后写到客户端。
  3. @GetMapping(value = "/hello")是@RequestMapping(value = "/hello",method = RequestMethod.GET)的简写版
  4. @Configuration 声明当前的类是配置类
  5. PropertySource加载配置文件的注解
  6. @SpringBootConfiguration 表示当前的引导类是一个配置类
  7. @EnableAutoConfiguration 自动装配 自动创建对象放入到IOC容器中
    创建的是哪些对象？就是spring.factories中配置类中的所有对象，但是并非全部创建对象，是有选择性的创建对象，这里的选择性是看项目中是否有相关的启动器（带有starter的依赖）
  8. @ComponentScan 用来包扫描的 默认扫描的是当前包和子包

三.SpringBoot原理

1.springBoot是如何帮助我们创建对象的

springboot 集成了喝多的主流框架，在使用某一个框架的时候，添加这个框架的启动器会自动创建对象，添加启动器为添加依赖

2.springBoot的工作原理

加载spring.factories,这个文件有springBoot整合的所有主流框架的配置类，这些配置类中的对象会有选择性的初始化到ioc容器中，有一个@ConditionalOnClas注解

3. 拦截器设置

  1. 添加一个拦截器实现HandlerInterceptor接口
package com.leyou.interceptor;

import lombok.extern.slf4j.Slf4j;
import org.springframework.web.servlet.HandlerInterceptor;
import org.springframework.web.servlet.ModelAndView;

import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

/**
 * @author cyy
 * @date 2020/3/31 9:18
 */
@Slf4j
public class LoginInterceptor implements HandlerInterceptor {

    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {

        log.debug("执行了拦截器的前置方法");
        return true;
    }

    @Override
    public void postHandle(HttpServletRequest request, HttpServletResponse response, Object handler, ModelAndView modelAndView) throws Exception {

        log.debug("进入了方法中");
    }

    @Override
    public void afterCompletion(HttpServletRequest request, HttpServletResponse response, Object handler, Exception ex) throws Exception {
    log.debug("进入了拦截器的后置方法");
    }
}
  1. 编写一个拦截器的配置类（SpringBoot规定）
  • 实现WebMvcConfigurer接口
  • 添加@Configuration注解
package com.leyou.config;

import com.leyou.interceptor.LoginInterceptor;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.InterceptorRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;

/**
 * @author cyy
 * @date 2020/3/31 10:16
 */
@Configuration
public class MyMvcConfig  implements WebMvcConfigurer{

    //添加一个拦截器

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
//        addPathPatterns添加拦截器的规则
        registry.addInterceptor(new LoginInterceptor()).addPathPatterns("/**");
    }
}

2.编写controller

package com.leyou.controller;

import com.leyou.domain.User;
import com.leyou.service.UserService;
import org.apache.coyote.Request;
import org.springframework.beans.factory.annotation.Autowired;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

import javax.servlet.http.HttpServletRequest;
import java.util.List;

/**
 * @author cyy
 * @date 2020/3/31 15:10
 */
@Controller
@RequestMapping("/user")
public class UserController {

    @Autowired
    private HttpServletRequest request;
    @Autowired
   private UserService userService;

    @GetMapping("/list")
    public String list(){

        List<User> list = userService.findAll();
        request.setAttribute("userList",list);
        return "list";
    }

    @GetMapping("/{id}")
    public String findById(@PathVariable Long id){

        User user = userService.findById(id);
        request.setAttribute("user",user);
        return "view";
    }

}
