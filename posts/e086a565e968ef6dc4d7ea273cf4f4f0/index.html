JavaScript——forEach()方法

代码示例：

数组变量.forEach(值变量名 => {代码块})
//每遍历一个值，就作为形参传入给代码块，执行一次该函数头，继续遍历

  举例说明：

<script>
    let arr = [1, 2, 3, 4];
    //arr.forEach(val => {});
    arr.forEach(val => {
        console.log(val);
    });
</script>
brr.forEach(val => {
    console.log(val + "，你好");
})

crr.forEach(o => {
    console.log(o.name);
})

drr.forEach(o => {
    if(o.age < 18) console.log(o.age);
})

！注意：只遍历有值的元素
