Function的用法

定义函数的基本方法：

       1，函数声明：

            function fun1(){

          }

       2，函数表达式：

            var fun2 =function (){

          }

       3，Function:

           var fun3 = new Function(){

          }

Function 可以用来新建函数对象：

语法：

1，如果不传参，创建的就是一个空的函数

//var 函数名 = new Function()

2，如果只传一个参数，这个参数就是函数体

 //var 函数名 = new Function("函数体")

3，如果传多个参数，最后一个参数为函数体，前面的参数都是该函数的形参
