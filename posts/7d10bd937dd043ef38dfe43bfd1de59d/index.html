MAC安装MySQL不能启动，更改方式

Mac系统，安装MySQL 8.0.20版本 ，突然启动不了

一、在系统偏好设置中没办法停止服务，只能强制退出，用command+空格，输入activity monitor，进入运行进程中，关闭mysql.
二、终端输入:
cd /usr/local/mysql/bin   

sudo su

安全模式启动，然后输入：

./mysqld_safe --skip-grant-tables &
三、新开一个窗口登入更改密码：
mysql -u -root

user mysql;

update user set authentication_string="" where user='root';

ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';

重启mysql,就OK
