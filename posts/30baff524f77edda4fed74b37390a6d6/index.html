关于linux更换国内镜像源

  1. 先确定自己的系统的版本
    • 终端命令行输入：cat /etc/issue
  2. 第二步当然是百度了，比如：debian11的镜像源。随缘自己找一个点进去那些网址全部复制下来。
  3. 备份一下源文件：sudo cp /etc/apt/sources.list /etc/apt/sources
  4. 然后打开这个源文件
    sudo vim /etc/apt/sources.list
    (一定要加sudo，不然配置文件没权限修改)
  5. 输入：ggdG 删除所有内容。(注意大小写) [不会用vim的啥都不要管，打开上面那个文件后直接输入ggdG，你是看不到你输入的这个字母的]
  6. 把你复制的那些deb打头的那些网址粘贴进来就行了，鼠标右键粘贴总会吧。
  7. 键盘左上角ESC键点击，输入法英文状态(这是重点),输入:wq(先输冒号)。
  8. 好了这就可以了，更新一下吧
    sudo apt-get upgrade
    sudo apt-get update
