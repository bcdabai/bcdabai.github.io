weget安装卸载

介绍:

wget 是一个从网络上自动下载文件的自由工具，支持通过 HTTP、HTTPS、FTP 三个最常见的 TCP/IP协议 下载，并可以使用 HTTP 代理。"wget" 这个名称来源于 “World Wide Web” 与 “get” 的结合。

所谓自动下载，是指 wget 可以在用户退出系统的之后在继续后台执行，直到下载任务完成。

  1. 使用weget下载报错
    -bash: wget: command not found
  2. 安装weget命令: yum -y install wget 发现weget已经存在
  3. 卸载wget — 命令 yum remove wget
  4. 再运行安装命令，报错发现是没有安装Deltarpm包
  5. 安装deltarpm ：yum install deltarpm -y
  6. 再安装weget成功!!  yum -y install weget
