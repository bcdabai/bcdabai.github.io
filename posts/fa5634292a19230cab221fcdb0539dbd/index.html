求1 + 2！ + 3！ + 4！ + 5！+.....的阶乘

这是一道很简单的题目了。就是用递归的思想。像我这种菜鸟一下就能写出来，

我码的代码

//算法：递归 
#include <iostream>
using namespace std;

//求阶乘函数 
int Factorial(int n)
{
	if(n == 1) return 1;
	else
		return (n * Factorial(n - 1));
}

int main()
{
	int t;
	int sum = 0;
	cin >> t;
	for(int i = 1; i <= t; i++) {
		sum += Factorial(i);
	}
	cout << sum << endl;
	
	return 0;
}
