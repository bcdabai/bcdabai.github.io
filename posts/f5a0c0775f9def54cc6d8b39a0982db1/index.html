C语言中用for循环实现的倒金字塔

图案示例

其实和之前说过的正立金字塔一毛一样，只不过这个每一行的星星规律不同。

第一行9个

第二行7个

不难看出规律为：2*（5-i）-1

#include<stdio.h>
int main(){
	int i,j,k;
	for(i=0;i<5;i++){
		for(k=1;k<=5;k++){
			if(k<=i+1-1)printf(" ");
		}
		for(j=1;j<=2*(5-i)-1;j++){
			
			printf("*"); 
		}
		printf("\n");
	}
}
