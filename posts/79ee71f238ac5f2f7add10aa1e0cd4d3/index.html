(css)解决弹窗下的下拉选择框被遮挡或显示不全问题

修改前：

原因：

1.原框架设置了overflow:hidden或auto，使得内容超出后会自动出现滚动条。
2.设置了定位


解决方案：

将overflow的值改成visible 且用position: static !important;来清除定位

/deep/ .ivu-table,
/deep/.ivu-table-wrapper {
  overflow: visible;
  position: static !important;
}
