Excel 快捷键

给表格添加所有边框（All boarders）：Alt + H + B + A

筛选Filter：Ctrl+Shift+L

复制上一个单元格: Ctrl + D

从上一个单元格复制值: Ctrl +Shift + "（双引号）

从上一个单元格复制公式: Ctrl + '（单引号）

复制左侧的一个单元格：Ctrl + R

复制单元格的值：Ctrl+V -> Ctrl-> V

SUM: Alt + =

输入当前日期: Ctrl + ;（分号）

输入当前时间: Ctrl +Shift + :

拆分字符串的自动填充：Ctrl + E

应用通用格式 ：Ctrl + Shift + ~

货币格式 ：Ctrl + Shift + $

百分比格式 ：Ctrl + Shift + %

科学记数格式 ：Ctrl + shift + ^

日期格式 ：Ctrl + shift + #

时间格式 ：Ctrl + shift + @

数字格式 ：Ctrl + shift + ！
