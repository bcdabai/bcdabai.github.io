Mac安装redis与后台启动

安装使用以下命令：

安装：brew install redis

安装后的目录是：/usr/local/bin

启动redis：redis-server

停止redis：redis-cli shutdown

设置后台运行

sudo vim /usr/local/etc/redis.conf

搜索文件找到：daemonize no这一行，修改为：daemonize yes

启动redis并查看启动进程

#后台方式启动
redis-server /usr/local/etc/redis.conf 
查看启动进程
ps aux | grep redis

进入客户端调用

redis-cli
