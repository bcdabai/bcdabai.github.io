4.输入两个正整数 m 和 n(m<n)，求 m 到 n 之间(包括 m 和 n)所有素数的和，要求定义并调用函 数 is_prime(x)来判断 x 是否为素数(素数是除 1 以外只能被自身整除的自然数) Python3
例如，输入 1 和 10，那么这两个数之间的素数有 2、3、5、7，其和是 17
输入：m n
输出：素数和

def is_prime(n):
    if(n<2):
        return False
    for i in range(2,n):
        if n%i==0:
            return False
    return True

m,n=(input()).split(" ")
m=int(m)
n=int(n)
sum=0
for i in range(m,n+1):
    if is_prime(i):
        sum+=i
print(sum)
