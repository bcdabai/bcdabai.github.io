问题： org.mybatis.spring.MyBatisSystemException: nested exception is org.apache.ibatis.executor.ExecutorException: Error selecting key or setting result to parameter object. Cause: org.apache.ibatis.reflection.ReflectionException: Could not set property 'id' of 解决方式： 对于org.apache.ibatis.executor.ExecutorException: There was no TypeHandler found for parameter...这个异常，是由于javaType和jdbcType的类型不匹配造成的。 解决方法： <selectKey keyProperty="id" resultType="int" order="AFTER"> 该为 <selectKey keyProperty="id" resultType="Long" order="AFTER">
