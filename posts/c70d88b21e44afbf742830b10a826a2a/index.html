IDEAdebug调试时查看时间戳和Date类型的对象，格式化为yyyy-mm-hh

平时debug时遇到Date或者时间戳时看时间不好判断到底是几月几号，希望直接格式化为想要的格式

比如 下面，Date直接显示为 yyyy-mm-hh的格式

解决方案：

无需下载插件，如图

//Date格式化
return new SimpleDateFormat("yyyy-MM-dd HH:mm:ss").format(this);

//Long格式化
Long time = this;
if (time.toString().length() == 10) {
//Long为秒时
    return new SimpleDateFormat("yyyy-MM-dd HH:mm:ss").format(new Date(time * 1000));
}
if (time.toString().length() == 13) {
//时间戳
    return new SimpleDateFormat("yyyy-MM-dd HH:mm:ss").format(new Date(time));
}
