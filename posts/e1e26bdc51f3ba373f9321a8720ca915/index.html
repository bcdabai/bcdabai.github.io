如何使用命令拉取指定git分支

使用git clone 遇见git did not exit cleanly (exit code 128)

使用Git Bash Here拉取，报错提示：fetch-pack invalid index-pack output

只需要，点击Git Bash Here，输入：git clone -b 分支名 --depth=1 git仓库地址（http://xxx.git）
