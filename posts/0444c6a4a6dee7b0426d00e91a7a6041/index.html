本地起node服务（用于脚本）

一、新建一个文件夹，在该文件下创建index.js子文件。

二、初始化node,构建package.json

在文件夹下cmd运行：

npm init -y

会在根目录下生成一个package.json文件。

三、如果需要安装插件

🌰：安装axios

在文件夹下cmd运行：

npm install axios

会在根目录下生成一个node_modules文件夹和package-lock.json文件。

在index.js中使用：

const axios = require('axios')

return axios({
	method: 'get',
	url: url,
})
	.then(() => {})
	.catch()

四、启动项目

在文件夹下cmd运行：

node ./index.js
