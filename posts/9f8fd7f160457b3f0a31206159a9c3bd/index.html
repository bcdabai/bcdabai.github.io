Jenkins获取gitlab代码

1.gitlab上设置个人访问令牌

登录gitlab--选择项目进入--User Settings--Access Tokens

2.Jenkins，安装gitlab插件

系统管理--插件管理--搜索GitLab Plugin插件安装（在available plugins里搜索安装，我是已经安装过了所以在installed plugins中）

 3.Jenkins，配置gitlab链接

Jenkins--系统管理--系统配置--Gitlab编辑

填写连接名称、gitlab地址、gitlab凭证

 gitlab凭证添加：

类型选择：Gitlab API token

API token：填写第一步中gitlab中生成的个人令牌字符串

id、描述：填写第一步gitlab填写的token name即可

 配置完凭证后，点击test connection测试直到success为止

 4.jenkins,配置对应工程中的git信息

5.Jenkins构建

开始测试构建，构建成功，表示配置成功
