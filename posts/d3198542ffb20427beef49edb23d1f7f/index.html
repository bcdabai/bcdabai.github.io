游戏：Java编程实现猜数（0——100），最多猜10次

 随机产生一个整数，用户输入猜的数字，系统提示猜大了或者猜小了，输入次数不超过10次；如果正确，则显示“you are so smart”，

public class guessFigure {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		Scanner input=new Scanner(System.in);
		int answer = (int) (Math.random() * 100) + 1;
		for(int i=1;i<11;i++) {
			System.out.println("please input an figure between 0~100 you guess:");
			double figure =input.nextDouble();
			if (figure<answer)
				System.out.println("your answer is a littlle low ");
			else if(figure>answer)
				System.out.println("your answer is a littlle high ");
			else 
				System.out.println("you are so smart ");
			if(i==10) {
				System.out.println("your aready input 10 times");
				}
			}
		}
	}
