Java：1+2+3+....+100的和

如题：

public class Summ{
    public static void main(String[] args){
        int sum = 0;

        for(int i=1;i<=100;i++) {

            sum += i;
        }
        System.out.println("1+2+3+....+100的和:"+sum);
        }
    }
