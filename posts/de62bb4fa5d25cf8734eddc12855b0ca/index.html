解决报错：Module not found: Error: ‘element-plus/lib/theme-chalk/index.css 和 Can‘t resolve...

当在使用element-plus的icon时有以下两种报错的解决方法

报错1：

解决方法：打开 element.js 文件（路径为 src > plugins > element.js）

// 将文件中的两条数据都改了:

import 'element-plus/lib/theme-chalk/index.css'

import locale from 'element-plus/lib/locale/lang/zh-cn'

// 改为:

import 'element-plus/theme-chalk/index.css'  // 直接将该条的 lib 去除即可，否则会报错

import locale from 'element-plus/es/locale/lang/zh-cn'  // 将 lib 换成 es

当报以下错误时

解决方法：打开  element.js 文件

 将文件中的

import 'element-plus/es/theme-chalk/index.css'

改为

import 'element-plus/theme-chalk/index.css'
