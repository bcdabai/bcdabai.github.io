满屏幕的爆红

说找不到less依赖，但是less依赖早已下载好并且投入使用了。就像删除重新下载，结果就连删除都失败了，于是换了一种下载方式：使用yarn

全局安装yarn

npm install -g yarn

下载yarn

yarn install 

结果出现了以下错误： 

 解决方法：删除了package.lock.json

重新下载

又出现了更多错误，顺着报错去找，npm没有换源   运行

npm config set registry https://registry.npm.taobao.org

重新下载，success！项目重新正常运行！！！
