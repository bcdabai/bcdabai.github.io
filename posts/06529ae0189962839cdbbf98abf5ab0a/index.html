Python 编写一个函数，输出整数m的全部素数因子，如m= 120，输出2,2,2,3,5

def isprime(num):  #判断因子是不是素数
    flag = 1
    for i in range(2,num):
        if num % i == 0:
            flag = 0
    return flag      #如果是素数返回1 不是返0
