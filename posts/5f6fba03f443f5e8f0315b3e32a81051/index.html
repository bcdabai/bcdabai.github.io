一则ORA-00922: missing or invalid option 错误的解决方法

SQL语句：

create table tempdb.dbo.temptable_20191018205038214 as select * from k_menu;

抛出的错误：

> ORA-00922: missing or invalid option

导致原因：表名中不能有符号点 "."，而且长度不能太长，最大30字符，
否则超过长度会报

> ORA-00972: identifier is too long

改成下面这样就OK了。

create table temptable_20191018205038214  as  select * from k_menu;
