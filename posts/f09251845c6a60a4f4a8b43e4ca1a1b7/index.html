JS文本流转字符串FileReader()

	let reader = new FileReader()
	// 传入需要被转换的文本流 file,这个是转字符串的关键方法
	reader.readAsText(file)
	// onload是异步的,封装的话可以用promise
	reader.onload = () => {
		// 输出字符串
		console.log(reader.result)
	}

注意：如果你从接口拿到的拿到的文本流是个字符串格式，务必转成文本流格式
