PTA 4-1 计算分段函数[1]

本题目要求计算下列分段函数f(x)的值：

输入格式:

输入在一行中给出实数x。

输出格式:

在一行中按“f(x) = result”的格式输出，其中x与result都保留一位小数。

输入样例1:

10

输出样例1：

f(10.0) = 0.1

输入样例2:

0

输出样例2：

f(0.0) = 0.0

答案：

#include <stdio.h>
int main()
{
	double result,x;
	scanf("%lf",&x);
	if(x==0){
	printf("f(0.0) = 0.0");
    }
	else if(x!=0){
	result=1.0/x;
	printf("f(%.1lf) = %.1lf\n",x,result);
    }
	return 0;
}

考察点：

浮点数控制几位小数的方法如下：m是包括整数部分位数、小数部分位数和小数点占位一共的位数，而n是小数部分位数。

%m.nf 或 %m.nlf

也可以有简便的书写方法，例如，本题要求控制一位小数，则可以只写小数点和小数部分：

printf("f(%.1lf) = %.1lf\n",x,result);
