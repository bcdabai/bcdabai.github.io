C语言:计算t=1+1/2+1/3+...+1/n

##题目描述

计算t=1+1/2+1/3+...+1/n

##输入描述

整型变量n

##输出描述

t(保留六位小数)

#include<stdio.h>
int main()
{
    int N;
    
    double i,sum=0;
    
    scanf("%d",&N);
    
    for(i=1.0;i<=N;i++)
    {
       sum+= (1.0/i);
    }
    
    printf("%.6lf",sum);
    
    return 0;
}
