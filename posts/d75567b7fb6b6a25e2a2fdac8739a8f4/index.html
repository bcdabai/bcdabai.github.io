解决SPSS中can not create java virtual machine问题

SPSS安装后要输入激活码，但是打开认证的时候就提示“can not create java virtual machine”错误，无法继续进行。

解决方法是，在安装目录下，创建命名为“lservrc"的文本文件，把激活码复制进去，保存。然后删除扩展名，再打开SPSS可以看到功能要比没有激活的多了很多。
