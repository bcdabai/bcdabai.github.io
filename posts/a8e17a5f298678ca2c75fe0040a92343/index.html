在jetson上编译Qt5.12.9
https://blog.csdn.net/GeForeverr/article/details/109160215
其中qt的编译配置文件最好改成这样，否则有可能会无法启动具有ui界面的程序
（我这边的话，linuxfb用不了，要用xcb才行，所以需要改一下；在远程登陆的桌面中，直接运行不了，需要加上命令 [-platform xcb]）

#!/bin/sh
#为防止污染qt的源文件文件夹，可以在另外的地方新建文件夹，然后指定好configure的相对路径
../qt-everywhere-src-5.12.9/configure \
-prefix /opt/qt-5.12.9-arm \
-confirm-license \
-opensource \
-release \
-make libs \
-xplatform linux-aarch64-gnu-g++ \
-pch \
-qt-libjpeg \
-qt-libpng \
-qt-zlib \
-linuxfb \
-xcb \
-opengl es2 \
-no-opengl \
-no-sse2 \
-no-cups \
-no-glib \
-no-dbus \
#-no-xcb \
#-no-separate-debug-info \
