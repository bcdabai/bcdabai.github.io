Vuex

Vue有五个核心概念，state, getters, mutations, actions, modules。

  1. state => 基本数据
  2. getters => 从基本数据派生的数据
  3. mutations => 提交更改数据的方法，同步！
  4. actions => 像一个装饰器，包裹mutations，使之可以异步。
  5. modules => 模块化Vuex
